recipe_name: 香烤豆腐
practice: 烤
flavor: 香辣味
time: 30分钟
grade: 新手尝试
materials: 主料千叶豆腐四百克辅料烧烤酱一大勺蚝油一大勺植物油一大勺孜然粉适量烧烤料适量白芝麻适量辣椒粉适量白砂糖一小勺
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162210273579919.jpg>第1步: 准备好食材</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162210273534634.jpg>第2步: 千叶豆腐切0.5毫米厚的片</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162210273688245.jpg>第3步: 豆腐片放大碗中，加入1大勺烧烤酱、1大勺蚝油、1小勺白砂糖抓匀，使每块豆腐片都包裹上酱汁。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162210273618477.jpg>第4步: 盖保鲜膜冷藏腌制一夜。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162210273651679.jpg>第5步: 腌好的豆腐片码在垫了油纸的烤盘中，每块上刷一层薄油，撒上辣椒粉、烧烤料、孜然粉、白芝麻。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162210273641846.jpg>第6步: 送入烤箱中层180℃烤15分钟，烤箱预不预热都可以。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162210273780590.jpg>第7步: 取出翻面，并得重复刷油撒粉的动作，继续烤15分钟。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162210357275942.jpg>第8步: 鲜香微辣，因为腌制了一夜，非常入味。</a>
