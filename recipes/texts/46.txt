recipe_name: 不用煎烤炸也可以做出美味的鲳鱼
practice: 烧
flavor: 家常味
time: 15分钟
grade: 新手尝试
materials: 主料鲳鱼2条大葱150克生姜3片辅料六月鲜10克轻盐酱油30毫升味达美臻品料酒30毫升清水200毫升
step: <a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164517189458353.jpg>第1步: 准备食材：鲳鱼，大葱，生姜，料酒和酱油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164517189702296.jpg>第2步: 先将切好的大葱倒入锅内，铺平垫底，这样将鲳鱼放在上面也不容易粘锅</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164517189887705.jpg>第3步: 放上几片生姜片可以去腥</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164517189781481.jpg>第4步: 将鲳鱼洗净放入锅内，在鲳鱼的表面划上几刀斜刀，可以更入味</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164517189778059.jpg>第5步: 倒入30毫升味达美臻品料酒</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164517189847019.jpg>第6步: 倒入30毫升10克轻盐特级原汁酱油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164517189836521.jpg>第7步: 倒入200毫升清水，盖上盖子大火炖煮10分钟</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164517189841336.jpg>第8步: 打开锅盖，此时鲳鱼已经熟了，将鲳鱼捞出</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164517189858637.jpg>第9步: 将汤汁收干即可</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164517196900941.jpg>第10步: 不放一滴油，不用煎烤炸也可以做出美味的鲳鱼</a>
