recipe_name: 红烧带鱼
practice: 烧
flavor: 咸鲜味
time: 10分钟
grade: 新手尝试
materials: 主料带鱼300克葱丝少许姜丝2片料酒1勺食盐1克辅料葱段15克姜1片香叶1片干辣椒1个陈醋1勺蚝油1勺老抽1勺白胡椒粉1克白糖2克酱油1勺
step: <a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264723784549.jpg>第1步: 先把需要的食材提前准备好。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264722788919.jpg>第2步: 带鱼去掉表面银色部分及内脏，里面的黑膜及鱼血也一起去掉哦。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264723008203.jpg>第3步: 处理好的带鱼用清水冲洗干净放入碗中，加入葱丝、姜丝、1勺料酒、1克食盐抓拌均匀后腌制10分钟。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264722712872.jpg>第4步: 腌带鱼的同时调个料汁：1勺陈醋、1勺酱油、1勺老抽、少许蚝油、2克白糖、1克黑胡椒粉搅拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264722931530.jpg>第5步: 带鱼腌好后去掉葱丝、姜丝，用厨房纸擦干表面水分，避免煎的时候嘣到自己。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264723429356.jpg>第6步: 锅中倒入少格琳诺尔有机亚麻籽食用油，放入处理好的带鱼煎至完全定型后再翻面，两面全部煎好以后先盛出备用。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264723107004.jpg>第7步: 锅中重新倒入少许格琳诺尔有机亚麻籽食用油。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264723459361.jpg>第8步: 放入葱段、香叶、干辣椒炒香。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264723837869.jpg>第9步: 然后倒入煎好的带鱼，倒入调好的料汁。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264723300461.jpg>第10步: 加小半碗清水大火烧开后转中小火煮至汤汁浓稠后大火收汁。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264723756719.jpg>第11步: 带鱼比较容易熟，所以时间不宜过长。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163264723685884.jpg>第12步: 格琳诺尔有机亚麻籽油，宝宝的聪明油</a>
