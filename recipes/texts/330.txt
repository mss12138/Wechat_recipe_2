recipe_name: 蒜蓉西兰花
practice: 烧
flavor: 家常味
time: 10分钟
grade: 未知
materials: 主料西兰花300克辅料小米辣3个大蒜1头生抽酱油少许蚝油少许食盐1匙鸡精1匙白糖1匙水淀粉1勺
step: <a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188155765309.jpg>第1步: 选购新鲜西兰花，上面开花的不要，老了。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188155868827.jpg>第2步: 用你发财的小手，掰成均匀小块。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188155801372.jpg>第3步: 锅中烧水，加入食盐和食用油，水开后放入西兰花焯水一分钟。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188155937891.jpg>第4步: 捞出后冷水冲洗。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188156077532.jpg>第5步: 将西兰花的花朝下，根部向上，均匀码放在小碗中。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188156156348.jpg>第6步: 大蒜拍碎切成末。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188156332482.jpg>第7步: 小米辣切成小段。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188156340143.jpg>第8步: 锅中放油，放入大蒜末煸香。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188156472596.jpg>第9步: 放入蚝油炒匀。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188156589792.jpg>第10步: 加入一小碗忘情水。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188156535070.jpg>第11步: 加入小米辣。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188156638976.jpg>第12步: 紧接着，放入食盐，鸡精，白糖，生抽酱油。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188156758760.jpg>第13步: 加入水淀粉，搅匀，使汤汁粘稠即可。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188156841949.jpg>第14步: 找个盘子扣在小碗上，然后来个180度后空翻。（此图是后空翻后的画面）</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188156980130.jpg>第15步: 翻完，拿走小碗，是酱紫的。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163188157018124.jpg>第16步: 将料汁浇在西兰花上即可。</a>
