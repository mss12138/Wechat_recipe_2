recipe_name: 酸菜鱼
practice: 烧
flavor: 酸味
time: 30分钟
grade: 中级掌勺
materials: 主料百叶半张黄豆芽1把花鲢鱼1条薄盐骨汤酸菜鱼调料1包辅料食用油2汤匙料酒2汤匙香菜1根生姜1块香葱2根
step: <a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747979943654.jpg>第1步: 准备好全部食材，这里我用的是三斤左右重的花鲢鱼（胖头鱼），做酸菜鱼可以选用鱼很多，如草鱼、黑鱼、鲈鱼、鳜鱼和龙利鱼等肉厚刺少的都可以，大家选择自己喜欢的鱼来做；调料我是这种袋装薄盐骨汤的酸菜鱼调料</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747979679605.jpg>第2步: 配菜也是按自己的喜好来放，今儿我选用的是百叶和黄豆芽，都是吸味起鲜的食材，全部清洗干净：百叶切丝，葱切末，香菜切段；生姜分别切丝和切片，放一旁备用</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747979835021.jpg>第3步: 接下来处理花鲢鱼：去鳞去腮去内脏，里里外外洗净后擦干水分</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747979836854.jpg>第4步: 先将鱼头切开，接着鱼身两侧将鱼线拽出来，这步可去除腥味；拽的时候边拽边轻轻拍打，就可轻松拽出来了</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747979696832.jpg>第5步: 一只手按着鱼身，一只手拿刀沿着脊背、贴着鱼骨将二块鱼肉片下来，然后再将鱼肚上的一排鱼刺也片下来；剩余的鱼骨切块，与鱼头鱼尾一起，可用油煎一下炖汤喝超级鲜美</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747979677318.jpg>第6步: 将鱼肉朝上、鱼皮朝下，斜刀从尾部开始片鱼，第一刀片至鱼皮处不要切断；第二刀片好切断，二片鱼肉展开为一个蝴蝶状；鱼片尽可能片的薄些，一汆就熟还不易变老</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747980170515.jpg>第7步: 鱼片切好后用流水冲洗，多洗几遍，可使鱼肉更加雪白鲜嫩，捞出控干水分</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747980011855.jpg>第8步: 把备好的酸菜鱼调料拆开：里面分别有腌鱼包、酸菜包、调料包和2小袋白芝麻</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747980712223.jpg>第9步: 接着把鱼片放入碗中，加入葱末、姜丝和料酒，再倒入腌鱼包食材</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747980048036.jpg>第10步: 用手抓拌均匀，腌制20分钟，就不用再放其它调料了</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747980256725.jpg>第11步: 下面来做酸菜鱼了，先炒配菜：起锅烧油，倒入黄豆芽和百叶丝，翻炒</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747980383015.jpg>第12步: 把黄豆芽和百叶丝炒软炒至断生，盛入碗中垫底</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747980686269.jpg>第13步: 锅中重新倒油，先把酸菜包里的盐水哔掉，再把酸菜倒入锅中，翻炒；其实将酸菜浸泡后再炒，味道会更好</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163748757301675.jpg>第14步: 炒出香味，倒入调料包和适量清水搅拌均匀，这这包调料是由盐、味精、猪骨、鸡骨、牛骨等多种肉骨熬制出来的复合型骨汤，酸菜鱼所有的重点就是要有这么一锅汤了，这种汤非常鲜美，营养更加丰富；同时还是减盐调料，减了25%的盐分，吃起来更加健康</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747980398335.jpg>第15步: 大火煮开尝一下味道，若觉得过咸可以再加点水；然后一片一片下入鱼片，别着急搅动</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747980527680.jpg>第16步: 待鱼片微微定型，转大火轻轻推动鱼片，待再次沸腾煮至鱼片卷曲、变色即可关火</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163747980828667.jpg>第17步: 出锅后倒入装有百叶豆芽的碗中，最后撒上葱花、香菜和白芝麻增色增香</a>
