recipe_name: 干锅羊排
practice: 干煸
flavor: 香辣味
time: 15分钟
grade: 新手尝试
materials: 主料羊排骨1斤半洋葱1个红辣椒1个青辣椒1个胡萝卜1个芹菜2根姜几片辅料菜籽油适量盐适量辣椒粉适量孜然粉适量白胡椒粉适量
step: <a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104682564666.jpg>第1步: 羊排剁成1寸半到2寸长的块，太大块的肉我剔了下来，做成了烤羊肉串。剁好的羊排泡半小时的水，水就红就换水，直到水变清。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104682956222.jpg>第2步: 将洗干净的羊排放入锅中，放几片姜，先煮熟。如果是嫩羊排，就用电饭锅煮，成年羊排就用高压锅煮，不然后面嚼不动。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104683105135.jpg>第3步: 煮好的羊排捞起，滤干水。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104683053384.jpg>第4步: 洋葱、青红椒、胡萝卜和芹菜洗好切好，再切几片姜。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104682906540.jpg>第5步: 锅烧热，多倒一些油，烧热，倒入羊排炸至微焦，然后盛起。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104683312878.jpg>第6步: 锅内留一点底油，下姜片，将洋葱、青红椒和胡萝卜炒至断生。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104683092722.jpg>第7步: 将前面炸好的羊排倒入。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104683309815.jpg>第8步: 加入1小勺盐。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104683404616.jpg>第9步: 加1小勺辣椒粉，也可以放干辣椒。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104683570390.jpg>第10步: 加1小勺孜然粉。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104683522787.jpg>第11步: 再加1小勺白胡椒粉。混合翻炒一两分钟。可以尝下够不够味，根据自己的口味适当再加点调料。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104683628802.jpg>第12步: 最后倒入芹菜，翻炒至断生就可以关火了。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164104743615321.jpg>第13步: 起锅装盘，香辣有嚼劲，太好吃了。</a>
