recipe_name: 香煎鸡翅包饭
practice: 煎
flavor: 香辣味
time: 2小时
grade: 中级掌勺
materials: 主料鸡翅12只糯米一碗葱姜蒜末适量干香菇4朵辅料油一勺盐少许蚝油两勺生抽三勺料酒一勺黑胡椒粉适量辣椒粉适量孜然粉适量
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674819908448.jpg>第1步: 糯米淘洗干净，加入两倍水，浸泡两个小时以上，如果时间充足可以泡一个晚上。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674819976780.jpg>第2步: 泡好的糯米变得胖乎乎的，将糯米捞出来，平铺在铺了纱布的蒸锅里。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674819934523.jpg>第3步: 大火蒸30分钟就熟透了，盛出来，放凉。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674819992905.jpg>第4步: 接着咱们来处理鸡翅，将鸡翅两端，皮和骨头之间的经络剪断，然后轻轻将鸡翅的骨头取出来。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820061560.jpg>第5步: 全部都取出来后，放入盆中，加入生抽、蚝油、料酒拌匀，腌制一个小时左右。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674819979966.jpg>第6步: 葱姜蒜都切成末，干香菇用清水泡发，也切成小颗粒。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820120580.jpg>第7步: 新锅先烧开一锅水，洗刷干净后稍微烧干。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820045160.jpg>第8步: 倒入油，烧至五成热后，将姜蒜末和干香菇碎到入锅中，中小火煸炒出香味。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820135912.jpg>第9步: 凉透的糯米饭倒入锅中，慢慢将米饭充分炒散。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820121469.jpg>第10步: 糯米饭中加入一勺生抽上色增香，尝尝咸淡，酌情加盐调味，炒好后盛出来，稍微放凉点。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820147271.jpg>第11步: 这时候鸡翅腌制好了，将炒好的糯米饭用筷子塞入鸡翅中，稍微塞满点，煎出来会更饱满，造型会更漂亮。
没煎的时候有点软哒哒的，不要着急，等煎熟后鸡翅会紧缩。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820137447.jpg>第12步: 锅里倒入一勺油，（鸡翅在煎的过程中会析出油脂，所以油尽量少放点。）烧至六成热的时候，将鸡翅摆入锅中，中火慢煎。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820116803.jpg>第13步: 鸡翅两面都煎至微微上色。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820165017.jpg>第14步: 淋入少许水。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820215310.jpg>第15步: 盖上锅盖，将鸡翅完全焖熟，水没过鸡翅的三分之一就可以了，水太多了会影响糯米饭的口感。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820237843.jpg>第16步: 锅底还有少许水的时候，打开盖子将鸡翅两面的水分都煎干就可以了。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820255433.jpg>第17步: 将鸡翅一个个夹出来，一个好的油锅，可以将鸡翅里面的油脂都煎出来，多余的油脂倒掉。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674820206291.jpg>第18步: 用水将锅简单冲洗下就干净了。</a>
