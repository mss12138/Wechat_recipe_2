recipe_name: 健脾除湿酿秋葵
practice: 其它工艺
flavor: 咸鲜味
time: 10分钟
grade: 初级入门
materials: 主料秋葵9个白芸豆50克薏米10克熟鸡肉丁20克杂菜(胡萝卜、玉米、青豆丁)20克辅料鲜酱油1小勺蚝油1小勺料酒1小勺白砂糖1小勺
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088594492203.jpg>第1步: 准备好所有食材及调味料。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088571436811.jpg>第2步: 白芸豆加水浸泡一昼夜，天热的话放冰箱冷藏。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088571472287.jpg>第3步: 薏米也加水浸泡一昼夜，天热的话放冰箱冷藏。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088571592103.jpg>第4步: 浸泡得胖胖的白芸豆撕去豆皮。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088571541959.jpg>第5步: 白芸豆和薏米放入电饭锅，加足够的水煮约1小时，煮到豆子绵软出沙。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088571444743.jpg>第6步: 秋葵用盐水浸泡10分钟洗净，烧一锅开水，水中加少许盐护色，下秋葵煮沸再煮1~2分钟，不要把秋葵煮软。捞出用冷水过凉。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088643800782.jpg>第7步: 杂菜也焯熟。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088571659087.jpg>第8步: 芸豆和薏米放碗中，加入杂菜和熟鸡肉丁，用勺子压碎豆子。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088571622009.jpg>第9步: 鲜酱油、料酒、蚝油、白砂糖都放入碗中，搅拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088571696845.jpg>第10步: 用刀划开秋葵一边的皮，注意不要划穿另一面。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088571662668.jpg>第11步: 填入调好味的芸豆薏米馅。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162088571626679.jpg>第12步: 秋葵爽脆，豆子绵软，咸鲜可口。</a>
