recipe_name: 西芹拌白木耳
practice: 拌
flavor: 酸甜味
time: 15分钟
grade: 新手尝试
materials: 主料西芹300克白木耳10克辅料枸杞2克六月鲜10克轻盐特级原汁酱油2勺味达美清香米醋2勺酒酿30克

<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162173760419107.jpg>第2步: 锅内倒入适量的清水</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162173760246797.jpg>第3步: 将水煮开</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162173759449726.jpg>第4步: 将洗净的西芹切成小段倒入进行汆烫</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162173760092874.jpg>第5步: 将提前泡发好的白木耳倒入一起进行汆烫</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162173760720947.jpg>第6步: 差不多汆烫10分钟即可</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162173760542372.jpg>第7步: 将汆烫好的白木耳和西芹盛入盘中</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162173760544672.jpg>第8步: 在碗中放入30g酒酿</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162173760517576.jpg>第9步: 倒入2勺六月鲜10克轻盐特级原汁酱油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162173760392784.jpg>第10步: 倒入2勺味达美清香米醋</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162173760484382.jpg>第11步: 调味汁搅拌均匀之后倒入事先汆烫好的西芹和白木耳，搅拌均匀即可</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162173760672481.jpg>第12步: 最后撒上一些枸杞点缀</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162174918194286.jpg>第13步: 酸中带有一些咸甜，非常开胃适合夏天食用</a>
