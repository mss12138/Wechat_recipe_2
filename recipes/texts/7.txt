recipe_name: 西兰花小饼【宝宝辅食】
practice: 煎
flavor: 家常味
time: 15分钟
grade: 新手尝试
materials: 主料西兰花30克面粉35克辅料鸡蛋1个
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164843193327717.gif>第1步: 西兰花焯水，捞出。你们也能换成其他的绿叶菜，菠菜、油菜啥的都可以。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164843194485287.gif>第2步: 把焯好的西兰花稍微晾一晾，放入料理机当中。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164843197285968.gif>第3步: 再加上蛋清、30克的面粉。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164843198674011.gif>第4步: 启动料理机，把搅打好的蔬菜泥放入裱花袋当中，剪出一个小口，备用。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164843200350157.gif>第5步: 蛋黄里加入5克面粉，搅拌到均匀没有颗粒的状态。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164843201883125.gif>第6步: 同样的，给蛋黄液装入裱花袋当中，剪一个小口，放到一边备用。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164843203320320.gif>第7步: 我用的是不粘锅，所以这里就不用刷油了。把刚才的蛋黄液，挤入到锅中。不用挤太多，这个就是中间的花蕊了。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164843205009249.gif>第8步: 再次往锅里挤入西兰花面糊，煎到小饼定型。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164843207641376.gif>第9步: 因为这个小饼不算大，所以还是挺好熟的，如果担心熟成度，可以在锅里稍微加点水，盖盖焖一下，熟透取出来就行。</a>
