recipe_name: 藜麦山药蒸肉饼
practice: 蒸
flavor: 家常味
time: 30分钟
grade: 新手尝试
materials: 主料藜麦 15克猪肉70克山药 120克辅料鸡蛋1个小葱6克香菇粉3克
step: <a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564772003057.jpg>第1步: 准备食材</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564774074917.jpg>第2步: 藜麦倒入碗中，提前浸泡2小时。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564776295166.jpg>第3步: 山药去皮，切成小块。山药含有大量粘液，在处理山药的时候，一定要戴手套防护，否则碰到山药的粘液手很容易过敏发痒。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564777486914.jpg>第4步: 猪肉切成小块。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564782144322.jpg>第5步: 小葱切末。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564784568758.jpg>第6步: 山药倒入破壁机中，加入猪肉，葱末，打入鸡蛋。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564790285880.gif>第7步: 启动破壁机，搅打细腻。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564793314321.gif>第8步: 倒入碗中，加入泡洗好的藜麦。加入香菇粉，继续搅拌均匀</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564795673697.gif>第9步: 倒入盘中。看！最后要像我动图中这样，面糊是不易滴落的状态。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564799129189.gif>第10步: 用勺子把肉糊铺匀，如果勺子比较粘的话，可以把勺子沾一下水，就没那么粘啦。用普通耐高温的盘子就可以了</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564801249249.jpg>第11步: 放入蒸锅，盖盖。大火蒸20分钟。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564804699580.gif>第12步: 熟透取出，切块或者用模具压成可爱的形状。放凉之后再脱模效果会更好。就可以拿出来切成宝宝的方便拿的小块了。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162564806839206.jpg>第13步: 肉类辅食也可以做得软绵绵的，很好嚼的。</a>
