recipe_name: 黄瓜炒肉片
practice: 炒
flavor: 家常味
time: 5分钟
grade: 初级入门
materials: 主料黄瓜1根小米辣1个精瘦肉100克辅料食用油适量葱姜少许料酒1小勺生抽1小勺生粉1小勺食盐半小勺白糖半小勺白胡椒粉少许
step: <a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163834393669908.jpg>第1步: 黄瓜和猪肉准备好，一个小米辣是为了增加风味和搭配色泽的，不能吃辣的可以不放</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163834394005075.jpg>第2步: 首先将黄瓜用水清洗干净，斜刀先切成小段，翻过来再切成菱形薄片；小米辣去蒂切成辣椒圈，香葱切末，生姜切丝</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163834394084198.jpg>第3步: 精瘦肉冲洗干净后也切成薄片，新手下厨，若担心切不好，可以将猪肉在冰箱微微冻硬一会儿，切起来就顺手多了</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163834393631010.jpg>第4步: 把肉片放入碗中，加入葱姜、料酒、生抽，食盐、白糖、生粉和白胡椒粉，抓拌均匀</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163834394019472.jpg>第5步: 再淋入少许食用油锁住水分，腌制10分钟</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163834394009737.jpg>第6步: 锅热倒油，放入腌好的肉片，翻炒，炒至肉片断生变色，先盛出来备用</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163834394022344.jpg>第7步: 锅中留底油，下入葱姜和小米辣，煸出香味</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163834394478516.jpg>第8步: 放入黄瓜，翻炒半分~一分钟，待黄瓜变色断生</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163834394425121.jpg>第9步: 倒入肉片和少许的食盐，肉片是腌制过的，所以食盐的量不要多，要不太咸会影响口感</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163834394686490.jpg>第10步: 翻炒均匀，即可关火，因为肉片已熟的，所以翻炒时间不易过久</a>
