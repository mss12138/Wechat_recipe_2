recipe_name: 崇明绿皮茄子炖鱼
practice: 炖
flavor: 家常味
time: 30分钟
grade: 新手尝试
materials: 主料青鱼500克绿皮茄子300克生姜10克葱10克辅料六月鲜特级原汁酱油2勺六月鲜红烧酱油2勺味达美臻品料酒2勺水200毫升
step: <a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214460755295.jpg>第1步: 准备食材：绿皮茄子洗净切块，青鱼洗净切块，生姜切片</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214461180895.jpg>第2步: 将锅先加热，锅内先放入生姜片</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214461211281.jpg>第3步: 将青鱼块放入锅内煎制一会儿，让鱼肉内的脂肪可以渗出</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214461008772.jpg>第4步: 倒入2勺料酒，去腥</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214461413068.jpg>第5步: 倒入2勺老抽（红烧酱油）</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214461445035.jpg>第6步: 倒入2勺生抽（原汁酱油）</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214461164987.jpg>第7步: 轻轻翻拌均匀，让鱼肉染上酱汁</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214461150343.jpg>第8步: 放入洗净的绿皮茄子块，翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214462498127.jpg>第9步: 倒入200ml清水</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214461255070.jpg>第10步: 盖上盖子大火炖煮10分钟</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214461378934.jpg>第11步: 打开锅盖，轻轻翻动让其上色更均匀</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163214461391774.jpg>第12步: 最后撒上葱花即可关火</a>
