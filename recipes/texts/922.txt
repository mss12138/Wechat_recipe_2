recipe_name: 菠萝烤翅
practice: 烤
flavor: 果味
time: 60分钟
grade: 中级掌勺
materials: 主料菠萝半个鸡翅9个辅料蒜末2勺姜片适量蜂蜜1勺食盐适量鸡精适量黑胡椒粉适量料酒10克生抽15克老抽5克蚝油8克菠萝水25克食用油2勺
step: <a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162082997762022.jpg>第1步: 菠萝切片</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162082997923731.jpg>第2步: 鸡翅表层划上两刀，方便入味</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162082998016501.jpg>第3步: 鸡翅放入碗中，倒入料酒、老抽、生抽、蚝油、食盐、鸡精、姜片、蒜末、黑胡椒粉、菠萝汁（增加鸡翅的果香味）</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162082998211208.jpg>第4步: 将鸡翅和酱料抓拌均匀，盖上保鲜膜，至少腌制两个小时</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162082998403870.jpg>第5步: 烤盘里面放两层锡纸或者油纸，然后摆入鸡翅，鸡翅表面再摆放菠萝，这样果香味会更浓郁</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162082998580327.jpg>第6步: 送入预热好的烤箱，200度烤15分钟</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162082998709094.jpg>第7步: 将菠萝拨到一边，在鸡翅表面刷一层腌鸡翅的料汁，这样颜色更好看</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162082998815029.jpg>第8步: 将1勺蜂蜜和2勺食用油混合搅拌均匀，然后刷到鸡翅表层，这样吃起来更香甜，颜色也更好看</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162082999016765.jpg>第9步: 再次将鸡翅送入烤箱，200度烤20分钟</a>
