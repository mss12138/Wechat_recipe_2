recipe_name: 家常窖焗鸡翅
practice: 烘焙
flavor: 咸鲜味
time: 2小时
grade: 新手尝试
materials: 主料鸡翅700克盐1小勺姜片适量葱段适量生抽酱油1勺蚝油1勺料酒1勺黑胡椒粉小半勺辅料盐焗鸡粉15克（半小包）
step: <a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163988004424216.jpg>第1步: 鸡翅700克洗干净，水里放盐1小勺，浸泡20分钟，可以去血水和入味，然后沥干水分备用。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163988004408295.jpg>第2步: 放入姜片适量、葱段适量、蚝油1勺、生抽酱油1勺、料酒1勺、黑胡椒粉小半勺、盐焗鸡粉15克（半小包）。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163988004414643.jpg>第3步: 搅拌均匀后包上保鲜膜，腌制2个小时入味。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163988004486058.jpg>第4步: 取出腌好的鸡翅，整齐的排列在烤盘上。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163988004437908.jpg>第5步: 烤箱180℃提前预热，烤制25-30分钟，10分钟的时候翻个面，刷一层食用油。如果要烤的颜色均匀一些，每5分钟翻个面，再刷一层油。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163988004476755.jpg>第6步: 烤至表面金黄焦脆即可。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163988004593477.jpg>第7步: 家常窖焗鸡翅，完成O(∩_∩)O</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163988004527582.jpg>第8步: 外壳酥脆，入口鲜嫩多汁，好吃到舔手指。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163988004482229.jpg>第9步: 教程快手，漂亮又美味，非常适合宴客哟。</a>
