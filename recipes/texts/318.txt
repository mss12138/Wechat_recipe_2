recipe_name: 广式脆皮烧鸭家庭版
practice: 烤
flavor: 咸鲜味
time: 数天
grade: 未知
materials: 主料鲜鸭或者樱桃谷鸭（2.8斤-3斤）1只白糖25克鸡精20克十三香1克沙姜粉1克五香粉1克八角粉（肉桂粉）1克以上调料搅拌均匀备用69克葱一根姜一大块蒜子10颗s钩一个味精5克盐15克辅料柱候酱25克绵白糖10克沙茶酱2克红腐乳1块花生酱1.5克以上凉拌均匀为酱料酱料45克酱油5克清水40克浓汤宝半块海鲜酱10克味精2克
step: <a href=https://st-cn.meishij.net/rs/19/224/6056019/n6056019_163229498045586.jpg>第1步: 首先鸭子买回来要纳盆，放入冷水解冻。如图。我这边是北方用的都是冻鸭子，鲜的超市买不到，如果两广地区的朋友，买鲜鸭子很方便</a>
<a href=None>第2步: 鸭子解冻好了以后，用清水在泡45分钟目的是把鸭子原有的血水泡干净避免腥臊味这步很关键，期间隔20分钟换次水</a>
<a href=None>第3步: 解冻的时候，我们把酱和盐调好，然后切葱一点葱花，蒜末，姜末，备用，锅烧热留适量油，把葱姜蒜炒香，放入酱料，炒出香味备用即可</a>
<a href=None>第4步: 鸭子买回来虽然是处理好的，但是膛内还仍然有多余的油脂和内脏，要处理干净。然后控诉多余水分，</a>
<a href=None>第5步: 先均匀往膛内抹25克盐，腌制10分钟，目的再次把多余血水杀出来，方便二次入味。</a>
<a href=None>第6步: 在抹30克盐，抹好以后在往膛内抹25克酱料冷藏腌制6个小时即可</a>
<a href=None>第7步: 用S钩，勾住鸭子脖劲，锅内适量水，抹过鸭子即可。烧开关火，鸭子正反面烫20秒即可冲凉水，</a>
<a href=https://st-cn.meishij.net/rs/19/224/6056019/n6056019_163229498105985.jpg>第8步: 均匀淋上皮水，风干6-8个小时，（一定要用风扇吹）【皮水比例：50克白水、麦芽糖6克、大红浙醋5克、蜂蜜1克、食用碱0.5克】</a>
<a href=https://st-cn.meishij.net/rs/19/224/6056019/n6056019_163229498272371.jpg>第9步: 烤箱预热240度，预热好后放入鸭子</a>
<a href=https://st-cn.meishij.net/rs/19/224/6056019/n6056019_163229498439397.jpg>第10步: 210度烤制15分钟目的是为了上色</a>
<a href=https://st-cn.meishij.net/rs/19/224/6056019/n6056019_163229498608303.jpg>第11步: 15分钟后，烤箱降到200度，在烤10-15分钟</a>
<a href=https://st-cn.meishij.net/rs/19/224/6056019/n6056019_163229498779888.jpg>第12步: 出炉，</a>
<a href=https://st-cn.meishij.net/rs/19/224/6056019/n6056019_163229498849238.jpg>第13步: 如果不怕麻烦在放完盐酱的情况下可以在肚子里放30清水，10克东古，2片葱，1片姜，一个拍蒜。最后用长竹签缝上（像缝衣服一样）然后烫皮，挂皮水风干烤制</a>
<a href=None>第14步: 最后把剩下的盐和酱到在锅里，放入40了清水5克酱油，加热搅拌均匀，放入蘸料碟里蘸食即可</a>
