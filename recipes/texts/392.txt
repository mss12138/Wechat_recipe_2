recipe_name: 柠檬凤爪＆烤羊肉串，夏天就应该这么吃！
practice: 拌
flavor: 家常味
time: 30分钟
grade: 初级入门
materials: 主料鸡爪1000克柠檬1个辅料蒜1头料酒1勺蚝油1勺原汁酱油2勺糖2勺香油少许小葱1根洋葱头2个姜粉少许花椒少许小茴香少许香叶2片桂皮1块盐少许
step: <a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163041191703231.jpg>第1步: 准备好全部食材，鸡爪清洗干净。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163041233713931.jpg>第2步: 把鸡爪放入锅内（冷水下锅），加入料酒、盐、葱段、姜粉，大火煮沸。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163041236184428.jpg>第3步: 水沸腾后，捞出浮沫。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163041239275178.jpg>第4步: 关火，一拨升起，快速把鸡爪放入冰水！</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163041242412787.jpg>第5步: 再次冲洗干净，放回锅内，注入纯净水，没过鸡爪即可，再放入料包、葱段、少许盐、1勺蚝油，中小火炖煮15~20分钟。（这里我设置的是1200w中火）</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163041246219704.jpg>第6步: 煮好后，升起来控水晾凉。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163041247427096.jpg>第7步: 柠檬用盐搓洗干净，擦干水分，切薄片，去籽。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163041252050512.jpg>第8步: 准备好的调料全部放入大碗内，再倒入半碗纯净水拌匀。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163041252941097.jpg>第9步: 把料汁倒入密实袋，柠檬片也一并放进去。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163041269276011.jpg>第10步: 把鸡爪放进去，挤出空气，轻轻揉匀，冷藏至少2小时。（冷藏腌渍久一点，味道更佳哟！）</a>
