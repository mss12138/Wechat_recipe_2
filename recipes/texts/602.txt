recipe_name: 台式三杯鸡
practice: 炖
flavor: 咸鲜味
time: 30分钟
grade: 初级入门
materials: 主料鸡翅根600克青椒半个红椒半个辅料酒酿500克老抽15克生抽30克罗勒叶1把大蒜1头姜1块小葱2棵盐少许香油少许
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162556746567487.jpg>第1步: 鸡翅根泡水10分钟；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162556746526906.jpg>第2步: 剁成两半；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162556746577329.jpg>第3步: 入温水锅中煮开，避开浮沫捞出；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162556746538139.jpg>第4步: 配菜准备好：新鲜罗勒叶、青椒、红椒清洗干净，大蒜1头剥皮，酒酿只取汁水；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162556746500329.jpg>第5步: 罗勒摘下来，青红椒去籽切块；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162556746599422.jpg>第6步: 炒锅中倒少许油，大蒜、姜片、葱段入锅，煸炒出香味；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162556746641258.jpg>第7步: 鸡腿入锅煸炒2分钟；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162556746622671.jpg>第8步: 倒入酒酿、老抽、生抽，大火煮开后转小火，盖盖子焖炖；老抽上色，生抽提鲜，酒酿要充足，不要加水；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162556746605974.jpg>第9步: 汤汁浓香，鸡肉软烂入味，青红椒入锅翻拌变色；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162556746679731.jpg>第10步: 罗勒入锅，全部浸在汤汁中，炖1分钟，释放出香味，起锅前适量加盐，淋少许芝麻香油。</a>
