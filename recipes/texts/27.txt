recipe_name: 糖醋虾饼【宝宝辅食】
practice: 煮
flavor: 咸鲜味
time: 30分钟
grade: 新手尝试
materials: 主料虾仁160克卷心菜40克蛋清30克辅料柠檬适量油适量盐适量鲜菇粉适量
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164687510246409.gif>第1步: 卷心菜切碎，放入料理机当中。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164687513005202.gif>第2步: 然后来处理一下虾仁，给虾仁去掉虾线，同样放到料理机中。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164687513539872.gif>第3步: 打入蛋清，这是为了让虾饼更加软嫩Q弹，再放入鲜菇粉、盐来调味。家里有柠檬汁的，可以少加几滴来去腥，没有的用白醋代替也行。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164687516040399.gif>第4步: 启动料理机，给这些食材都充分搅打成糊糊的样子。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164687516922937.gif>第5步: 不粘锅刷油，取适量的虾泥整理成小饼的样子。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164687519180226.gif>第6步: 开小火，烙到虾饼底部微黄定型了，就给它翻个面。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164687519989432.gif>第7步: 趁着烙饼的时候，我们可以来调个酱汁。2勺的番茄酱、5克糖、3克的淀粉和适量清水，这个糖醋汁你不管做什么都特别香。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164687522713094.gif>第8步: 给酱汁搅和匀了，倒入锅中。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164687523445940.gif>第9步: 翻炒几下，等到汤汁逐渐浓稠了，就能取出来装盘了。</a>
