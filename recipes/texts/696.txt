recipe_name: 烤牛奶
practice: 烤
flavor: 甜味
time: 30分钟
grade: 初级入门
materials: 主料牛奶500毫升蛋黄2个鸡蛋1个辅料细砂糖25克芝士片2片玉米淀粉适量金黄面包糠适量
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380311095.jpg>第1步: 材料大合影，准备一口厚底奶锅或者不粘锅；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380277376.jpg>第2步: 牛奶、1.5个蛋黄、细砂糖、玉米淀粉50克同入小锅；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380255013.jpg>第3步: 小火加热，把芝士片入牛奶中，边加热边用耐高温刮刀轻轻混合均匀；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380302895.jpg>第4步: 随着温度的升高，芝士片融化，且材料逐渐变成浓稠状；此时要不停地用刮刀抄底搅拌，防止糊底；有小气泡不断产生时即关火；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380303948.jpg>第5步: 提前清洁好2个耐高温保鲜盒，将牛奶糊倒入其中，表面用刮刀尽量抹平；盖上盖子，入冰箱冷藏40分钟以上，直到完全凝固定形；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380387193.jpg>第6步: 倒扣，轻松脱模，用利刀切成均匀的等份；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380354989.jpg>第7步: 码放在铺了油纸的烤盘里，表面刷上提前留出来的半个蛋黄；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380434693.jpg>第8步: 用普通烤箱需要预热200度，用空气炸烤箱无需预热，直接将牛奶块送入；中层，上下火200度，15-20分钟，根据烤箱的实际情况、表面蛋黄液的薄厚等调整温度或时间；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380446034.jpg>第9步: 明黄色的烤牛奶上有块块褐色的烤痕，趁热吃香浓软糯，但要注意防烫噢；稍凉后再吃也不错，冷藏后再吃会有冰冰凉的口感，但Q弹感消失；怎么吃可自行决定；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380442608.jpg>第10步: 另一盒牛奶切块；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380484859.jpg>第11步: 将裹料准备好：1个鸡蛋打散、玉米淀粉、金黄面包糠；牛奶块依次蘸裹蛋液、玉米淀粉、面包糠；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380448327.jpg>第12步: 码放在烤盘上；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162418380488349.jpg>第13步: 入空气炸烤箱中层200度，12分钟左右；如用普通烤箱需预热200度，中层，15分钟左右；出炉后趁热吃，外皮酥脆，内部浓香柔软，注意防烫噢。</a>
