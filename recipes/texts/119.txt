recipe_name: 酸汤鱼
practice: 炖
flavor: 酸味
time: 30分钟
grade: 初中水平
materials: 主料花鲢鱼700克蟹味菇300克辅料植物油适量盐适量酸汤肥牛底料110克姜20克
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160674743213906.jpg>第1步: 主料：新鲜花鲢鱼700克、蟹味菇300克
辅料：植物油、盐、花椒粒1克、辣皮子10个、大葱一根、姜20克、香菜50克、酸汤肥牛底料110克</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160674743256680.jpg>第2步: 把新鲜花鲢鱼收拾干净多清洗几次，去掉脊背上的鱼线，切块，用料酒和盐拌匀腌制10分钟左右，这样鱼不腥不易碎</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160674743230650.jpg>第3步: 把蟹味菇用淡盐水泡泡洗净，辣皮子泡泡，姜切片，大葱切段</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160674743219873.jpg>第4步: 把臻米多功能料理锅火力调到最大档位，加植物油烧热，加姜片和花椒粒、辣皮子爆出香味以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160674743384633.jpg>第5步: 加清水或者高汤都可以，用鸡汤最好了，加酸汤肥牛底料110克，一包就可以了，味道特别的足</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160674743320269.jpg>第6步: 加锅盖，大火烧开，这个多功能锅，有三档火力，5秒速热，3分钟沸腾，蒸煮煎烤焖样样精通，不粘锅底，轻油少烟，好用好清洗，一擦即净。4L黄金容量，操作简单。</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160674743388065.jpg>第7步: 下腌制好的鱼块，不要搅拌</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160674751228955.jpg>第8步: 下蟹味菇，稍微铺均匀一些，让汤汁盖住鱼块和蟹味菇</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160674743313417.jpg>第9步: 用大火煮开，再小火慢慢10分钟左右即可，具体时间看鱼块的重量确定</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160674743395001.jpg>第10步: 加少许盐，等鱼块和酸汤味道融合在一起了，酸香四溢，鱼块鲜美了，蟹味菇入味了，就这样的</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160674746156246.jpg>第11步: 加香菜段，即可上桌吃了，鱼肉鲜嫩不腥不腻，蟹味菇鲜美，酸汤四溢，尤其是酸汤太好喝了，比肉汤都鲜美开胃，一口气可以喝几碗了，泡米饭尤其的好。可以一边吃一边煮，还可以下不同的食材一起煮，比火锅都好吃了。</a>
