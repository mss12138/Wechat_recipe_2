recipe_name: 自制凉皮
practice: 其它工艺
flavor: 家常味
time: 5分钟
grade: 初级入门
materials: 主料中裕面包用小麦粉1200克蒜泥一头蒜黄瓜一根胡萝卜一根辅料水半碗盐适量醋三勺辣油子适量芝麻酱三勺生抽一勺
step: <a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944709392469.jpg>第1步: 先揉面，水量一点点加入，搅成絮状再上手，要做到盆光手光面光，揉好盖住醒面半小时</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944709569861.jpg>第2步: 醒发好开始洗面，用手抓揉面团，面水变白换一盆清水接着洗</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944709768935.jpg>第3步: 反复洗面四五次，最后面团抱在一起洗出来的水不是白色的即可</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944709995414.jpg>第4步: 这是洗好的面筋，有韧性有气孔</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944710261870.jpg>第5步: 接下来开始蒸面筋，水烧开大火蒸15分钟</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944710600582.jpg>第6步: 蒸好后面筋柔软有马蜂窝气孔，没有任何添加剂</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944710849462.jpg>第7步: 面筋放凉切块备用</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944711004732.jpg>第8步: 辣椒粉加芝麻 用热油浇香 自制油泼辣子就好啦</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944711307524.jpg>第9步: 配料</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944711503729.jpg>第10步: 洗面水静置沉淀五小时后，把表面水倒掉只留底部沉淀，加入半碗清水，把面水搅拌均匀</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944711989153.jpg>第11步: 热锅烧开水 模具底部刷层油防粘，一勺面水放入模具，再把模具放入烧开的水中，一定要大火盖上盖子蒸两分钟，表面起大泡就可以了</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944712218905.jpg>第12步: 拿出的凉皮快速放入凉水盆中降温，轻轻一揭即可</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162944712464997.jpg>第13步: 拿出的凉皮快速放入凉水盆中降温，轻轻一揭即可</a>
