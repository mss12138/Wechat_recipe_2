recipe_name: 脆皮五花肉
practice: 烤
flavor: 蒜香味
time: 90分钟
grade: 初级入门
materials: 主料五花肉1000克辅料孜然5克姜1块干辣椒2个姜蒜末适量盐7克糖10克生抽20克孜然粉5克醋少许桂皮1段
step: <a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503683170612.jpg>第1步: 买了一大块五花肉，取中间一截来做这个烤五花肉</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503683164675.jpg>第2步: 孜然粒，桂皮，生姜盒干辣椒备好用来猪肉</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503683134638.jpg>第3步: 把煮肉料盒五花肉放入锅里，水没过肉</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503683178293.jpg>第4步: 大火煮开转小火煮20-25分钟</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503684222630.jpg>第5步: 煮好的肉捞出控水，皮朝下切连刀块，皮的部分留约5毫米不要切透。</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503684224403.jpg>第6步: 把姜蒜末，生抽，盐，糖，孜然粉放入碗里搅拌均匀成为料汁</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503684280029.jpg>第7步: 把料汁均匀涂抹在肉上，切口处也要涂抹，这样入味均匀。涂抹后腌2--3小时，让其充分入味</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503684281518.jpg>第8步: 腌好后用钢针类的锐器扎孔，排着扎均匀，扎孔烤出来皮会更脆</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503685063761.jpg>第9步: 在肉皮上在撒一点盐，涂抹一层醋，也是为了烤出来更脆</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503685076212.jpg>第10步: 用锡纸包裹底部，只把皮露出来，然后放入烤箱180度烤30分钟。我用的空气炸功能烤的</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503685069139.jpg>第11步: 30分钟后拿出来，在皮上再刷一次醋，继续入烤箱190-200度再烤20分钟</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162503685555263.jpg>第12步: 烤好的脆皮五花肉，皮吃起来非常脆，真香！</a>
