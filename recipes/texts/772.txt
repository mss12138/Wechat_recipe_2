recipe_name: 空气炸鸡翅&薯条
practice: 炸
flavor: 家常味
time: 30分钟
grade: 初级入门
materials: 主料鸡翅中4个大土豆1个辅料姜1块蒜3瓣黑胡椒粉2克盐1克酱油15克料酒15克
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121107242.jpg>第1步: 鸡翅中和调料准备好；鸡翅数量和调料品种、用量自行调整，能吃辣的加辣椒面或者干辣椒均可，做成孜然味的也可，再或者用现成的各种腌肉料、烧烤料也可；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121114992.jpg>第2步: 鸡翅内侧划两刀，可充分入味，正面划两刀也可；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121172541.jpg>第3步: 鸡翅和调料全部入碗中，混合均匀，腌20分钟，中途可翻拌一两次；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121156490.jpg>第4步: 这是腌了20分钟的鸡翅中，味道和部分料汁已经渗入到肉中了；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121178698.jpg>第5步: 在空气炸锅的炸篮底部码放姜片垫底，鸡翅码放在姜片上，炸锅无需预热；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121125989.jpg>第6步: 将炸篮推入炸锅中；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121262655.jpg>第7步: 200度，全程18分钟；可根据鸡翅大小、多少、炸锅的实际温度、喜欢的口感等来调整时间；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121263456.jpg>第8步: 13分钟时将炸篮拉出，用刷子将腌翅的剩余料汁刷在鸡翅表面，翻面与否无所谓；料汁刷两三遍，更利于上色；也可以刷蜂蜜，美拉德反应让颜色更诱人；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121363295.jpg>第9步: 送入炸锅中再烤5分钟；图中这是烤好的状态，颜色倍儿漂亮，味儿倍香；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121575248.jpg>第10步: 炸鸡翅时别闲着，把大土豆削皮；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121356036.jpg>第11步: 先切片再切条，粗细比铅笔略粗一点儿；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121528608.jpg>第12步: 入开水锅中焯烫1分钟，土豆有6分熟了，表面的淀粉糊化，进空气炸锅后会很快形成一层焦皮儿；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121478209.jpg>第13步: 捞出控水；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121353837.jpg>第14步: 将炸完鸡翅的炸锅清洗干净，把薯条随意倒进去，想吃无油的就直接推入炸锅中，想要上色更快也更香一点儿，可以用油壶在表面喷少许植物油或者随意刷点油；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162288121377245.jpg>第15步: 180度18分钟；根据薯条的粗细调整时间；中途可拉出炸篮颠颠锅，使薯条均匀受热。</a>
