recipe_name: 芋头烧鸭腿
practice: 炖
flavor: 家常味
time: 30分钟
grade: 新手尝试
materials: 主料芋头500克鸭腿2只干山楂50克生姜10克小葱10克辅料六月鲜红烧酱油1勺味达美味极鲜酱油3勺味达美清香米醋1勺冰糖15克八角10克清水500毫升

<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293414575210.jpg>第2步: 锅内放入两片生姜</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293414774791.jpg>第3步: 放入洗净切块的鸭腿</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293414778592.jpg>第4步: 倒入适量的水没过鸭腿肉</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293414663496.jpg>第5步: 大火煮沸进行焯水，放入1勺米醋进行去腥</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293414658075.jpg>第6步: 煮沸后即可关火捞出鸭腿肉</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293414605970.jpg>第7步: 锅内放入一片生姜</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293414723672.jpg>第8步: 倒入焯过水的鸭腿肉</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293414346367.jpg>第9步: 倒入3勺味达美味极鲜酱油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163295059798632.jpg>第10步: 倒入500ml清水</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293415262019.jpg>第11步: 倒入50g干山楂</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293415366073.jpg>第12步: 放入15克冰糖</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293415421764.jpg>第13步: 放入10克八角</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293415260705.jpg>第14步: 盖上盖子大火炖煮15分钟，打开锅盖倒入1勺六月鲜红烧酱油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293415553232.jpg>第15步: 放入洗净的芋头块，盖上盖子炖煮15分钟</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163293415673816.jpg>第16步: 待芋头酥烂了，即可收汁关火</a>
