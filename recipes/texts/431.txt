recipe_name: 上海名菜四喜烤麸
practice: 烧
flavor: 咸鲜味
time: 30分钟
grade: 初中水平
materials: 主料烤麸(水发挤干后)350克黑木耳(干)10克黄花菜(干)20克花生米50克辅料油5大勺六月鲜 特级酱油2大勺六月鲜 红烧酱油2大勺白砂糖1大勺
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946378831036.jpg>第1步: 准备好所有食材与调味料。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946378896676.jpg>第2步: 干泡麸切块，在清水中泡发，反复挤捏，洗去酸味，捞起挤干水分。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946378829463.jpg>第3步: 挤出的水如图呈乳白色的话就得再洗再挤，直到水清。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946378837469.jpg>第4步: 下开水锅中焯水，去掉豆腥味，反复用冷水冲凉。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946378881239.jpg>第5步: 黑木耳用清水泡发，清洗干净，摘去硬根。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946378841902.jpg>第6步: 黄花菜用温水泡发，也摘去硬根。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946378905590.jpg>第7步: 花生米清洗沥干后，冷油下锅炸熟，炸过的花生米更易煮到绵软。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946378966852.jpg>第8步: 锅烧热下油荡锅，倒入烤麸翻炒，下花生米、木耳、黄花菜。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946378957236.jpg>第9步: 下六月鲜 特级酱油2大勺，六月鲜 红烧酱油2大勺，白砂糖1大勺，加水与食材平齐，用肉汤更佳。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946379134401.jpg>第10步: 加盖煮开转小火焖煮15分钟，注意观察水量，和食材的软硬程度。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946379155726.jpg>第11步: 大火收浓汤汁，出锅装盘。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162946379176915.jpg>第12步: 浓油赤酱，成菜酱香浓郁，油润绵软又有弹性，鲜咸中透着甜蜜，配菜木耳脆、花生香，口感丰富，十分下饭。</a>
