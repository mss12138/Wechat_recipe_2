recipe_name: 虎皮辣子烧茄子
practice: 烧
flavor: 茄汁味
time: 10分钟
grade: 初中水平
materials: 主料长茄子2根青辣椒4个西红柿1个辅料植物油适量盐10克白糖2克生抽10克蚝油10克姜适量大蒜2瓣
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317153516.jpg>第1步: 主料：长茄子两根、尖椒3个、西红柿一个
配料：植物油、盐、大蒜2瓣、姜5克、白糖2克、水淀粉适量、酱油5克（生抽）、蚝油2克、米醋2克</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317213317.jpg>第2步: 把青辣椒洗净去籽（去籽可选，有的为了增加辣味就不用去籽了），切两段，最好和茄子长短一样，葱姜蒜洗净切末</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317233704.jpg>第3步: 把茄子洗净切长条，不要太长了，5-6公分即可，西红柿洗净切块，就是提味提色，不可多加</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317220584.jpg>第4步: 把茄子撒上少许面粉拌均匀，取一小碗，加生抽和蚝油、白糖、盐拌均匀调成味汁（这样便入做菜时，不会手忙脚乱，特别适合厨房新手用）</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317282268.jpg>第5步: 热锅凉油，油热8成下辣椒，炸出虎皮捞出来沥油</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317286543.jpg>第6步: 再下茄子条，也是把茄子炸软即可捞出来沥油（不可太软，稍微变软即可）</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317369591.jpg>第7步: 锅底留底油，加葱姜蒜翻炒出香味</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317379357.jpg>第8步: 加茄子、辣椒、西红柿块翻炒均匀出香味以后，加调味汁翻炒均匀以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895332997798.jpg>第9步: 加西红柿块翻炒出香味</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317325179.jpg>第10步: 加水淀粉翻炒均匀以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317343598.jpg>第11步: 稍微煮一会入味，出锅前，沿着锅边加米醋翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317367202.jpg>第12步: </a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162895317406048.jpg>第13步: 这道菜就可以出锅了色香味俱全，米醋就是画龙点睛的作用，一般我们新疆人做菜加西红柿多，出锅前再加几滴醋提香气味道特别好吃，不可多加，多了会有酸味，真是特别的开胃下饭哟。</a>
