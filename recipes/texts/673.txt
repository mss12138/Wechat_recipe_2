recipe_name: 剁椒鱼片
practice: 蒸
flavor: 咸鲜味
time: 15分钟
grade: 初中水平
materials: 主料鲈鱼1条(约2斤)辅料剁辣椒约100克小葱3棵生姜5片柠檬半个盐1/2小勺白砂糖1小勺味达美臻品料酒1大勺六月鲜柠檬蒸鱼酱油1大勺
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437495508915.jpg>第1步: 准备一条大鲈鱼，葱姜柠檬料酒去腥，剁椒盐糖酱油调味。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437495691471.jpg>第2步: 从鱼尾下刀，平刀沿脊骨往鱼头方向片下半条鱼肉，至鱼头处切断，同样的方法片下另半条鱼肉。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437495676423.jpg>第3步: 鱼骨斩大块，鱼头对半劈开。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437495777059.jpg>第4步: 刀顺着鱼肋骨慢慢往下片，取掉鱼肋排骨。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437495600388.jpg>第5步: 现在片鱼片，右手握刀，鱼尾在左，刀尽可能倾斜，从鱼头方向往鱼尾方向片鱼片，因为是蒸鱼片，鱼片可以厚一点。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437495678418.jpg>第6步: 鱼片清洗干净，鱼骨也清洗一下</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437495718967.jpg>第7步: 盘底铺上葱姜，盖上鱼骨，撒一半的盐糖抹匀</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437495646350.jpg>第8步: 鱼片中加入剩下的盐糖，1大勺味达美臻品料酒</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437495544716.jpg>第9步: 反复抓捏，直到液体被鱼片吸收，鱼片粘稠</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437495749051.jpg>第10步: 将鱼片鱼皮朝上码在鱼骨表面，挤入半个柠檬汁，这个动作我以为拍了，实际却忘了按下录像按键，还有后面的蒸鱼酱油也是。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437496945939.jpg>第11步: 铺上剁椒，淋上六月鲜柠檬蒸鱼酱油。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437497468806.jpg>第12步: 冷水上锅，大火煮开，继续蒸10分钟，注意全程大火。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162437498316280.jpg>第13步: 时间到，出锅，撒上葱花，鲜香微辣，色香味俱全！</a>
