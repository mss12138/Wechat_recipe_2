recipe_name: 回锅肉
practice: 炒
flavor: 酱香味
time: 15分钟
grade: 中级掌勺
materials: 主料五花肉一块洋葱半个葱姜适量蒜苗三根豆瓣酱一勺花椒一把辅料酱油一勺老抽一勺蚝油一勺料酒一勺
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515402553900.jpg>第1步: 五花肉凉水下锅，加入葱姜花椒，汆烫后取出来。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515402565517.jpg>第2步: 汆烫好的五花肉放入水龙头下冲洗干净。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515402530512.jpg>第3步: 然后切成不是很薄的片。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515402590754.jpg>第4步: 将五花肉片摆入盘中。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515402512932.jpg>第5步: 放入蒸锅，大火蒸半个小时。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515406376322.jpg>第6步: 蒸好的五花肉变得透明了，将多余的油脂倒掉，经过蒸制后，五花肉不仅变得柔软，而且内部的油脂大大降低，吃起来更健康。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515407438472.jpg>第7步: 所有的配菜和调料准备好。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515408011758.jpg>第8步: </a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515408634505.jpg>第9步: 1勺料酒、1勺酱油、1勺老抽、1勺蚝油，倒入碗中，调匀成料汁 备用。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515409167999.jpg>第10步: 锅里倒入少许的油，将蒸好的五花肉倒入油锅中，煸炒至微微焦黄。加入一勺豆瓣酱炒出香味。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515409587058.jpg>第11步: 将调好的料汁倒进去，翻炒均匀，让肉片表面均匀上色。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163515410793907.jpg>第12步: 撒入洋葱片、蒜苗，翻炒均匀就可以出锅啦。</a>
