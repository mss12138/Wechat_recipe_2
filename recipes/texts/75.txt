recipe_name: 青椒炒牛杂
practice: 炒
flavor: 香辣味
time: 15分钟
grade: 初中水平
materials: 主料牛杂500克洋葱100克青辣椒4个辅料植物油适量盐适量生抽两大勺姜10克
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659209555.jpg>第1步: 主料：青辣椒4个、牛杂400克、皮牙子（洋葱）100克
辅料：植物油40克、盐10克、辣皮3个、花椒粒5个、生抽10克、大蒜1瓣、鸡精1克、孜然粉1克</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659284160.jpg>第2步: 把买来的牛杂清洗几次，这样就干净多了，我这是买的冷鲜，需要多清几次，去掉异味，下开水里，加花椒粒煮十几分钟捞出来待用</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659234535.jpg>第3步: 大蒜切片或者末，姜切末，皮牙子切块，青辣椒切块</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659250630.jpg>第4步: 热锅凉油，油热7成，下辣皮子和姜末青辣椒翻炒出香味、辣味以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659248576.jpg>第5步: 加牛杂快速翻炒均匀，要中火翻炒，目的就是把辣皮子和姜末的味道，滚进牛杂里，一点要翻炒几下，大约2分钟</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659279215.jpg>第6步: 加料酒翻炒均匀，加皮牙子翻炒均匀，是为了增加香味和色彩，有卖相有食欲</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659251945.jpg>第7步: 加盐翻炒均匀，这样可以把辣皮子辣味滚进牛杂里，如果太干了，可以加少许的清水一起翻炒</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659290387.jpg>第8步: 加生抽翻炒均匀，全程要快</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659249205.jpg>第9步: 加入花椒粉翻炒均匀出香味</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659203056.jpg>第10步: 加入鸡精或者味精翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659207946.jpg>第11步: 加大蒜快速翻炒出蒜香味，加孜然粉翻炒均匀，千万不可多加，目的是提香味，这样就可出锅了</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163999659222783.jpg>第12步: 牛杂麻辣鲜香，有嚼劲真是太开胃下饭了，新疆人下酒菜就是它了。</a>
