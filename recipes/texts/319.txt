recipe_name: 黑椒牛肉粒炒菱角
practice: 炒
flavor: 黑椒味
time: 30分钟
grade: 新手尝试
materials: 主料牛排1块熟菱角170克辅料青椒1个红椒1个黑椒酱适量无盐黄油少许黑胡椒碎少许亚麻籽油少许蚝油少许轻盐酱油少许
step: <a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163228037789094.jpg>第1步: 准备一块牛排、适量新鲜上市的菱角、青红椒各一个；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163228037790807.jpg>第2步: 将菱角洗净后提前煮熟，剥去外壳；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163228037727460.jpg>第3步: 牛排切成小粒，青红椒切成小丁备用；一般市售的牛排都是腌制过的，改个刀炒制起来非常嫩滑的，很方便；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163228037755889.jpg>第4步: 炒锅微微烧热，放入一块无盐黄油；这里提醒一下：黄油入锅时，锅体不需要太高温度，温度太高容易焦黄，影响口感；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163228037733396.jpg>第5步: 待黄油融化后，放入牛肉粒，中小火煸炒至表面焦黄；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163228037725931.jpg>第6步: 放入青红椒、菱角；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163228045907348.jpg>第7步: 倒入适量黑椒酱、蚝油、轻盐酱油大火翻炒入味；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163228047750858.jpg>第8步: 出锅前撒上黑胡椒碎；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163228065427428.jpg>第9步: 上桌后，淋上亚麻籽油即可享用。</a>
