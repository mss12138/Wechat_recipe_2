recipe_name: 肉末蒸蛋
practice: 蒸
flavor: 葱香味
time: 10分钟
grade: 中级掌勺
materials: 主料鸡蛋2个猪瘦肉50克辅料盐少许格琳诺尔有机亚麻籽油适量酱油少许生粉小勺
step: <a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_163159921138357.jpg>第1步: 食材：鸡蛋2个、猪瘦肉20克、香葱、格琳诺尔有机亚麻籽油、食用盐、生抽、生粉。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_163159924049270.jpg>第2步: 鸡蛋去壳，打入一个玻璃大碗里，可以看见这个鸡蛋是很新鲜，蛋黄完美，颜色很深。如果蛋黄是散开的，说明鸡蛋坏了。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_163159925461367.jpg>第3步: 鸡蛋用筷子顺时针调均匀，接着倒入100克温水，再次用筷子顺时针调均。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_163159926497465.jpg>第4步: 蛋液过滤，然后倒入蒸盘，用保鲜膜包裹，牙签戳几个洞，让蒸汽跑出去，就会有水珠流下来。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_163159927931512.jpg>第5步: 蒸锅倒入适量的水，隔水放入蒸笼，把鸡蛋盘放入。盖上盖盖子，小火慢炖8分钟，然后焖2分钟开盖。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_163159929936659.jpg>第6步: 猪瘦肉清洗干净，切薄片，然后剁碎成肉末状。加生粉、油、盐、酱油、搅拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_163159931905567.jpg>第7步: 锅烧热，倒入适量的水，倒入猪瘦肉爆炒熟，快速盛出。肉末别炒太久，熟了即可。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_163159933302901.jpg>第8步: 蒸熟的鸡蛋取出之后，撕开保鲜膜，用刀划小方块，然后把香葱、炒熟肉末、酱油、油，浇在蒸蛋上面。一道味美的肉末蒸蛋就做好了。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_163159937800749.jpg>第9步: 格琳诺尔有机亚麻籽油，宝宝的聪明油</a>
