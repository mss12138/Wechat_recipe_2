recipe_name: 木耳蒸鸡
practice: 蒸
flavor: 咸鲜味
time: 30分钟
grade: 新手尝试
materials: 主料鸡半只干木耳1把姜1小块小葱几根红椒丝适量辅料料酒1汤匙盐1小勺生抽1汤匙蚝油1汤匙玉米油半汤匙
step: <a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231974725175.jpg>第1步: 干木耳1把提前泡发。中间要换几次水。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231975117246.jpg>第2步: 鸡剁块（2-3厘米大），泡几分钟水泡去血水，然后冲洗干净装入大碗中，加入姜丝。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231975127109.jpg>第3步: 加1汤匙料酒。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231974912028.jpg>第4步: 加1小勺盐。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231975175715.jpg>第5步: 加1汤匙生抽。（喜欢颜色深一点的可以加一点老抽。）</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231975296329.jpg>第6步: 加1汤匙蚝油。拌匀，腌至少1小时。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231974931804.jpg>第7步: 将泡发好的木耳洗干净，滤干水，铺在盘中。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231975065952.jpg>第8步: 鸡肉最后加半汤匙玉米油，拌匀。这样锁住鸡肉中的水分。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231975146298.jpg>第9步: 将鸡块铺在木耳上。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231975221704.jpg>第10步: 蒸20-25分钟。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231975247175.jpg>第11步: 切一些葱段和红椒丝（或胡萝卜丝）做装饰。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_163231975416973.jpg>第12步: 最后一分钟，将葱段和红椒丝撒在鸡肉上，盖上再焖两三分钟就好了。</a>
