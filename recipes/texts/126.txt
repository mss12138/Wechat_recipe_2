recipe_name: 茭瓜鸡蛋蒸饺，茭瓜快速入味窍门
practice: 蒸
flavor: 咸鲜味
time: 30分钟
grade: 未知
materials: 主料茭瓜1个500克鸡蛋2个辅料盐(杀水)6克盐2克白胡椒粉1克蚝油5克香油10克麻辣油5克
step: <a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973312822723.jpg>第1步: 茭瓜切掉头尾，清洗干净，不必去皮</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973312931849.jpg>第2步: 切成小米丁，别切太大，否则后面不好吃，也别过小，否则口感不好，丁的大小是约5毫米的正方形丁</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973312997278.jpg>第3步: 加6克盐，杀出多余水分，然后反复两次，把茭瓜的水攥干</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973313019039.jpg>第4步: 杀出多余的水分后，再来调味，这样更容易入味了，吃起来也不会有瓜的生涩感了，加入香油，麻辣油，蚝油，白胡椒粉，盐，一定按照顺序放进去，搅拌均匀</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973313140935.jpg>第5步: 锅里放油，油量稍微宽一点</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973313164607.jpg>第6步: 倒入鸡蛋液，搅打鸡蛋的时候，可以打一点点水进去，这样炒出来的鸡蛋更嫩滑</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973313234827.jpg>第7步: 炒鸡蛋注意⚠️：快炒，大火，炒熟炒散炒碎，一定大火快炒，这样可以更好的锁住鸡蛋里面的水分，切勿小火慢慢炒，或者反复长时间翻炒，这样会把鸡蛋的水份炒干，炒出来的鸡蛋口感像吃鸡蛋皮一样不好吃，所以只要看见蛋液结块就可以立刻出锅了</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973313389696.jpg>第8步: 把炒好的鸡蛋放茭瓜里</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973313325952.jpg>第9步: 搅拌均匀</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973313485040.jpg>第10步: 包饺子</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973313473944.jpg>第11步: 捏紧封口</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973313586158.jpg>第12步: 全部包好</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973313561784.jpg>第13步: 锅里烧开水，水上汽后，放入饺子，蒸10分钟即可</a>
<a href=https://st-cn.meishij.net/rs/05/216/13679005/n13679005_163973313604274.jpg>第14步: 出锅</a>
