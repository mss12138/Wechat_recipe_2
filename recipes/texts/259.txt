recipe_name: 爆炒牛杂
practice: 炒
flavor: 香辣味
time: 15分钟
grade: 初中水平
materials: 主料牛杂500克洋葱150克毛芹菜10克辅料植物油适量盐适量酱油两大勺辣皮子20个
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808090632.jpg>第1步: 主料：处理好的500克
辅料：植物油40克、盐10克、辣皮子30个、花椒粒2克、生抽10克、大蒜两瓣、味精2克、孜然粉2克、毛芹菜50克、皮牙子100克</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808067916.jpg>第2步: 把辣皮子（最好用新疆的线干红辣椒）用凉水泡泡，最好泡20分钟一样，为了节省时间，可以用热水泡一分钟，这样辣皮子炒出来不糊特别的香浓，捞起来切段，大蒜切片或者末，姜切末，毛芹菜、皮牙子切段和丝</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808189803.jpg>第3步: 把买来的牛杂清洗几次，这样就干净多了，我这是买的冷鲜，需要多清几次，去掉异味，下开水煮几分钟捞出来，待用</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808140248.jpg>第4步: 热锅凉油，油热7成，下辣皮子和姜末翻炒出香味、辣味以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808195405.jpg>第5步: 热锅凉油，油热7成，下辣皮子和姜末翻炒出香味、辣味以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808158409.jpg>第6步: 加料酒翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808122561.jpg>第7步: 加生抽翻炒均匀，全程要快</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808103523.jpg>第8步: 加盐翻炒均匀，这样可以把辣皮子辣味滚进牛杂里，如果太干了，可以加少许的清水一起翻炒</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808279200.jpg>第9步: 加几根毛芹菜段和皮牙子翻炒均匀，是为了增加香味和色彩，有卖相有食欲</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808278979.jpg>第10步: 加入花椒粉翻炒均匀出香味</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808202279.jpg>第11步: 加入鸡精或者味精翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808137445.jpg>第12步: 加大蒜快速翻炒出蒜香味</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163240808298574.jpg>第13步: 加孜然粉翻炒均匀，千万不可多加，目的是提香味，这样就可出锅了，牛杂麻辣鲜香，有嚼劲真是太开胃下饭了，新疆人下酒菜就是它了。</a>
