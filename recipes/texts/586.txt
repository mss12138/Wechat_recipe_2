recipe_name: 夏日轻食玉米金盅
practice: 其它工艺
flavor: 家常味
time: 90分钟
grade: 初中水平
materials: 主料玉米粉200克奶粉30克鸡胸肉120克辅料芹菜110克红椒1根木耳少许炼乳适量零卡糖25克泡打粉4克温水110克橄榄油少许盐少许六月鲜8克轻盐特级原汁酱油葱花少许姜末少许生粉少许白胡椒粉少许
step: <a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162605825885176.jpg>第1步: 准备好玉米金盅所需的食材：玉米粉200克、奶粉30克、炼乳适量、泡打粉4克、零卡糖25克、温水110克；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162605825829288.jpg>第2步: 将200克玉米粉、25克零卡糖、4克泡打粉混合，加入适量炼乳和110克温水搅拌均匀，揉成团，盖上保鲜膜醒30分钟左右；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162605825891814.jpg>第3步: 醒好之后，将面团分成45克左右的小剂子，并搓揉成团；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162605825875953.jpg>第4步: 将模具表面刷上植物油防粘，然后将揉成团的小剂子放在模具里压成型；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162605825804646.jpg>第5步: 依次做好后放入蒸箱蒸20分钟；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162605825852285.jpg>第6步: 蒸好后略微晾凉，脱模备用，奶香浓郁的玉米金盅就做好啦；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162605825990736.jpg>第7步: 接下来开始准备轻食食材：将鸡胸肉洗净后切成小丁，放入少许盐、生粉、葱花、姜末、白胡椒粉抓匀，腌制片刻。将芹菜、红椒、木耳洗净后也切成小丁</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162605825953889.jpg>第8步: 炒锅烧热后，倒入少许橄榄油，放入鸡丁翻炒变色后，放入红椒丁炒出香味；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162605825947565.jpg>第9步: 再放入芹菜丁、木耳丁，淋入4毫升六月鲜8克轻盐原汁酱油，翻炒入味即可；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162605903040890.jpg>第10步: 最后将炒好的轻食食材盛入玉米金盅中，就可以享用一道既营养丰富，又轻盐轻脂的美味啦！</a>
