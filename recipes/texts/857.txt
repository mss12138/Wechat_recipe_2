recipe_name: 蒜末丝瓜蒸粉丝
practice: 蒸
flavor: 蒜香味
time: 30分钟
grade: 未知
materials: 主料丝瓜1根绿豆粉丝1把辅料大蒜1颗辣椒2个香葱1根
step: <a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_162161899627579.jpg>第1步: 粉丝提前用冷水泡软，丝瓜也去皮洗净，切成1厘米厚的片，再将大蒜剥去外皮，切成蒜末备用；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_162161899751415.jpg>第2步: 将泡好的粉丝中倒入1勺生抽，搅拌均匀，增加粉丝的底味；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_162161899866561.jpg>第3步: 将切好的丝瓜边角料铺在碗底，再放上泡好的粉丝，最后码上丝瓜片备用；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_162161899954710.jpg>第4步: 下面开始制作金银蒜：将切好的蒜末用清水洗掉表面粘液，滤干水分备用，这样操作能减少大蒜的苦涩味和蒜味，并且后续炒制时不容易粘锅；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_162161900142771.jpg>第5步: 准备炸金蒜，起锅倒入适量食用油，油热6成时，倒入一半蒜末，转成中小火，煸炒至表面金黄；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_162161900222375.jpg>第6步: 剩余的另一半的水洗蒜末中调入1勺生抽、1勺蚝油、1勺糖、小半勺盐搅拌均匀，再倒入锅中，利用锅内余热将两种蒜搅拌均匀，这样美味的“金银蒜”蒜末酱就做好了；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_162161900377714.jpg>第7步: 将做好的蒜蓉酱均匀的淋在碗中，再将蒸碗放入蒸锅中，上汽后继续用大火蒸8分钟左右即可；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_162161900435985.jpg>第8步: 出锅后撒上葱花或者辣椒碎点缀即可，如果想要看着更诱人则可以淋上适量热油；</a>
