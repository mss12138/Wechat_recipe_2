recipe_name: 三色藜麦鸡蛋饼
practice: 煎
flavor: 甜味
time: 15分钟
grade: 新手尝试
materials: 主料三色藜麦50克盐2克清水75克鸡蛋1个低筋面粉100克辅料玉米油适量
step: <a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162864315482959.jpg>第1步: 将三色藜麦淘洗干净，然后倒入适量清水浸泡15分钟。快浸泡好，准备好其他材料。（藜麦如何挑选：优质藜麦直径大于小米，颗粒饱满，扁圆形状，胚芽侧包紧而长）</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162864316827467.jpg>第2步: 将浸泡好的藜麦倒入干净不粘锅中，加入适量清水，开中火，盖锅盖，煮开，然后再计时煮10分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162864318354576.jpg>第3步: 然后捞出到干净盆中。看，超级多的“小尾巴”，出芽率超高。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162864319716389.jpg>第4步: 然后打入鸡蛋，倒入面粉。加入盐。开始调味了哈，配料就这么简单，没有复杂的。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162864321555169.jpg>第5步: 然后倒入75克清水，这个水量可以略调整，将所有材料搅拌成均匀无颗粒的面糊，略带粘稠感，但是有流动性的。喜欢粘稠一点就少加点水，喜欢稀一点的面糊，就多加个一点点，这个没有什么太多的讲究。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162864322943485.jpg>第6步: 然后不粘平底锅，倒入适量玉米油，晃匀。然后开中火将油烧热。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162864324710402.jpg>第7步: 将一半面糊倒入，然后晃平或者借助锅铲摊平，转小火，将一面煎熟略带金黄。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162864326610966.jpg>第8步: 然后翻面，再将另一面煎熟，两面略带金黄即可。不带也没事，煎熟即可。</a>
<a href=https://st-cn.meishij.net/rs/197/154/2663697/n2663697_162864328290809.jpg>第9步: 取出，切片，装盘。</a>
