recipe_name: 蔬菜千层烘蛋，减脂餐头号种子选手！
practice: 烤
flavor: 咸鲜味
time: 60分钟
grade: 新手尝试
materials: 主料鸡蛋2个培根80克辅料胡萝卜 2个西葫芦1个黄瓜1个芝士干酪 4片黑胡椒粉少许盐 少许
step: <a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162071542880008.jpg>第1步: 先来处理食材，胡萝卜、西葫芦、黄瓜、芝士干酪切成长条片状。太厚容易折断，熟得也慢，太薄又过于考验刀工，所以最简便的方法就是用刨刀直接刨出薄片。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162071544101914.jpg>第2步: 把各类食材一片搭着一片地叠放在菜板上。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162071545649444.jpg>第3步: 从顶端开始慢慢卷起来，然后盘成一个大圆。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162071546488853.jpg>第4步: 把所有的蔬菜卷好后，放入铺了油纸的容器。这个容器得选择耐高温、可入烤箱的焗盘、烤盘或模具。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162071547296526.jpg>第5步: 鸡蛋磕入碗里，加入少许盐和黑胡椒粉，打散，混合均匀。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162071548160424.jpg>第6步: 蛋液淋在蔬菜上。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162071548873968.jpg>第7步: 送入预热好的烤箱，上下火190度，中下层烘烤25-30分钟即可，出炉后趁热吃更加美味。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162071549882107.jpg>第8步: 开吃。</a>
product: <a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162071550709768.jpg>蔬菜千层烘蛋，减脂餐头号种子选手！成品图</a>
