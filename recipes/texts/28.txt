recipe_name: 胡萝卜黄瓜炒肉丁
practice: 炒
flavor: 家常味
time: 15分钟
grade: 初中水平
materials: 主料胡萝卜150克黄瓜一根黄萝卜150克猪肉200克辅料植物油适量生抽适量蚝油2勺姜5克
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164680205860323.jpg>第1步: 主料：胡萝卜150克、黄萝卜150、黄瓜一根、猪瘦肉300克
辅料：植物油、盐、生抽、蚝油、干淀粉5克、料酒2克、姜5克</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164680207363241.jpg>第2步: 把胡萝卜洗净切小丁丁，黄萝卜也洗净切小丁丁，黄瓜洗净切小丁丁，把胡萝卜和胡萝卜下开水焯真3分钟左右，捞出来，过凉水沥干水分（过凉水口感比较脆可选），这样去掉异味，而且可以快速炒熟</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164680205860456.jpg>第3步: 猪瘦肉切小丁丁，用五香粉和生抽拌均匀，加少许的干淀粉拌均匀，在加一勺植物油拌均匀，这样水分不易流失，炒出来的瘦肉比较香浓鲜嫩</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164680226500678.jpg>第4步: 热锅凉油，加植物油，烧到6成热，下猪肉丁，炒到变色出香味捞出来，中间可加料酒去腥味翻炒</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164680207352641.jpg>第5步: 锅底留底油，加姜末炒出香味，下胡萝卜和胡萝卜丁丁炒出水分和香味，多翻炒几下</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164680230550202.jpg>第6步: 下黄瓜丁丁，一起翻炒均匀出香味，这样黄瓜易熟，可以在后面下锅炒</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164680207361858.jpg>第7步: 加盐翻炒均匀，多翻炒几下，炒出三种丁丁的香味</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164680207306147.jpg>第8步: 加炒好的肉丁翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164680207406672.jpg>第9步: 加入生抽翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164680207489889.jpg>第10步: 加入蚝油翻炒均匀，出酱香味就可以出锅了</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164680238930743.jpg>第11步: 这道菜酸香味俱全了，孩子看着颜色亮丽也有食欲了。</a>
