recipe_name: 家常糖醋小排
practice: 烧
flavor: 酸甜味
time: 60分钟
grade: 新手尝试
materials: 主料猪小排500克黄酒1大勺生抽2大勺香醋3大勺白砂糖4大勺辅料热开水适量
step: <a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163875451569524.jpg>第1步: 提前准备一下调料，把黄酒1大勺、生抽2大勺、香醋3大勺、白砂糖4大勺，放一起搅拌均匀备用。（感觉糖可以减掉1勺，正好适合我的味道）</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163875451523862.jpg>第2步: 猪小排500克，切成小块，浸泡在温水里10分钟左右，泡出血水。然后沥干水分，用厨房纸吸干水分。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163875451504916.jpg>第3步: 锅里不加油，大火把锅烧至热至冒烟后，放入猪小排，先不翻面，把一面煎出焦脆，再翻炒慢煎5分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163875451526072.jpg>第4步: 煎出猪油，猪排表面煎至金黄。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163875451510593.jpg>第5步: 倒入热开水适量，以淹没猪排为止。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163875451506560.jpg>第6步: 倒入前面准备好的调料。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163875451646070.jpg>第7步: 开大火煮开后转小火，盖上盖子闷烧40分钟左右，中途记得翻炒几次，防止沾底。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163875451669805.jpg>第8步: 最后开大火，翻炒收汁即可。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163875451649772.jpg>第9步: 糖醋小排，完成O(∩_∩)O</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163875451645658.jpg>第10步: 浓油赤酱，色泽油亮，口味酸甜，味道好极了。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_163875451612813.jpg>第11步: 真的超级简单，而且色香味俱全。</a>
