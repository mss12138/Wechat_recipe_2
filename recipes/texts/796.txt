recipe_name: 鲜嫩多汁，好吃到舔手指！柠香黑胡椒鸡翅
practice: 烤
flavor: 家常味
time: 60分钟
grade: 新手尝试
materials: 主料鸡翅12只辅料柠檬 一只米醋1勺蜂蜜 3勺生抽2勺蚝油 1勺料酒2勺葱段一把姜片一把迷迭香5根黑胡椒量白芝麻一把蜂蜜水 适量
step: <a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162244320763477.jpg>第1步: 材料准备，鸡翅清洗干净。柠檬表皮用盐搓洗干净。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162244322955274.jpg>第2步: 鸡翅正反两面各划2道，用厨房纸巾擦干表面。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162244323760701.jpg>第3步: 放入干净碗中，加葱段、姜片、蜂蜜、料酒、醋、生抽、蚝油，一半的迷迭香。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162244325775469.jpg>第4步: 磨上适量黑胡椒。带手套抓匀腌制2小时以上。（建议装入密封袋冷藏过夜，更入味）</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162244330259920.jpg>第5步: 柠檬切片，去籽。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162244331073795.jpg>第6步: 预热烤箱上下210度。金盘垫锡纸，放烤架。烤架上可刷一层薄薄的玉米油，再把鸡翅整齐排列，这样鸡翅皮不会被粘掉。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162244332372620.jpg>第7步: 放入预热好的烤箱中层偏下位置，210度烤30分钟。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162244335012278.jpg>第8步: 烤到15分钟，暂停取出，翻面，刷一层蜂蜜水（蜂蜜2勺加水1勺调匀）。继续放回去烤。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162244335704350.jpg>第9步: 烤完取出，装盘，磨上适量黑胡椒，撒上一把白芝麻，装饰2根迷迭香！</a>
