recipe_name: 酱烧鱼
practice: 烧
flavor: 酱香味
time: 15分钟
grade: 中级掌勺
materials: 主料鲈鱼一条大蒜一个生姜一块生抽两勺料酒两勺辅料葱一根红烧酱油一勺豆瓣酱一勺盐适量油适量香菜一根小红尖椒两个
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161646425698013.jpg>第1步: 将鱼清洗干净，鱼鳍鱼尾剪掉，身体两边都斜斜地打几个花刀便于入味。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161646425784351.jpg>第2步: 淋入生抽/料酒。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161646425785409.jpg>第3步: 加入几片姜片和小红尖椒段，再撒入少许盐，用手抹匀，刀缝里也要抹，腌制半个小时左右，中间翻动几次，让味道更均匀。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161646425800545.jpg>第4步: 研制好的鱼用厨房纸将表面的水分稍微吸一下，保持鱼身的干爽，煎的时候就不会粘锅了。油锅烧热，油热后撒入少许盐，将腌制好的鱼放入锅中，用小火煎至底部微微焦黄，可以轻轻挪动的时候，翻面，两面都煎好。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161646425788614.jpg>第5步: 然后将鱼轻轻推向锅边。放入葱白段/姜末/蒜末和剩余的小红尖椒段，翻炒出香味。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161646425811965.jpg>第6步: 加入没过鱼身的热水（一定要加热水）</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161646425866079.jpg>第7步: 加入两勺红烧酱油，</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161646425809911.jpg>第8步: 一勺豆瓣酱</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161646425801618.jpg>第9步: 大火煮开后，转小火焖煮15分钟。焖至汤汁微微收干的时候，撒上葱花，摆上香菜装饰，这道非常好吃的酱焖鲈鱼就做好了。</a>
