recipe_name: #年年有鱼#白菜炖鱼
practice: 煮
flavor: 家常味
time: 30分钟
grade: 新手尝试
materials: 主料青鱼500克大白菜500克小葱5克生姜2片辅料豆瓣酱2勺水100毫升味达美臻品料酒2勺
step: <a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908147952.jpg>第1步: 准备食材：白菜洗净四层小碎片，青鱼洗净切块</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908453061.jpg>第2步: 碗内先放入2勺豆瓣酱</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908519453.jpg>第3步: 加入适量的清水（100毫升）</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908316354.jpg>第4步: 搅拌均匀，将豆瓣酱稀释，这样可以更好的入味</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908488532.jpg>第5步: 锅底先铺上白菜垫底</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908527587.jpg>第6步: 将青鱼块放在上面</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908531913.jpg>第7步: 放上两片姜片</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908511044.jpg>第8步: 倒入2勺味达美臻品料酒</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908500279.jpg>第9步: 倒入事先稀释的豆瓣酱</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908550305.jpg>第10步: 盖上盖子大火炖煮开后，炖煮10分钟</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908431414.jpg>第11步: 打开锅盖，此时的白菜开始有些瘪下去，继续炖煮，让汁水可以收干，白菜和鱼也会更入味</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908560493.jpg>第12步: 最后放上小葱即可出锅</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164092908611576.jpg>第13步: 这道有好头彩又营养的健康菜无油烹饪可以让我们减少油脂的摄入，吃得更健康</a>
