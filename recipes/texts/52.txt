recipe_name: 砂锅红烧肉
practice: 煮
flavor: 咸鲜味
time: 60分钟
grade: 初级入门
materials: 主料五花肉适量冰糖一把辅料盐少许老抽一勺生抽一勺蚝油一勺料酒一勺姜一块
step: <a href=https://st-cn.meishij.net/rs/246/212/8615746/n8615746_164510944292541.jpg>第1步: 五花肉无需去皮，切成大小约1.5到2厘米见方的块状。</a>
<a href=https://st-cn.meishij.net/rs/246/212/8615746/n8615746_164510944257101.jpg>第2步: 切好的五花肉放入砂锅中，加入姜片，适量的凉水没过肉。</a>
<a href=https://st-cn.meishij.net/rs/246/212/8615746/n8615746_164510944228797.jpg>第3步: 大火把水烧开，水开后马上关火，这样做能焯去猪肉的腥味和血水，把焯好水的猪肉用温水冲洗干净。</a>
<a href=https://st-cn.meishij.net/rs/246/212/8615746/n8615746_164510944241286.jpg>第4步: 把砂锅洗干净，烧干水分，加入少许油做引子，油烧热以后下入控干水的猪肉，煸炒出猪肉中的油脂。</a>
<a href=https://st-cn.meishij.net/rs/246/212/8615746/n8615746_164510944237380.jpg>第5步: 然后加入老抽，生抽，蚝油，料酒，冰糖，少许盐，加入适量的清水没过猪肉，老抽和冰糖一定要加，这样猪肉的味道和颜色一定不会差。</a>
<a href=https://st-cn.meishij.net/rs/246/212/8615746/n8615746_164510944256800.jpg>第6步: 盖上锅盖，大火把水烧开，然后转小火，慢慢煨上一个小时，让猪肉充分入味。</a>
<a href=https://st-cn.meishij.net/rs/246/212/8615746/n8615746_164510944766465.jpg>第7步: 时间到了打开锅盖，大火收汁，这样做出来的汤汁才会浓稠厚重地裹在猪肉上，收汁的时候上色也会更漂亮。</a>
<a href=https://st-cn.meishij.net/rs/246/212/8615746/n8615746_164510944754460.jpg>第8步: 最后装盘出锅即可，原汁原味的家常红烧肉就做好了，也可以根据自己的口味添加一些鹌鹑蛋，鲍鱼之类的食材。</a>
