recipe_name: 菌汤排骨炖冬瓜
practice: 炖
flavor: 咸鲜味
time: 60分钟
grade: 新手尝试
materials: 主料肋排1000克冬瓜700克辅料菌汤料包1袋盐适量胡椒粉2克格琳诺尔有机亚麻籽油10克
step: <a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261615096300.jpg>第1步: 今天的菌汤是用的菌汤料包，里面有：茶树菇，虫草花，鸡枞菌，杏鲍菇，姬松茸，羊肚菌，还有2颗红枣。</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261615083718.jpg>第2步: 把干的食用菌清洗一下，然后用水泡软</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261616569585.jpg>第3步: 肋排凉水入锅，焯烫出血沫</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261616547719.jpg>第4步: 捞出用热水冲洗干净表面的浮沫，然后控水</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261616530631.jpg>第5步: 锅里放格琳诺尔有机亚麻籽油</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261616673462.jpg>第6步: 先放姜片，然后放入排骨炒2分钟左右</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261616403696.jpg>第7步: 放入泡好的食用菌</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261616694758.jpg>第8步: 加开水没过排骨</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261616706282.jpg>第9步: 小火炖40分钟</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261616861357.jpg>第10步: 放入冬瓜块，加盐和胡椒粉调味</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261618374702.jpg>第11步: 再继续炖15分钟即可出锅</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_163261647381783.jpg>第12步: 出锅撒香葱碎即可。“格琳诺尔有机亚麻籽油，宝宝的聪明油”</a>
