recipe_name: 茄子肉包子
practice: 蒸
flavor: 家常味
time: 数小时
grade: 中级掌勺
materials: 主料茄子2个面粉1000克猪肉500克辅料植物油适量盐适量生抽适量蚝油10克酵母10克
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462769481439.jpg>第1步: 主料：面粉500克、茄子500克、猪肉400克
辅料：香葱5根、酵母5克、生抽20克、蚝油10克、姜10克、五香粉1克、花生油10克</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462769427759.jpg>第2步: 面粉里加酵母（面粉：酵母大约100:1），再加入适量水，搅成雪花状，揉成光滑的面团，盖上盖子或者盖上湿纱布，在常温下发酵。两倍大以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462769400123.jpg>第3步: 把肥瘦相宜的猪肉，切大块，香葱洗净切段、姜切片</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462769522980.jpg>第4步: 一起加绞肉机里，绞成肉泥，不用绞的太细了，包子馅太细了，没有口感了</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462769544965.jpg>第5步: 把它们放在大碗里，加生抽、蚝油、五香粉拌均匀，静置一会入味（不用打水了，因为后面的茄子出水分很多，可以综合味道）</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462769630989.jpg>第6步: 把茄子去皮切丁丁，不易太小了，就这样不错，加盐拌均匀，软一些了就倒进肉馅里，这样好拌均匀一些</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462769601894.jpg>第7步: 加熟的菜籽油或者花生油、鸡精拌均匀，静置一会入味，这样包出来的包子味道浓郁好吃</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462769696091.jpg>第8步: 发好的面团取出（此时面团大约是原来的两倍大以上），手指沾上面粉，从面团中间插进去，插出的坑不会回缩则发酵好，加少许的面粉揉均匀，这样可以排到空气，这样蒸出来的包子不回缩，分成小剂子。擀成中间厚边缘薄的面皮</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462769661044.jpg>第9步: 加馅，不要加的太多了，八分满即可，这样蒸出来的包子不易露馅，也好包。</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462769700120.jpg>第10步: 捏褶收口，包成自己喜欢的包子形状即可。</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462769783954.jpg>第11步: 包好以后放入蒸翕里，盖上盖子，等二次发酵15分钟左右，待包子二次发酵成胖乎乎的就说明已经发酵好了</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_162462786148748.jpg>第12步: 蒸锅加水，大火烧开以后，加包子，大火蒸20分钟左右，具体看包子的数量确定时候
时间到后先不要着急揭盖，焖3-5分钟左右再打开，又白又鲜美的包子就好了，可以做早餐和主食，是四季可吃的包子。</a>
