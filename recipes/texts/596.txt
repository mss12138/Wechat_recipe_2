recipe_name: 酱香烤鸡翅
practice: 烘焙
flavor: 酱香味
time: 60分钟
grade: 初级入门
materials: 主料鸡翅中8个辅料小葱（切段）2棵生姜（切丝）3片盐少许白砂糖1小勺六月香豆瓣酱1.5大勺梅林番茄沙司3大勺味达美臻品料酒1大勺
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162565750947506.jpg>第1步: 准备好鸡翅、番茄沙司、料酒、豆瓣酱、葱姜和盐糖。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162565751024963.jpg>第2步: 鸡翅正面割两刀、反面割三刀，方便入味</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162565750999515.jpg>第3步: 加入少许盐、糖、1大勺味达美臻品料酒。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162565751071437.jpg>第4步: 一勺半六月香豆瓣酱。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162565751005086.jpg>第5步: 加入3大勺番茄沙司，加葱段和姜丝去腥。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162565751013198.jpg>第6步: 抓匀，使每个鸡翅都裹上酱汁。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162565751047276.jpg>第7步: 用保鲜膜密封，冷藏腌制过夜。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162565751035062.jpg>第8步: 烤盘垫锡纸，烤架上均匀摆好鸡翅。为了上色更好，刷一层麦芽糖水。也可以刷蜂蜜水，没有可不刷。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162565751076055.jpg>第9步: 烤箱上下火200℃预热，中层烤20分钟，取出翻面并刷麦芽糖水，继续烤15分钟。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162565751185943.jpg>第10步: 出炉啦，诱人不？</a>
