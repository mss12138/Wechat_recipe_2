recipe_name: 懒人版鱼香肉丝
practice: 炒
flavor: 酸甜味
time: 10分钟
grade: 新手尝试
materials: 主料瘦肉丝150克盐1克黑胡椒适量料酒1勺蛋清1个淀粉适量黑木耳100克胡萝卜50克鱼香肉丝酱2大勺辅料白砂糖5克香醋5克
step: <a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393341823959.jpg>第1步: 瘦肉丝150克用，盐1克、黑胡椒适量、料酒1勺、蛋清1个、淀粉适量，放一起腌制一下。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393341847412.jpg>第2步: 拌匀后腌制10分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393341860978.jpg>第3步: 黑木耳100克，用手掰成小块，胡萝卜50克切丝备用。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393341818221.jpg>第4步: 腌制好的肉丝，过油炒一下，炒至颜色变白，即可捞出备用。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393341869659.jpg>第5步: 肉丝里的油倒回锅里。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393341866319.jpg>第6步: 再倒一点油。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393341907125.jpg>第7步: 先放入胡萝卜丝，稍微炒一会，让胡萝卜丝变软，再加入准备好的黑木耳。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393341905998.jpg>第8步: 加入两勺鱼香肉丝酱2大勺，翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393341926735.jpg>第9步: 加入香醋5克。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393341964089.jpg>第10步: 白砂糖5克，翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393342066276.jpg>第11步: 倒入炒好的肉丝。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393342031924.jpg>第12步: 翻炒均匀即可。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393342031803.jpg>第13步: 懒人版鱼香肉丝，完成</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162393342042821.jpg>第14步: 酸甜微辣的鱼香肉丝，配一碗香喷喷的白米饭，完美O(∩_∩)O</a>
