recipe_name: 蒸槐花【孔老师教做菜】
practice: 蒸
flavor: 家常味
time: 5分钟
grade: 新手尝试
materials: 主料槐花400克面粉120克辅料盐适量大蒜适量红油辣子适量
step: <a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162198882522935.gif>第1步: 槐花除花梗，留下花苞和花朵，这一步需要一些耐心。用清水把摘好的槐花清洗干净，最好多清洗几遍。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162198882524458.gif>第2步: 捞出槐花，沥干水分。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162198889026083.gif>第3步: 在沥干的槐花中加入面粉。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162198889513791.gif>第4步: 然后用手轻轻的拌均，使槐花上面均匀的沾上一层面粉，如果感觉面粉不够可以再逐量加入。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162198890166468.gif>第5步: 将拌好面的槐花放入蒸盘。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162198890789315.gif>第6步: 蒸烤箱时间设定20分钟。（没有蒸烤箱可用蒸锅，水开后蒸15分钟即可。）</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162198891545902.gif>第7步: 蒸好的槐花倒入盆中，准备拌入调味料。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162198892076654.gif>第8步: 将蒜头切碎，放入盆中。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162198882877746.gif>第9步: 再调入盐和红油辣子。调味料可以根据自己的喜好去添加。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162198892894751.gif>第10步: 将调味料拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162198893378225.gif>第11步: 倒出装盘，槐花的清香朴面而来，一碗都不过瘾图片。又到槐花飘香的时节，来一份蒸槐花，重温儿时的味道，开蒸吧！</a>
