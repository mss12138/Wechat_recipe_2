recipe_name: 粽香糯米鸡【宝宝辅食】
practice: 煮
flavor: 家常味
time: 10分钟
grade: 新手尝试
materials: 主料糯米饭200克鸡腿肉120克豌豆30克干贝丁8克辅料淀粉5克香菇1个油、酱油、素蚝油、鲜菇粉、粽子叶适量
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454278988.jpg>第1步: · 参考月龄 ·
24个月以上，对食材不过敏的宝宝
· 食材准备 ·
糯米饭200克 / 鸡腿肉120克 / 豌豆30克
豌豆30克 / 干贝丁8克 / 淀粉5克 / 香菇1个
辅料：油、酱油、素蚝油、鲜菇粉、粽子叶</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454379774.gif>第2步: 干贝丁倒入干净大碗中，放入葱段、姜片和适量清水，提前泡发1夜的时间。现在天气越来越热了，建议放到冰箱里冷藏泡发。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454379207.gif>第3步: 鸡腿肉切成小丁，加入酱油、素蚝油、鲜菇粉和淀粉。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454375912.gif>第4步: 充分抓拌鸡腿丁，腌制15分钟，让鸡腿丁更加入味。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454351091.gif>第5步: 豌豆和香菇焯水。豌豆和香菇的组合是比较经典的了，如果家里有别的食材也可以加上去。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454377580.gif>第6步: 香菇切成小丁，如果家里的宝宝月龄比较小的话，为了方便咀嚼消化，可以把豌豆表面的外皮剥除。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454397635.gif>第7步: 泡好的干贝丁撕成小丝。干贝的提鲜效果很好，如果你家的干贝吃没了，换成海米、虾皮这样的同类食材，也可以。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454433809.gif>第8步: 锅中倒油，放入鸡腿肉，翻炒到变色。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454474989.gif>第9步: 加入豌豆、香菇和干贝，继续翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454485894.gif>第10步: 加入糯米饭，再次翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454532365.gif>第11步: 取一个干净的大碗，在碗中摆入粽子叶，炒好的糯米鸡肉放在粽叶上。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454590229.gif>第12步: 收拢粽叶，放入盘子里。上锅，中火蒸制20分钟。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454504185.gif>第13步: 熟透开盖，把粽叶去掉。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162348454570016.jpg>第14步: 揭开粽叶，一股热气带着粽叶清香和肉香扑面而来，软糯的口感，真的太喜欢了。

再多说一句啊，粽香糯米饭虽然好吃，但宝宝少吃几口，感受下节日氛围就好了，剩下的就全家分享吧，来个美味的端午限定！</a>
