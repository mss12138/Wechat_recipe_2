recipe_name: 早餐饼之煎蛋全麦薄饼
practice: 烙
flavor: 家常味
time: 10分钟
grade: 初级入门
materials: 主料全麦面粉100克中筋面粉100克鸡蛋适量辅料室温凉水130克生菜适量植物油少许番茄沙司适量蒜茸辣酱适量
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162547737319974.jpg>第1步: 全麦面粉和中筋面粉入盆，先混合均匀；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162547737378850.jpg>第2步: 逐量倒入室温凉水，搅拌成絮状，再手揉成较光滑的面团；面团宜软，水量是粉量的65%左右；蒙保鲜膜饧15分钟左右；如能提前一晚揉好放冰箱冷藏，第二天使用时更加滋润柔软；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162547737327763.jpg>第3步: 饧面时煎蛋、洗菜；不粘锅中倒几滴油，鸡蛋依次入锅，中火煎至底面定型、微黄；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162547737431693.jpg>第4步: 翻面煎至两面微黄，是否溏心随喜好；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162547737423846.jpg>第5步: 面团分均等份，面剂子大小根据锅的大小调整，因为要快速烙饼，所以剂子不宜大；擀成锅底大小的饼坯，厚度约1-2毫米；暂时不用的面团要用保鲜膜蒙住，防水分蒸发；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162547737412440.jpg>第6步: 不粘锅中无需倒油，将饼坯入锅，烙15秒定型，翻身烙，盖上盖子焖烙1分钟左右，两面微黄，饼全熟，倒出后接着烙下一张；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162547737473363.jpg>第7步: 取一张饼，上铺生菜、煎蛋、蒜茸辣酱；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162547737441975.jpg>第8步: 卷成卷，为防止卷皮松开，可提前裁剪烘焙油纸将其卷起来；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162547737426493.jpg>第9步: 换一种口味，将蒜茸辣酱换成番茄沙司，酸酸甜甜的适合小朋友及怕咸怕辣者；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162547737451496.jpg>第10步: 卷成卷，用烘焙油纸拦腰缠起来。</a>
