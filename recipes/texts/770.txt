recipe_name: 蒜薹炒小河虾
practice: 炒
flavor: 家常味
time: 10分钟
grade: 初中水平
materials: 主料蒜薹200克新鲜小河虾400克辅料植物油适量盐适量生抽5克料酒5克姜5克辣皮子4个
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_161857472631208.jpg>第1步: 主料：蒜薹200克、小河虾400克
辅料：植物油、盐、姜、生抽3克、辣皮子2个（怕辣的，可以加青辣椒）、啤酒5克（或者料酒）</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_161857472603224.jpg>第2步: 把蒜薹洗净去掉吗头尾切段，姜切馍馍，辣皮子泡一泡切开，这样辣味浓</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_161857472679885.jpg>第3步: 把河虾洗洗沥干水分，（把小河虾里面身上的脏东西还有泥沙全部清洗出来）。多清洗几遍之后看水清了就可以了。</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_161857472772424.jpg>第4步: 热锅凉油，油热7成，下姜末、辣皮子翻炒出香味出辣味</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_161857472785487.jpg>第5步: 加河虾翻炒均匀出香味，出水分变色，看着红彤彤的</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_161857472790516.jpg>第6步: 加点料酒翻炒均匀出香味，虾油的香味也出来了</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_161857472852706.jpg>第7步: 加生抽翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_161857472845249.jpg>第8步: 把河虾拨在一边，加蒜薹段翻炒均匀，出蒜香味以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_161857472834539.jpg>第9步: 加盐翻炒均匀，再把河虾一起翻炒均匀，多翻炒几下，这样河虾和蒜薹融合在一起味道好</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_161857472835035.jpg>第10步: 加味精或者蔬之鲜</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_161857472802360.jpg>第11步: 翻炒均匀出锅上桌，河虾我认为不用加水，直接干炒最鲜美</a>
