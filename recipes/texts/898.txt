recipe_name: 香煎寸骨
practice: 煎
flavor: 咸鲜味
time: 数小时
grade: 初级入门
materials: 主料寸骨6根辅料食盐适量青花椒适量香葱适量姜片少许二荆条适量蒜末适量生抽适量老抽适量孜然粉少许白胡椒粉少许辣椒粉少许植物油适量
step: <a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162098016322711.jpg>第1步: 先准备腌制寸骨用的花椒盐：将炒锅烧热后，放入足量的食盐、适量青花椒，中小火翻炒至盐变黄；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162098016309187.jpg>第2步: 将花椒盐略微放凉后倒入洗净的寸骨中，充分抓匀；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162098016374481.jpg>第3步: 盖上保鲜膜，放入冰箱冷藏腌制半天以上；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162098016343958.jpg>第4步: 腌制结束后，不用清洗，直接放入开水锅中出下水，再捞出清洗干净；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162098016359267.jpg>第5步: 另起一锅，放入足量的清水，加入葱段、姜片，放入处理好的寸骨，撒1小勺食盐，大火烧开后转小火慢炖一个小时左右，只要筷子能轻松戳进去就行，将煮好的寸骨捞出沥干水分；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162098016315081.jpg>第6步: 不粘锅烧热后，倒入适量植物油，放入寸骨两面煎至金黄，（因为寸骨已煮熟，这时火候可以开大些，主要把寸骨表面煎黄即可），途中撒些蒜末；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162098016618578.jpg>第7步: 淋上生抽、老抽翻炒入味；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162098016667928.jpg>第8步: 出锅前撒青红椒、葱花；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162098016604986.jpg>第9步: 最后撒孜然粉、白胡椒粉、辣椒粉，外焦内嫩的寸骨就做好啦！</a>
