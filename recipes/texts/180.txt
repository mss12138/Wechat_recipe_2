recipe_name: 香煎小黄鱼
practice: 煎
flavor: 咸鲜味
time: 15分钟
grade: 初中水平
materials: 主料小黄鱼12条辅料小葱适量生姜适量白砂糖少许味达美 臻品料酒2大勺六月鲜 12克轻盐特级昆布酱油2大勺黑胡椒粉少许
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163600482678904.jpg>第1步: 准备好小黄鱼、葱姜和调味料。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163600482516941.jpg>第2步: 倒入少许白砂糖、2大勺料酒、2大勺生抽、适量葱姜，洒点黑胡椒粉，抓匀后加盖腌制1小时左右。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163600482802675.jpg>第3步: 腌制后的小黄鱼排入通风透气的篮子，在阴凉处风干鱼皮表面。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163600482561326.jpg>第4步: 或者用厨房纸巾吸干鱼皮表面水分。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163600482756073.jpg>第5步: 热锅下多点油，下葱姜熬个葱油，葱姜微焦时捞出丢弃。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163600482587362.jpg>第6步: 轻轻放入小黄鱼，先不要动它们，煎到轻轻晃动锅子，小黄鱼能在锅里滑动。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163600482676325.jpg>第7步: 给小黄鱼翻面，同样煎到晃动锅子能滑动，煎到两面金黄就可以出锅了。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163600482797894.jpg>第8步: 垫上吸油纸，撒上葱花就可以美美地上桌啦！</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163600482895189.jpg>第9步: 皮脆脆的，肉嫩嫩的，香香的，太好吃了！</a>
