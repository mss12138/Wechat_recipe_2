recipe_name: 脆皮脆皮鸡
practice: 烤
flavor: 咸鲜味
time: 60分钟
grade: 初中水平
materials: 主料小鸡1k克辅料生抽30克蚝油20克番茄酱20克细砂糖20克料酒20克甜椒粉10克蒜泥10克软化的黄油适量
step: <a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330161625152.jpg>第1步: 准备好所有食材，黄油是烤鸡之前才需要用到的，所以暂时没有入镜
ps：鸡挑选小一点的，两斤左右的比较合适，太小了也没肉，大了口感会差一些~
甜椒粉不辣，主要是为了提供好看的色泽</a>
<a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330161677620.jpg>第2步: 准备一只干净的容器</a>
<a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330161717734.jpg>第3步: 将除黄油以外的所有辅料都放入容器内，搅拌均匀</a>
<a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330161087234.jpg>第4步: 小嫩鸡洗干净，擦干表面的水分，放入容器内，均匀的抹上腌料</a>
<a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330161853935.jpg>第5步: 找一个干净的密封袋，将抹上腌料的鸡放入袋中，将容器内的腌料倒入密封袋内</a>
<a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330161887367.jpg>第6步: 密封好，放入冰箱冷藏24小时以上</a>
<a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330161889301.jpg>第7步: 准备一个烤盘，铺好锡纸，垫上烤网，准备一副干净的一次性手套，和提前软化的黄油</a>
<a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330162366285.jpg>第8步: 将腌制好的鸡从袋中取出，放在烤网上，平铺，用手稍稍用力压一压
这一步可以提前进行，将腌制好的鸡提前取出，风干表皮，或者用你能想到的一切方法，弄干表皮，为下一步做准备</a>
<a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330162462680.jpg>第9步: 将鸡翅用锡纸包裹住，用提前软化好的黄油，均匀的涂抹在鸡的表面</a>
<a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330163026725.jpg>第10步: 就像这样~
可以根据个人喜好，撒现磨的黑胡椒，因为要给孩子吃，我就没撒了，撒了一点干罗勒叶，也很香</a>
<a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330162609807.jpg>第11步: 烤箱提前预热</a>
<a href=https://st-cn.meishij.net/rs/39/167/4416789/n4416789_162330162933994.jpg>第12步: 上下火180℃，30分钟左右，然后转上下火，200℃，10分钟左右，上色</a>
