recipe_name: 鸡肉饭团串串【宝宝辅食】
practice: 烙
flavor: 家常味
time: 10分钟
grade: 新手尝试
materials: 主料鸡胸肉200克米饭200克油麦菜30克胡萝卜20克辅料素蚝油1勺番茄酱2勺淀粉5克清水200克
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250423445.jpg>第1步: · 参考月龄 ·
12个月以上，对食材不过敏的宝宝
· 食材准备 ·
鸡胸肉200克
米饭200克 / 油麦菜30克 / 胡萝卜20克
酱汁：1勺素蚝油 / 2勺番茄酱
5克淀粉 / 200克清水</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250522435.gif>第2步: 洗干净的胡萝卜、油麦菜切碎。这个添加的蔬菜都是可以随便换的，家里有啥用上就可以了。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250559342.gif>第3步: 切好的胡萝卜和油麦菜拌入米饭当中，加上点鲜菇粉调味，这是我们家不能缺的调味料，搅拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250586871.gif>第4步: 双手抹上水</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250516113.gif>第5步: 取适量的米饭团成小球，放在一边备用。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250538258.gif>第6步: 鸡胸肉切丁，我想要让它成为泥状，就放进了料理机当中，搅打成泥。如果家里没有料理机的姐妹，也可以剁成肉泥，注意要尽量的细腻一些。

*鸡肉、猪肉或者是牛肉什么都可以用上，主要看家里的现有食材。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250673855.gif>第7步: 取适量的鸡肉泥整理成肉饼皮的样子，怕粘手的话，可以事先在手上抹点油。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250848166.gif>第8步: 之前准备好的米饭团</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250887208.gif>第9步: 放到肉饼中心，收口，揉圆。丸子的大小可以根据宝宝的咀嚼能力去做，月龄小的宝宝可以做得小一点，更好入口。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250983231.gif>第10步: 锅中烧水，水沸后转成小火，把肉丸放到锅中。如果开大火的话，下丸子的时候容易散，这个一定要注意了。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250933730.gif>第11步: 轻轻的搅动丸子</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141250902766.gif>第12步: 等到熟透后捞出。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141251225819.gif>第13步: 锅中加入1勺素蚝油、1勺番茄酱、5克淀粉和200克左右的清水，搅拌均匀。这个酱汁的味道是偏向于酸甜口味，夏季吃也有开胃的效果。而调味料的量都是可以根据自家宝宝的月龄来增减的，浓郁一点的味道，大宝宝更好接受。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141251072691.gif>第14步: 煮好的肉丸放到锅中，翻炒到汤汁逐渐浓稠。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141251041974.gif>第15步: 丸子充分粘上汤汁，用竹签把丸子穿起来。这样吃就很有意思了，不过毕竟是用到了竹签，月龄小的宝宝一定要在家长的看护下食用。如果家里没有竹签的话，就直接盛到盘里，直接作为一道菜就能上桌。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162141251072670.jpg>第16步: 裹满了酱汁的鸡肉饭团看起来实在是太有食欲了，而且从外表你根本发现不了里面竟然还有一层小惊喜，简直就是哄娃吃饭的“利器”！</a>
