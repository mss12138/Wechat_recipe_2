recipe_name: 酱香鸭锁骨
practice: 烧
flavor: 酱香味
time: 15分钟
grade: 初级入门
materials: 主料鸭锁骨750克辅料葱伴侣 6月香豆瓣酱1大勺味达美 臻品料酒2大勺六月鲜 红烧酱油1大勺六月鲜 特级原汁酱油1大勺生姜8片小葱2棵八角2个桂皮1小段
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163228590769914.jpg>第1步: 准备好所有调味料和食材。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163228590798952.jpg>第2步: 鸭锁骨长这样，超市有卖，一斤8块不到。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163228590733067.jpg>第3步: 有些鸭锁骨内侧有气管残留，扯掉它。还有些有鸭油附着，不控油的建议留着，控油或不喜油的可以剪掉它。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163228590773309.jpg>第4步: 整个的鸭锁骨太大了，从中间剪断，方便啃。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163228590740577.jpg>第5步: 冷水下锅，放一棵小葱，三片生姜，加水没过鸭锁骨，煮沸。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163228590763636.jpg>第6步: 冲洗掉血沫。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163228590862842.jpg>第7步: 焯过水的鸭锁骨放入高压锅，加2片姜，1大勺料酒，加盖密封，大火上气后再煮15分钟关火。注意：要等气放光后才能开盖。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163228590864645.jpg>第8步: 炒锅烧热下油，油热下白砂糖炒成焦糖色，如图。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163228590884001.jpg>第9步: 下鸭锁骨翻炒裹上糖色，加一大碗鸭汤或清水、1大勺葱伴侣6月香豆瓣酱、1大勺六月鲜红烧酱油、1大勺六月鲜特级原汁酱油、1大勺味达美臻品料酒、八角桂皮，大火煮沸。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163228590875188.jpg>第10步: 大火收浓汤汁。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163228590934517.jpg>第11步: 成品颜色红亮，酱香浓郁，齿颊留香。</a>
