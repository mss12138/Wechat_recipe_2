recipe_name: 鸡爪鲜虾煲
practice: 炖
flavor: 咸鲜味
time: 30分钟
grade: 新手尝试
materials: 主料鸡爪4个大虾8只土豆1个胡萝卜1根香菇3朵洋葱半个辅料姜片2片蒜瓣3瓣干辣椒1个八角1个香叶1片桂皮1小块冰糖5克食盐2克老抽2勺料酒1勺蚝油1勺
step: <a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214381695065.jpg>第1步: 先把需要的食材提前准备好。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214381893585.jpg>第2步: 干香菇放入小碗中，用温水提前泡发。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214382204920.jpg>第3步: 土豆、胡萝卜去皮切滚刀块。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214382761820.jpg>第4步: 洋葱切丝备用。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214383184284.jpg>第5步: 鸡爪用厨房剪剪掉脚趾甲后再用剪刀从中间剪断，方便入味。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214383551506.jpg>第6步: 锅中放入适量清水，冷水下鸡爪、姜片、料酒焯水。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214383837200.jpg>第7步: 焯好水的鸡爪捞出洗掉表面浮沫后控干水分。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214384365103.jpg>第8步: 锅中倒入少许食用油，放入洋葱、八角、干辣椒、姜片、蒜瓣、香叶、桂皮、冰糖炒出香味。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214385224606.jpg>第9步: 锅中倒入少许食用油，放入洋葱、八角、干辣椒、姜片、蒜瓣、香叶、桂皮、冰糖炒出香味。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214385774111.jpg>第10步: 倒入焯好水是鸡爪翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214386296146.jpg>第11步: 倒入土豆、胡萝卜翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214386642960.jpg>第12步: 2勺老抽、1勺蚝油调色调味，然后加入适量清水，水量没过食材即可，大火烧开后转小火慢慢炖至鸡爪软烂即可。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214386937487.jpg>第13步: 炖鸡爪的时候开始处理大虾，用厨房剪剪掉虾须、虾脚、虾线，用清水冲洗干净。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214387449656.jpg>第14步: 鸡爪炖至差不多时放入处理好的虾，盖上盖子继续烧3分钟左右即可，虾比较容易熟，时间不宜过长，避免影响口感。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163214388309687.jpg>第15步: 格琳诺尔有机亚麻籽油，宝宝的聪明油</a>
