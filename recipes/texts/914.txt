recipe_name: 超简单蚝油生菜，健康清甜，刮油又减肥，
practice: 炒
flavor: 家常味
time: 5分钟
grade: 新手尝试
materials: 主料生菜1颗辅料蚝油1大勺味精小半勺
step: <a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162096111801155.jpg>第1步: 先取一盆清水，加一勺盐，生菜1颗，切掉根部，一片片掰下来。浸泡在10分钟左右，杀杀菌，然后洗干净。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162096111810517.jpg>第2步: 手撕成一块块的备用（刀切也行，整叶炒也行，看心情）。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162096111888572.jpg>第3步: 锅里放适量的油。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162096111855763.jpg>第4步: 沥干水分的生菜，热油下锅，稍微翻炒一下去生。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162096111880302.jpg>第5步: 加入蚝油1大勺。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162096111828112.jpg>第6步: 翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162096111904843.jpg>第7步: 出一点水后加入味精小半勺。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162096111929059.jpg>第8步: 翻炒均匀就好了。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162096111954425.jpg>第9步: 生菜易熟，因为其维生素C含量较高，如果煮久容易发黑。所稍微翻炒一下即可。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162096112052889.jpg>第10步: 健康清甜，刮油又减肥，适合家常制作，非常快手！</a>
