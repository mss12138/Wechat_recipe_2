recipe_name: 红烧鸡翅土豆
practice: 焖
flavor: 家常味
time: 30分钟
grade: 初级入门
materials: 主料鸡翅中600克土豆1个青椒1个红椒半个辅料大蒜1头姜1块油少许盐适量生抽少许酱油适量冰糖8粒料酒30克
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162687183324263.jpg>第1步: 鸡翅中清洗干净；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162687183396974.jpg>第2步: 从中间剁一刀成两半，泡凉水20分钟，将其中的血水泡掉；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162687183380135.jpg>第3步: 土豆削皮，青红椒洗净，大蒜和姜准备好；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162687183344460.jpg>第4步: 土豆切滚刀块，青红椒去蒂去籽切块，姜切厚片；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162687183428141.jpg>第5步: 炒锅中火加热1分钟，倒少许油，不要等油热，将鸡翅、大蒜、姜片倒入锅中翻炒；鸡翅泡过水，所以水分较充足，经过煸炒后慢慢收缩，鸡皮中的部分油脂也跟着出来了；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162687183329037.jpg>第6步: 倒适量料酒翻炒去腥，再把土豆倒入锅中翻炒；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162687183300388.jpg>第7步: 倒适量酱油、生抽、盐、8粒冰糖、1碗水；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162687183498883.jpg>第8步: 待汤汁沸腾后，盖上盖子，转小火焖炖20分钟；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162687183445750.jpg>第9步: 肉烂汤浓，倒入青红椒块，翻炒一分钟变色即出锅。</a>
