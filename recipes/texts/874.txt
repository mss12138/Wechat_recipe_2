recipe_name: 鸡翅这样做，鲜嫩又入味，全家老小都爱吃
practice: 炖
flavor: 咸鲜味
time: 30分钟
grade: 未知
materials: 主料翅中若干辅料姜片若干葱花适量料酒适量冰糖1小把八角2-3个桂皮1个生抽适量老抽适量盐适量白芝麻适量
step: <a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162138995446152.jpg>第1步: 鸡翅洗净两面划刀。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162138995607466.jpg>第2步: 鸡翅冷水下锅，放若干姜片，再倒入适量料酒去腥，大火煮开。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162138995851848.jpg>第3步: 煮开后再煮五六分钟，煮出血沫，然后捞出。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162138996098276.jpg>第4步: 焯煮过的鸡翅过凉水，多冲几遍，冲净血沫等杂质，沥水备用。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162138996188887.jpg>第5步: 起锅烧油，放入冰糖小火炒糖色，炒至冰糖完全融化并呈现冒泡的焦糖色，倒入鸡翅翻炒上色。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162138996302855.jpg>第6步: 炒至鸡翅表皮微焦，放入八角和桂皮。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162138996464669.jpg>第7步: 淋入生抽和老抽，加强上色。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162138996586338.jpg>第8步: 倒入没过食材的温水或热水，加适量盐，大火煮开盖盖转小火炖20-30分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162138996731925.jpg>第9步: 出锅前大火收汁，撒白芝麻增香。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162138996995820.jpg>第10步: 在撒适量葱花装饰一下就可以出锅啦。</a>
