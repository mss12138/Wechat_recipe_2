recipe_name: 三鲜蒸馄饨【宝宝辅食】
practice: 煮
flavor: 家常味
time: 10分钟
grade: 新手尝试
materials: 主料猪肉200克韭菜100克即食玉米粒30克馄饨皮24张辅料紫菜3克香菇2个油、鲜菇粉、酱油、素蚝油、盐、葱花 （3种馅料）适量
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764142269.jpg>第1步: · 参考月龄 ·
12个月以上，对食材不过敏的宝宝
· 食材准备 ·
猪肉200克
韭菜100克 / 即食玉米粒30克
馄饨皮24张 / 紫菜3克 / 香菇2个
辅料：油、鲜菇粉、酱油、素蚝油、盐、葱花
（3种馅料）</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764162849.gif>第2步: 猪肉切成小丁，放入料理机当中，搅打细腻。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764126159.gif>第3步: 搅打好的猪肉分成三份</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764253269.gif>第4步: 分别加入油、酱油、素蚝油、鲜菇粉和盐进行调味。这个调味你们可以根据自己的口味来。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764251556.gif>第5步: 猪肉馅充分搅拌均匀，备用。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764223062.gif>第6步: 香菇焯水，捞出。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764281389.jpg>第7步: 香菇、韭菜切碎。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764343807.gif>第8步: 紫菜用清水浸泡到舒展散开，挤出多余的水分</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764394480.gif>第9步: 同样剁碎。没有紫菜的话，你们就换成虾皮之类的提鲜，搭配点蔬菜啥的，味道也好。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764352045.gif>第10步: 分别在3份肉馅当中加入刚刚处理好的食材</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764366998.gif>第11步: 一份加韭菜，做个韭菜馅的。再一份加上紫菜、葱花，最后一份加上香菇、即食玉米粒、葱花，充分搅拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764443440.gif>第12步: 在馄饨皮的四边抹上清水，把之前准备好的馅料放进去。这里馅料不要塞得太满了，适量就行。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764443971.gif>第13步: 馄饨皮对折，捏紧接口</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764400356.gif>第14步: 再抓住两边的边角对折。我这次做的比较多，是准备冷冻起来保存的。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764405877.gif>第15步: 馄饨上锅蒸15分钟</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764539562.gif>第16步: 熟透取出，装盘。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162901764555552.jpg>第17步: 这样做上一次，真是能吃很久，如果没想好怎么搭配，就直接参考我今天这三种，你们还有什么好吃的馅料？也欢迎来留言区一起分享啊图片</a>
