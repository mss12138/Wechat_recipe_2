recipe_name: 泰式甜辣豆腐
practice: 煎
flavor: 酸甜味
time: 10分钟
grade: 新手尝试
materials: 主料老豆腐4块辅料食用油2汤匙香葱少许老抽1小勺清水小半碗柠檬酸辣汁1汤匙泰式甜辣酱2汤匙
step: <a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164559678066887.jpg>第1步: 准备豆腐、柠檬酸辣汁和泰式甜辣酱</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164559677732648.jpg>第2步: 豆腐洗净切大块；葱白切段，葱绿切末；我选的是不老不嫩的豆腐，比老豆腐嫩滑且豆腥味少，但比嫩豆腐又要结实些，不至于一碰就碎，操作起来也更得心应手些</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164559678148292.jpg>第3步: 在做豆腐前我们先来调一个酱汁：碗中加入1小勺老抽和1汤匙柠檬酸辣汁；加入少许的老抽，主要为了上色更好看</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164559678138774.jpg>第4步: 接着再调入2汤匙泰式甜辣酱</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164559678154456.jpg>第5步: 然后加入小半碗清水和2小勺生粉，搅拌均匀，放一旁备用</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164559678037609.jpg>第6步: 平底锅烧热，倒入2汤匙食用油，先下入葱白爆香，再把切好的豆腐放进去</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164559678122562.jpg>第7步: 中火煎至底面金黄后，翻面，继续</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164559678210680.jpg>第8步: 煎到豆腐两面都变成金黄焦香，像这样就可以了</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164559678348639.jpg>第9步: 这时把事先调好的酱汁倒入锅中，盖上锅盖煮开后小火焖1分钟左右</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164559678556604.jpg>第10步: 待豆腐焖至上色且入味，最后转大火收浓汤汁；但切记不要收得太干，留点卤汁淋在豆腐上面，吃起来味道更佳</a>
