recipe_name: 蒜蓉蒸鸡蛋羹
practice: 蒸
flavor: 蒜香味
time: 10分钟
grade: 新手尝试
materials: 主料金针菇一把蒜一头辅料油适量盐适量生抽适量芝麻油适量葱花适量
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581435512424.jpg>第1步: 金针菇先别切，直接将头部放在水龙头下冲洗干净，尾部先不切，金针菇就不会散开，摆盘更漂亮。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581435464544.jpg>第2步: 将清洗好的金针菇切掉老根。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581435508039.jpg>第3步: 稍微挤干水份后直接摆放在盘子中，尽量将金针菇散开会，摆盘更漂亮。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581435574890.jpg>第4步: 接着制作蒜蓉酱:
剥蒜，多剥点，我这是一包金针菇配一头蒜，剥好后切成蒜末，别太碎，粗着点，有口感~更香。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581435556907.jpg>第5步: 炒锅里倒入更蒜末差不多量的油。烧热后关火。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581435521054.jpg>第6步: 下入蒜末，炒均匀，一定记得，用锅子的余温爆香蒜末，避免过度加热。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581435694463.jpg>第7步: 然后直接在锅里加盐调味，蒜蓉酱就完成了。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581435674228.jpg>第8步: 将蒜蓉酱直接淋在洗好的金针菇上，放入蒸锅，大火蒸6-8分钟后取出。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581435624448.jpg>第9步: 金针菇会自然蒸出水份，蒸出的汁水很甜，不要浪费。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581435624011.jpg>第10步: 将汁水倒出，在汁水中加入鲜味生抽和麻油，调好味道，不必太咸。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581435741724.jpg>第11步: 将汤汁再倒回金针菇盘子里面，最后撒上葱花，就完成了。</a>
