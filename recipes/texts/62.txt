recipe_name: 宝宝纯素四喜丸子【宝宝辅食】
practice: 蒸
flavor: 咸鲜味
time: 30分钟
grade: 新手尝试
materials: 主料豆腐180克山药60克香菇2个淀粉30克胡萝卜15克油菜7棵辅料油适量酱油适量葱花适量盐适量鲜菇粉适量
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315482455862.gif>第1步: 香菇、油菜焯水，捞出。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315483056179.jpg>第2步: 香菇、胡萝卜还有山药切丁，备用。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315486292449.gif>第3步: 豆腐碾碎，留着备用，我用的是老豆腐，它的水分比较少，粘合性也更好，我更推荐你们用它。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315486860080.gif>第4步: 不粘锅倒油，加入葱花煸香。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315489540911.gif>第5步: 锅中放入山药、胡萝卜还有香菇，翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315490338391.gif>第6步: 加入1勺酱油，适量鲜菇粉和盐来调味。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315493193374.gif>第7步: 炒好的杂蔬倒入豆腐碎当中，搅拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315493762129.gif>第8步: 碗里加上淀粉，再次拌匀了。这里淀粉的量，你们就根据实际的水分来，能成型成团就行。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315496623075.gif>第9步: 取适量的豆腐泥，揉圆备用。这些量我一共做了5个大丸子，你们就根据想要的大小做就行。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315497377384.gif>第10步: 丸子摆入盘中，冷水上锅蒸，我是蒸了15分钟，你们自己做的时候根据丸子的大小，调整一下火候时间，以熟透为准。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315499679441.gif>第11步: 这时候我们可以来做一个酱汁，不粘锅里加上2勺酱油、2勺番茄酱、4克的淀粉和大约80克的清水，搅拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315500583843.gif>第12步: 大火熬汁儿，等到汤汁浓稠就能盛出来。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315503293870.gif>第13步: 盘中摆上丸子，之前焯好的油菜在四周做装饰，最后淋入酱汁，这个纯素版的四喜丸子就能上桌了。</a>
