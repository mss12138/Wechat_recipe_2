recipe_name: 爱心咖喱面包鸡
practice: 烘焙
flavor: 咖喱味
time: 10分钟
grade: 初级入门
materials: 主料切片吐司10片鸡胸肉200克土豆80克胡萝卜60克洋葱50克辅料鸡蛋2个牛奶30克面包糠适量味好美咖喱酱100克盐少许糖1大勺淀粉1大勺料酒1大勺
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606880139123.jpg>第1步: 准备好所有食材。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606880264030.jpg>第2步: 取一片吐司，切去四边，用擀面杖擀平擀紧实。擀完所有的面包片，每2片吐司片组合成一个面包鸡。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606880284340.jpg>第3步: 鸡胸肉切成约1.5厘米见方的肉丁，加入1大勺料酒、少许盐和1大勺糖，搅拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606880155434.jpg>第4步: 胡萝卜、土豆、洋葱全部切丁并分别煮熟；鸡胸肉丁等水开后入锅，盖盖煮沸即可捞出，千万别煮老了；所有的丁类一起放入大碗混合。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606880281761.jpg>第5步: 挤入约100克的味好美咖喱酱，此酱咖喱味道浓郁，以我的口味来说略咸。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606880284200.jpg>第6步: 搅拌均匀。此时咖喱鸡肉馅比较湿，如果用锡纸包裹的话倒可以做成爆浆面包鸡，但我不打算用锡纸，得让它粘稠些以免漏浆。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606880285064.jpg>第7步: 咖喱鸡肉馅倒入不粘锅，用少许水调开干淀粉成水淀粉，淋入鸡肉馅，开小火边煮边搅拌，直到鸡肉馅粘稠。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606880247306.jpg>第8步: 两个鸡蛋打入碗中，加30克牛奶，打散备用。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606880351188.jpg>第9步: 取一片吐司片，中间舀上一大勺咖喱鸡肉馅，上面再盖一片吐司片。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606880359669.jpg>第10步: 用心形模具压出形状，去掉多余的吐司片边角。没有模具的请看小贴士。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606886327773.jpg>第11步: 做好的面包鸡生胚放入蛋液中，再面都裹上蛋液。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606887097558.jpg>第12步: 再两面裹上面包糠。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606887727631.jpg>第13步: 面包鸡表面刷油，烤箱上下火180℃预热，中上层烤10分钟左右，上色即可。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606888452129.jpg>第14步: 或者锅中倒入半锅油，加热到180℃左右，下面包鸡炸至两面金黄即可，出锅用厨房纸巾吸油。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162606889498337.jpg>第15步: 面包皮表面脆脆的，里面软软的，鸡肉馅咖喱香味浓郁、口感丰富，作为早餐，又美味又营养。</a>
