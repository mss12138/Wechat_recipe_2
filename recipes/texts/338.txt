recipe_name: 在家也成做出超级美味的大盘鸡，过程简单
practice: 炒
flavor: 咸鲜味
time: 30分钟
grade: 新手尝试
materials: 主料西红柿1个土豆1个胡萝卜半个彩椒（黄、红）各半个鸡腿2个面条150克辅料洋葱半个姜片2片蒜瓣3瓣干辣椒2个香叶1片桂皮1小块八角1个冰糖 10克老抽1勺蚝油1勺料酒1勺葱伴侣6月香豆瓣酱1勺
step: <a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179544418479.jpg>第1步: 先把需要的食材提前准备好。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179561056177.jpg>第2步: 胡萝卜、土豆去皮洗净切滚到块。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179562373570.jpg>第3步: 彩椒掰成小块备用， 没有彩椒的话可以用青椒代替，主要用来配色。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179562871808.jpg>第4步: 洋葱切小块备用。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179545293869.jpg>第5步: 西红柿顶部划十字，放开水中烫一下，然后去掉西红柿皮，用刀切成小块。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179545119811.jpg>第6步: 2个鸡腿洗净剁成小块用清水反复冲洗几遍。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179545393309.jpg>第7步: 锅中倒入适量清水，放入剁好的鸡块焯水，焯水时加入1勺料酒、2片姜片去腥。 鸡块焯好水后用清水洗掉表面浮沫控干水分。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179567868476.jpg>第8步: 锅中倒入少许格琳诺尔亚麻籽食用油。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179569329874.jpg>第9步: 热锅冷油，倒入处理好的鸡块中小火炒至鸡肉表面略微变色后盛出。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179569835016.jpg>第10步: 重新倒入少许底油放入冰糖炒出琥珀色，然后倒入洋葱、姜片、蒜瓣、干辣椒、桂皮、八角、香味炒出香味。 洋葱不要全部放进去，留一半最后再放增加香味。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179570357878.jpg>第11步: 然后倒入鸡块、西红柿翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179574804663.jpg>第12步: 继续倒入土豆、胡萝卜翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179547442300.jpg>第13步: 1勺葱伴侣6月香豆瓣酱、1勺蚝油、1勺老抽、2克食盐调味。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179577934436.jpg>第14步: 倒入适量热水，水量没过食材即可，小火慢炖30分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179549451578.jpg>第15步: 汤汁浓稠时放入剩下的洋葱、倒入彩椒翻炒均匀即可。 锅里的汤汁多留一些，不要收的太干，汤汁一会用来拌面。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179579752963.jpg>第16步: 收汁的同时，另起锅开始煮面，面条煮熟后过一下冷水防粘。 如果不想自己擀面条可以直接去超市买现成的湿面条，有刀削面的话最好用刀削面。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163179583498733.jpg>第17步: 格琳诺尔有机亚麻籽油，宝宝的聪明油</a>
