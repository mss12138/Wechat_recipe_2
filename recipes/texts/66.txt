recipe_name: #年年有余#酱青鱼
practice: 其它工艺
flavor: 咸鲜味
time: 15分钟
grade: 新手尝试
materials: 主料青鱼500克生姜20克葱白10克辅料六月鲜8克轻盐特级原汁酱油45毫升味达美臻品料酒15毫升
step: <a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352989698374.jpg>第1步: 准备食材：青鱼洗净，切成厚片，生姜切片，料酒和轻盐酱油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990127548.jpg>第2步: 将青鱼块放入密封碗中</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990215106.jpg>第3步: 倒入1勺（15毫升）料酒</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990464614.jpg>第4步: 倒入3勺（45毫升）8克轻盐酱油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990257507.jpg>第5步: 放入2片生姜</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990272331.jpg>第6步: 用生姜将鱼肉全部浸上酱汁，然后盖上密封盖，放入冰箱冷藏72小时</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990387948.jpg>第7步: 腌制好的酱青鱼</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990433142.jpg>第8步: 将腌制好的青鱼放入不粘锅内</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990476168.jpg>第9步: 放入姜片</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990427555.jpg>第10步: 将腌制用的酱汁一起倒入</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990492267.jpg>第11步: 盖上锅盖之后大火煮开后中小火炖煮10分钟</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990477170.jpg>第12步: 将汁水收干即可</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990383413.jpg>第13步: 将青鱼装入盘中</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164352990468935.jpg>第14步: 最后用葱白丝点缀即可</a>
