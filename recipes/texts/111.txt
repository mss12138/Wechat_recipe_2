recipe_name: 椒麻牛杂
practice: 烧
flavor: 麻辣味
time: 15分钟
grade: 新手尝试
materials: 主料牛杂1袋青蒜4根青花椒20粒干辣椒3个青辣椒2个小米椒5个姜适量香菜3根辅料油适量盐1小勺生抽1汤匙
step: <a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050700527819.jpg>第1步: 我买的一袋装的熟牛杂，倒掉里面的液体，剩下的差不多半斤重。清洗下，切成大小差不多的条。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050700800695.jpg>第2步: 干辣椒切段，姜切丝，青蒜、小米椒和青椒斜切。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050700852909.jpg>第3步: 锅烧热，先倒入少量油，下青红椒和蒜白煸炒至断生，然后盛起备用。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050701196383.jpg>第4步: 再倒多一点油，下青花椒、姜丝和干辣椒煸炒几下。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050701418629.jpg>第5步: 倒入牛杂翻炒几下。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050701144526.jpg>第6步: 加适量热水，要没过食材，加2汤匙料酒（我用的是自家做的酒酿）。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050701838525.jpg>第7步: 加1小勺盐拌匀。盖上锅盖，大火烧开转小火炖煮10分钟。中间要翻动一下，防止烧干水。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050701438223.jpg>第8步: 洗几根香菜切段。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050701797165.jpg>第9步: 倒入前面炒好的辣椒和蒜白，再加入蒜叶，翻炒几下。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050701672637.jpg>第10步: 尝下咸淡，根据自己的口味加入适量生抽翻炒拌匀，再煮一两分钟。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050702023224.jpg>第11步: 关火，加入香菜趁热翻拌几下。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164050701869119.jpg>第12步: 起锅装盘，就可以吃了。</a>
