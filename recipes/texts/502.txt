materials: 主料鸡腿1个姜2片小葱1根辅料油、鲜菇粉、酱油、素蚝油适量生菜、米饭适量
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780191876357.jpg>第1步: · 参考月龄 ·
12个月以上，对食材不过敏的宝宝
· 食材准备 ·
鸡腿1个
姜2片 / 小葱1根
辅料：油、鲜菇粉、酱油、素蚝油
一餐搭配：生菜、米饭</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780191869343.gif>第2步: 鸡腿去骨</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780191888098.gif>第3步: 切成小丁。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780191894181.gif>第4步: 鸡腿丁当中倒入1勺酱油、1勺素蚝油、适量的鲜菇粉和盐</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780191953397.gif>第5步: 抓拌均匀，腌制20分钟。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780191872999.gif>第6步: 小葱切段，我是比较喜欢用小葱的，家里没有的话，直接用大葱也一样。切的葱段不要太短，可以切得长一点，这样最后捡葱的时候也比较省劲儿。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780191935399.gif>第7步: 锅中倒油，这个油量是要比平时炒菜多一些的，放入葱段和姜片。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780192028079.gif>第8步: 反复煸炒到葱叶微黄，把葱段和姜片捡出来。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780192075088.gif>第9步: 葱油锅中加入鸡腿肉，翻炒到鸡肉变色。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780192064135.gif>第10步: 盖上锅盖，中小火焖至熟透，我用了大概10几分钟吧，然后开盖。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780192050828.gif>第11步: 鸡肉再翻炒几次，盛入盘中。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162780192074897.jpg>第12步: 这次我搭配的蔬菜也很简单，生菜快速焯下水，再加上点油盐拌匀，就很好吃了。我还提前焖了点二米饭，肉、饭、菜，1口下肚，那叫一个香啊！</a>
