recipe_name: 凉拌鱼腥草
practice: 拌
flavor: 香辣味
time: 10分钟
grade: 中级掌勺
materials: 主料鱼腥草200可蒜薹一把花生一把辅料油适量辣椒粉一勺大蒜三颗生抽两勺米醋一勺葱花适量白芝麻适量
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_158377749894335.jpg>第1步: 这就是鱼腥草，菜市场买回来一般都是一小把一小把的。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_158377750043723.jpg>第2步: 鱼腥草如果买回来不急着吃的话，放在水里泡着，这样做起来也比较好处理，将鱼腥草的根须都去干净，清洗干净后切成小段备用。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_158377750175427.jpg>第3步: 蒜薹也清洗干净，洗切好备用。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_158377750240098.jpg>第4步: 食材都准备好了，我们来准备个底料：蒜剁蒜末、葱切葱花放入碗中，喜欢吃辣的加1勺辣椒面，再加入适量的盐、少许胡椒粉，撒入白芝麻搅拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_158377750540791.jpg>第5步: 然后浇入烧开的热油，激发出葱、蒜和辣椒面的香味。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_158377750677931.jpg>第6步: 加入两勺生抽、一勺糯米醋，拌匀，凉拌汁就做好了，这个凉拌汁比较百搭，用来拌面条或者做凉拌菜都非常棒呢。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_158377750806573.jpg>第7步: 花生米淋上适量油，放入空气炸锅中220度炸8分钟，非常香酥，趁热撒入少许盐。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_158377751107200.jpg>第8步: 鱼腥草和蒜薹段都放入一个稍微大一点的容器里，容器大一点比较好翻拌。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_158377751394788.jpg>第9步: 将准备好的料汁倒进去。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_158377751608175.jpg>第10步: 拌匀后将炸好的花生米放进去拌匀就可以食用了，稍微腌制十分钟口感会更好，如果天气比较热，提前做好放在冰箱里冷藏一段时间更好吃呢。</a>
