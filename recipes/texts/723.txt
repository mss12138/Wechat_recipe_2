recipe_name: 黄焖鸡翅豆腐
practice: 炖
flavor: 家常味
time: 15分钟
grade: 初级入门
materials: 主料鸡翅中13个豆腐1块青椒半个红甜椒半个辅料油少许大蒜1头葱白2棵黄焖鸡调味料适量水适量
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162358948450924.jpg>第1步: 材料准备好：鸡翅泡水20分钟，青红椒去蒂去籽清洗干净；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162358948406286.jpg>第2步: 鸡翅中剁成两段，豆腐切大块，青红椒切滚刀块，葱白切段；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162358948586623.jpg>第3步: 炒锅中倒少许油，葱蒜煸炒出香味；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162358948529670.jpg>第4步: 鸡翅中入锅翻炒变紧致；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162358948509033.jpg>第5步: 倒入适量黄焖鸡调味料，选择自己喜欢的品牌，也可以自己调配：酱油、蚝油、胡椒粉、辣椒粉、白糖、盐，用量随口味调整；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162358948549990.jpg>第6步: 混合均匀，让鸡肉尽可能多吸入料汁；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162358948534745.jpg>第7步: 加适量热水，豆腐均匀入锅；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162358948538390.jpg>第8步: 煮沸后盖盖子，中火炖15分钟；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162358948601990.jpg>第9步: 尝尝味道，根据口味适量加盐或者酱油；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162358948625847.jpg>第10步: 关火，青红椒入块，利用余温使其变色，颜色更加艳丽诱人。</a>
