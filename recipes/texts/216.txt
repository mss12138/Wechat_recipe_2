recipe_name: 私房茄子
practice: 炒
flavor: 家常味
time: 30分钟
grade: 初级入门
materials: 主料茄子500克青椒适量西红柿100克辅料盐少许生抽4毫升蚝油1小勺植物油适量蒜末适量葱适量
step: <a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163515888702871.jpg>第1步: 准备好所需的食材；茄子500克、100克左右大小的西红柿、适量青椒、蒜头、葱；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163515888722660.jpg>第2步: 将茄子洗净后切成厚度为1厘米左右的片；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163515888761764.jpg>第3步: 切好的茄子片放入无油的平底锅中，小火慢慢煎软，略微有些金黄最好，千万不要糊了；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163515888711610.jpg>第4步: 将西红柿切成小块、青椒洗净后切成滚刀块、蒜切末、葱一部分切断，另一部分切成葱花；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163515888726905.jpg>第5步: 炒锅烧热后倒入适量植物油，正常炒菜的量就行，不需要太多，放入蒜末葱段炒出香味；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163515888723490.jpg>第6步: 再放入青椒翻炒；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163515895325173.jpg>第7步: 依次放入茄子、西红柿，撒少许盐翻炒均匀；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163515896708196.jpg>第8步: 再淋入约4毫升生抽、少许蚝油翻炒入味；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_163515899736587.jpg>第9步: 出锅前撒上葱花，一道少油不用水，跟肉一样好吃的私房茄子就做好啦！</a>
