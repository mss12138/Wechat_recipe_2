recipe_name: 网红雪碧渍黄瓜
practice: 拌
flavor: 酸甜味
time: 30分钟
grade: 新手尝试
materials: 主料黄瓜2根小米椒3个辅料雪碧适量味达美清香米醋1大勺白砂糖1小勺盐1/2小勺
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162315332834053.jpg>第1步: 准备好食材与调味料。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162315332889712.jpg>第2步: 小米椒去蒂切小粒。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162315332818671.jpg>第3步: 黄瓜先纵向剖开，可以一半小些一半大些，用刨皮刀在大的一半黄瓜切面刨出厚约1毫米的长条黄瓜片。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162315332917033.jpg>第4步: 小的一半切厚约1毫米的半圆形薄片。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162315333005436.jpg>第5步: 取一条长条黄瓜片从一头卷起，作为花芯竖立在盘子中央。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162315332953714.jpg>第6步: 再把一片片长条黄瓜片错开围在花芯外围，作为花瓣。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162315333011641.jpg>第7步: 均匀地撒上盐，再点缀红色的小米椒粒。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162315333060080.jpg>第8步: 淋上味达美清香米醋。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162315332978806.jpg>第9步: 倒入雪碧至盘子深度的八分满，不要倒满，黄瓜腌渍后会出水。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162315333116757.jpg>第10步: 最外圈的盘子边沿排上半圆形黄瓜片，花朵的形象就更漂亮了，冷藏冰镇半小时，更冰爽，酸酸甜甜，太开胃了！</a>
