recipe_name: 豆芽粉条这样做最好吃，不坨不沾锅，既当
practice: 炒
flavor: 咸鲜味
time: 10分钟
grade: 未知
materials: 主料黄豆芽适量粉条1小把辅料猪肉适量盐适量生抽适量鸡精适量葱段适量蒜瓣若干
step: <a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162417227960389.jpg>第1步: 准备适量黄豆芽洗净沥水备用。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162417228127835.jpg>第2步: 粉条提前一个小时左右，用开水浸泡，直至无硬芯。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162417228212637.jpg>第3步: 锅中不放油，先倒入切好的猪肉的肥肉部分煸炒，将肥肉炒成油梭子。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162417228362899.jpg>第4步: 再倒入瘦肉翻炒至变色。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162417228443670.jpg>第5步: 加入黄豆芽翻炒，炒至豆芽变软后，倒入粉条。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162417228538667.jpg>第6步: 加盐，鸡精，生抽调味。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162417228670131.jpg>第7步: 再放入青葱段儿翻炒，最后倒入拍扁的蒜瓣。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162417228886775.jpg>第8步: 翻炒入味后出锅装盘。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162417228932771.jpg>第9步: 好吃的豆芽炒粉条就完成了，非常下饭呢。</a>
