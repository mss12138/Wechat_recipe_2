recipe_name: 简易毛血旺
practice: 煮
flavor: 香辣味
time: 15分钟
grade: 未知
materials: 主料火锅底料1/2块鸭血250克海鲜菇30克火腿肠1根木耳20克毛肚200克豆芽200克辅料姜蒜一块小葱2根香菜2根花椒5克
step: <a href=https://st-cn.meishij.net/rs/202/56/12889202/n12889202_162323848140078.jpg>第1步: 锅里放油，把姜蒜和火锅底料放入翻炒出香味。然后加入水煮着。</a>
<a href=https://st-cn.meishij.net/rs/202/56/12889202/n12889202_162323848348379.jpg>第2步: 清洗配菜豆芽、木耳、海鲜菇</a>
<a href=https://st-cn.meishij.net/rs/202/56/12889202/n12889202_162323848437636.jpg>第3步: 毛肚多清洗一下</a>
<a href=https://st-cn.meishij.net/rs/202/56/12889202/n12889202_162323848696150.jpg>第4步: 鸭血切好备用，可以稍微厚点。</a>
<a href=https://st-cn.meishij.net/rs/202/56/12889202/n12889202_162323848803147.jpg>第5步: 火腿肠切滚刀块</a>
<a href=https://st-cn.meishij.net/rs/202/56/12889202/n12889202_162323848983081.jpg>第6步: 煮开后，很浓的火锅味道哈哈</a>
<a href=https://st-cn.meishij.net/rs/202/56/12889202/n12889202_162323849130538.jpg>第7步: 依次放入配菜</a>
<a href=https://st-cn.meishij.net/rs/202/56/12889202/n12889202_162323849382349.jpg>第8步: 放入鸭血 火腿肠</a>
<a href=https://st-cn.meishij.net/rs/202/56/12889202/n12889202_162323849451991.jpg>第9步: 最后放毛肚，这个比较容易熟。再次煮开后放入香菜和小葱。也可以出锅的时候再放然后浇入热油。根据自己情况来，我喜欢节省哈哈</a>
