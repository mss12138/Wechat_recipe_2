recipe_name: 奶香刀切小馒头
practice: 蒸
flavor: 家常味
time: 30分钟
grade: 新手尝试
materials: 主料中裕多用途麦芯粉500克辅料牛奶280克白糖80克酵母粉4克盐2克
step: <a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357093969543.jpg>第1步: 材料合影图：中筋面粉500克、牛奶280克、白糖80克、酵母4克，另外还用了两克盐。面粉的吸水性不一样，大家根据情况适当增减牛奶用量。牛奶预留一点，揉面时觉得干了再加。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357094152898.jpg>第2步: 牛奶隔水蒸热，大约30多度不烫手的温度，酵母放牛奶里融化。这一步是我的小经验，特别是在冬天做馒头，温牛奶对发酵起着很好的促进作用。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357094306199.jpg>第3步: 融化了酵母的牛奶倒入面粉里，加白糖，再放一小茶匙（约两克）的盐，先用筷子搅拌成絮状。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357094531805.jpg>第4步: 动手揉面吧，在揉的过程中能感觉到面团的干稀程度，自己适当调整。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357094703609.jpg>第5步: 这是揉好之后的面团。很多朋友问我的刀切馒头表面为何如此光滑，我的回答是只要发酵得当，秘诀是除了揉、还是揉，用劲反复揉过之后的面团，蒸出来的馒头一定很好看哦。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357094900739.jpg>第6步: 揉好的面团蒙上保鲜膜，在温暖处发酵一个小时至面团两倍大，冬天观察情况适当延长发酵时间。也可放在密闭空间（烤箱或微波炉），旁边放一碗热水发酵。其实我就是放在桌子上常温发酵的，因为咱们用的温牛奶和面，不用特别担心发酵问题。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357095252607.jpg>第7步: 取出一块面团，再次揉面，此刻如果发现面团稀了一些也没有关系，适当在案板上铺撒干面粉，混合揉面，可解决这个问题，而且这样的馒头口感更绵实。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357095416066.jpg>第8步: 揉好的面团进一步把它滚搓并整型成长柱状。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357095668335.jpg>第9步: 用刀把馒头切成一个个的馒头胚子，每个约30克。因为我是做的开口小馒头，所以切完一刀之后在中间再浅切了一刀（大约切至面团的三分之一处），这样蒸出来的馒头会自然裂开一个口子，美观之余也方便掰开食用哦。温馨提示：可能您需要一把锋利的刀。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357095970633.jpg>第10步: 一个个漂亮的小馒头切好了，把它们放入蒸锅里摆放整齐。加盖二次醒发半个小时左右。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357096135044.jpg>第11步: 这是馒头二次醒发后的样子。看到我发的这个图片，有网友再也忍不住了，给我留言：“楼主，其实你需要一个多层蒸锅。”朋友圈也有好友愿意送我一个多层蒸锅。我快乐的表示：“送我一个蒸锅，还你终生白花花的馒头。”</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357096459691.jpg>第12步: 蒸锅里加水，开火，旺火蒸20分钟，中途千万不要揭开锅盖。蒸好关火后也不要揭开，静待两、三分钟后再打开。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357096612400.jpg>第13步: 这是另外两锅的特写图。确实该买个多层蒸锅了，哈哈。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357096906237.jpg>第14步: 一堆可爱的小馒头。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_162357097134957.jpg>第15步: 几个可爱的小馒头。阳光明媚，馒头好吃！^_^</a>
