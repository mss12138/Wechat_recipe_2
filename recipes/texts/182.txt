recipe_name: 白萝卜香煎肉饼【宝宝辅食】
practice: 煎
flavor: 咸鲜味
time: 30分钟
grade: 新手尝试
materials: 主料猪肉末60克鸡蛋1个白萝卜30克淀粉10克辅料油适量葱花适量盐适量黑芝麻适量鲜菇粉适量酱油1勺淀粉适量
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163590576139391.gif>第1步: 白萝卜切成细丝，焯水，捞出。我家里是现成的白萝卜，你们有青萝卜的换成它也行。

萝卜本身水分比较充足，吃起来也是很清爽的，如果家里没有萝卜的，也能换成包菜、芹菜、油菜这些，适量搭配点蔬菜，不仅是为了荤素搭配，也能让不爱吃菜的宝宝更好接受蔬菜。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163590585572122.gif>第2步: 焯好的萝卜丝过凉水</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163590586706996.gif>第3步: 用手挤去多余的水分，放入肉馅当中。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163590588240587.gif>第4步: 肉馅里加上鲜菇粉、淀粉、葱花、盐还有1勺酱油来调味，这里调味的时候可以根据自己的口味来调。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163590589112159.gif>第5步: 半个蛋清打入肉馅当中，用筷子充分搅拌均匀。

*蛋清一方面是起到了让肉更嫩的作用，另一方面也是让各种食材更加黏合。如果宝宝鸡蛋过敏的话，可以改用水淀粉来调和，不过嫩度的话会稍微差一点。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163590592816551.gif>第6步: 不粘锅刷油，放入我们做好的萝卜肉泥，用硅胶铲整理成小饼的样子。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163590593763160.gif>第7步: 刚刚剩下的一般蛋黄液，用刷子均匀的刷在肉饼表面，再撒点黑芝麻做点缀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163590594999903.gif>第8步: 开小火，煎到底部有些微黄，就可以翻面了。直到两面同样微黄，确认熟透就能取出了。

*如果担心自己火候掌握不是很精准，怕不熟的话，可以稍微加点水，盖盖焖一下。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163590596374757.gif>第9步: 最后把肉饼切成条条的样子，就可以开吃了。</a>
