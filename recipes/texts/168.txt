recipe_name: 家庭版松鼠鱼
practice: 炸
flavor: 酸甜味
time: 10分钟
grade: 中级掌勺
materials: 主料鲈鱼1条辅料鸡蛋1个小葱适量姜片适量干淀粉适量盐1小勺白砂糖1~2大勺味达美 臻品料酒1大勺遵循自然 本味番茄沙司半瓶
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731083336213.jpg>第1步: 准备好食材和调味料，鱼用的是鲈鱼，其它鱼也行。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731083961131.jpg>第2步: 鲈鱼杀好洗净，擦干鱼皮水分，从鳃下鱼鳍下方下刀切掉鱼头。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731084420978.jpg>第3步: 再从鱼头向鱼尾方向，贴着脊骨片下鱼肉，片到鱼尾处不要切断鱼肉，翻面同样操作。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731084963173.jpg>第4步: 靠近鱼尾处，切断鱼脊骨。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731085491139.jpg>第5步: 再片掉鱼腹处的大骨头，这里的鱼肉太薄，剔掉大骨后也切不了花刀，可以斜着连肉带骨切掉。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731086007067.jpg>第6步: 顺着鱼身，直刀切宽约1厘米的长条，不要切断鱼皮。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731086614852.jpg>第7步: 转过90度，用斜刀片出长条齿状鱼肉，也不要切断鱼皮。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731087078480.jpg>第8步: 切好花刀的状态。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731087548757.jpg>第9步: 鱼肉用加了白醋的清水清洗，并挤干水分。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731088012039.jpg>第10步: 加入1小勺盐，1个鸡蛋的蛋清，抓捏到起胶质，注意手法轻柔，不要把鱼肉抓下来。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731088491707.jpg>第11步: 再加入葱段、姜片，1大勺料酒，抓匀，腌制15分钟以上。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731089139915.jpg>第12步: 准备一盘干淀粉，把鱼肉鱼皮都抹上干淀粉，每一个缝隙都要抹到，最后抖掉多余的淀粉。鱼头、鱼骨、鱼腹也抹上干淀粉。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731089775430.jpg>第13步: 葱姜捡出来在宽油锅中炸香捞掉丢弃。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731090317293.jpg>第14步: 双手捏着鱼皮对齐，鱼肉向下，先炸鱼肉中段，再左右，将鱼肉定型。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731090889970.jpg>第15步: 定型后鱼肉向上丢入油锅，用筷子夹着鱼尾扯出好看的形状定型，不停地用勺子舀热油浇在鱼身上。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731091223799.jpg>第16步: 炸到金黄酥脆出锅，鱼头鱼骨也炸黄炸脆，捞起沥油装盘。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731092133958.jpg>第17步: 炒锅留底油，倒入半瓶番茄沙司炒到起泡，加糖调节味道，再加开水稀释煮开，浇在鱼身上。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163731092973896.jpg>第18步: 鱼肉炸得金黄酥脆，裹着酸酸甜甜的番茄沙司，我家的老人小孩都爱吃。</a>
