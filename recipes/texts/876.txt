recipe_name: 可爱章鱼造型  芝士火腿面包
practice: 烤
flavor: 咸鲜味
time: 60分钟
grade: 新手尝试
materials: 主料高筋面粉 250克辅料细砂糖30克盐3克酵母3克奶粉10克鸡蛋1个牛奶50克水60克黄油 30克火腿肠 6根卡夫芝士粉 适量眼睛糖 12粒
step: <a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140777721219.jpg>第1步: 1准备食材。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140778342614.jpg>第2步: 主料部分除黄油外所有材料放入厨师机。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140783452582.jpg>第3步: 一档拌匀，转四档搅拌。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140784578635.jpg>第4步: 搅拌至扩展状态。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140785541441.jpg>第5步: 放入室温软化的黄油，继续搅拌至完全扩展状态。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140788513862.jpg>第6步: 面团取出滚圆，一次发酵至两倍大。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140789275131.jpg>第7步: 平均分成6份，松驰15分钟。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140793973388.jpg>第8步: 取一个面团擀长，底边压薄，把火腿肠放上去比一下，长出来的部分切成六条。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140796785904.jpg>第9步: 从上向下卷好，收口位置滚动一下，变得更贴合一点。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140797893967.jpg>第10步: 表面滚上一层卡夫芝士粉，二次发酵至两倍大。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140798877824.jpg>第11步: 送入预热好的烤箱180度烤18分钟。</a>
<a href=https://st-cn.meishij.net/rs/19/226/2681519/n2681519_162140800033384.jpg>第12步: 面包晾凉，用炼乳把眼睛糖粘上，也可以用巧克力融化后画出眼睛。</a>
