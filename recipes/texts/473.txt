recipe_name: 醋泡花生米
practice: 腌
flavor: 家常味
time: 10分钟
grade: 新手尝试
materials: 主料花生米250克辅料香醋120克煮肉料1份生抽10克盐2克糖5克洋葱适量青椒适量亚麻公社胡麻油1小勺
step: <a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162838963872737.jpg>第1步: 今年刚刚上市的新鲜百日红花生，我买的湿的回来自己晒干了，然后剥出花生米</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162838965067170.jpg>第2步: 剥出来的花生米洗净控水</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162838964908010.jpg>第3步: 备好煮肉料：八角，花椒，小茴香，桂皮，香叶，干辣椒和姜片</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162838964987019.jpg>第4步: 放入锅中加水</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162838965062308.jpg>第5步: 小火煮5分钟，盖盖再焖20分钟，让味道充分融入汁里，</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162838965019877.jpg>第6步: 把煮好的料汁过滤出来取150克料水，再放入醋，生抽，盐，糖搅拌均匀，成为醋料汁</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162838965051860.jpg>第7步: 把花生米放入醋料汁中，把过滤出来的大块的料再放进去，小茴像就不要了，否则吃的时候太麻烦，也可以都不要，但我喜欢把大块的料放进去一起浸泡，这样味更足</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162838966092666.jpg>第8步: 放冰箱浸泡了一晚上，花生米吸饱了汁水，非常爽脆</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162838965224870.jpg>第9步: 洋葱和青椒加盐醋和胡麻油拌均匀</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162838966555258.jpg>第10步: 再和泡好的花生米一起拌均匀就可以享用啦。“亚麻公社胡麻油，植物油中的脑白金”</a>
