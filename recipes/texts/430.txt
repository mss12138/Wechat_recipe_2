recipe_name: 丝瓜木耳炒鸡蛋
practice: 炒
flavor: 咸鲜味
time: 15分钟
grade: 新手尝试
materials: 主料丝瓜 一根鸡蛋2个木耳一把辅料生抽 2勺清水一小碗玉米油适量姜片2片料酒1勺
step: <a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162963534483734.jpg>第1步: 准备材料。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162963536892330.jpg>第2步: 鸡蛋打散，干净不粘锅中用厨房纸巾擦干水分。倒入2勺玉米油，开大火热油，倒入打散的鸡蛋，摊开，转中火将蛋液煎凝固。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162963587520546.jpg>第3步: 煎到两面凝固金黄，取出，切块。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162963588093127.jpg>第4步: 锅里再倒入2勺玉米油，大火热油，放姜片爆香，倒入切成条状的丝瓜，和提前泡开的木耳。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162963589060858.jpg>第5步: 翻炒1-2分钟，丝瓜略微变软，倒入1勺料酒去除丝瓜的涩味，提升风味。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162963591537619.jpg>第6步: 然后把之前处理好的鸡蛋，一起倒入翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162963592131922.jpg>第7步: 淋入一小碗清水，搅匀焖煮1分钟。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162963594528682.jpg>第8步: 快出锅前，淋2勺轻盐特级原汁酱油，炒匀调味。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162963595884914.jpg>第9步: 炒匀入味后，关火，装盘，好吃的丝瓜木耳炒鸡蛋就做好啦！</a>
