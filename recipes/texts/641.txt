recipe_name: 茶香鹌鹑蛋，低温烹调更入味哟！
practice: 煮
flavor: 家常味
time: 2小时
grade: 初级入门
materials: 主料鹌鹑蛋24枚红茶包1个辅料盐少许香叶1片桂皮1块姜2片葱1段原汁酱油适量冰糖少许蚝油1勺
step: <a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_162492978467270.jpg>第1步: 准备好全部食材。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_162492978879075.jpg>第2步: 把鹌鹑蛋洗干净。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_162492979014471.jpg>第3步: 准备一锅清水，冷水放入鹌鹑蛋，大火煮至沸腾。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_162492979307500.jpg>第4步: 关火，过下冷水，沥干水分。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_162492979505514.jpg>第5步: 将煮好的鹌鹑蛋表面敲出裂纹。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_162492979828563.jpg>第6步: 放入智能料理盒，注入纯净水至八分满。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_162492980145193.jpg>第7步: 依次加入全部配料。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_162492988768558.jpg>第8步: 盖好盖子，连接APP，设置80度2小时。（可以把这个模式设置成茶叶蛋模式哟）</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_162492989867638.jpg>第9步: 到时间后，继续焖12-20个小时再吃效果最好。</a>
