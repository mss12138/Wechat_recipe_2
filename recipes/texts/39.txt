recipe_name: 盐焗鸡腿 #电饭煲版#
practice: 焗
flavor: 咸鲜味
time: 30分钟
grade: 初级入门
materials: 主料鸡腿5只辅料生姜1块葱绿2根食盐1小勺花椒油1小勺葵花籽油1勺盐焗鸡粉20克
step: <a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403176291808.jpg>第1步: 准备好食材</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403176378310.jpg>第2步: 鸡腿清洗干净， 用盐水浸泡15分钟，把鸡肉中的血水泡掉，用于去除鸡肉中的腥味</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403176874085.jpg>第3步: 葱姜洗干净，葱切段，姜切丝备用</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403176245483.jpg>第4步: 把泡好的鸡腿捞出来，控干水分放入碗中，放入葱姜丝和半袋盐焗鸡粉，抓揉均匀</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403176441453.jpg>第5步: 再淋1勺食用油抓匀</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403176454177.jpg>第6步: 家里有花椒油的也可放些，微微地麻辣味道更好</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403177085826.jpg>第7步: 将鸡腿再次抓揉均匀，盖上保鲜膜，腌制1~2小时</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403176535241.jpg>第8步: 取出电饭煲，在底部刷少许食用油，再切些葱段姜片放在下面</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403177024177.jpg>第9步: 这时把腌制好的鸡腿、葱姜丝连同腌汁，一同放入电饭煲中</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403176935634.jpg>第10步: 盖上盖子，启动快速煮饭键，时间是25分钟左右</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403176703629.jpg>第11步: 时间到，打开盖子，一股香味扑鼻而来</a>
