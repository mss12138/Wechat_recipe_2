recipe_name: 花菜木耳炒肉片
practice: 炒
flavor: 家常味
time: 30分钟
grade: 新手尝试
materials: 主料有机花菜500克杏鲍菇100克猪腿肉100克黑木耳（干）10克辅料六月鲜10克轻盐特级原汁酱油30毫升味达美臻品料酒15毫升水100毫升葡萄籽油15毫升淀粉5克

<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543024883869.jpg>第2步: .将肉片放入碗中倒入5克淀粉进行揉捏上浆备用</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543024895315.jpg>第3步: 锅内放入适量的清水，大约800毫升</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543024819206.jpg>第4步: 水煮开后倒入黑木耳焯水2分钟捞出备用</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543024955932.jpg>第5步: .倒入花菜进行焯水2分钟捞出备用</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543024934015.jpg>第6步: 锅内倒入15毫升葡萄籽油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543024974169.jpg>第7步: 倒入上浆的猪肉片，快速煸炒</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543025070590.jpg>第8步: 待肉片颜色变白，倒入15毫升味达美臻品料酒，去腥增香</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543025090902.jpg>第9步: 猪肉片颜色全部变白即可盛出</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543024907931.jpg>第10步: 将杏鲍菇片倒入进行煸炒</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543025084530.jpg>第11步: 倒入泡发洗净的黑木耳继续煸炒</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543025074453.jpg>第12步: 倒入100毫升清水进行炖煮3分钟</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543024987320.jpg>第13步: 倒入之前焯过水的花菜</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543025195531.jpg>第14步: 将食材煸炒均匀</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543025379482.jpg>第15步: 倒入之前煸炒过的猪肉片继续煸炒</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543025335437.jpg>第16步: 倒入30毫升六月鲜10克轻盐特级原汁酱油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164543025330185.jpg>第17步: 翻炒均匀即可出锅</a>
