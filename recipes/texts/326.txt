recipe_name: 你家山药还在蒸着吃吗？来试试这盘素炒山
practice: 炒
flavor: 咸鲜味
time: 5分钟
grade: 新手尝试
materials: 主料山药1根胡萝卜半个彩椒15克木耳5克辅料大葱10克酱油1小勺鸡汁1勺食盐2克
step: <a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196757460753.jpg>第1步: 先把需要的食材提前准备好。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196757339499.jpg>第2步: 准备少许木耳放入碗中提前用温水泡发。

木耳一丢丢即可，泡完后的木耳会有好多哦。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196760005353.jpg>第3步: 山药去皮洗掉表面粘液，斜着切成薄片，厚度约1元硬币左右。

山药去皮时一定要带上手套，避免表面粘液引起皮肤发痒。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196760496089.jpg>第4步: 切好的山药放入清水中，避免山药接触空气后氧化。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196760140374.jpg>第5步: 胡萝卜去皮切薄片，彩椒掰成小块，这里的彩椒主要是为了配色没有可以不放，家里有西蓝花的也可以放几朵西蓝花。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196760307495.jpg>第6步: 少许大葱切丝。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196771373968.jpg>第7步: 锅中提前烧一锅开水，水开后放入胡萝卜、木耳跟山药焯水。

山药比较容易熟，所以一定要等胡萝卜跟木耳断生后再放入山药。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196771752258.jpg>第8步: 把焯好水的食材捞出控干水分放入碗中。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196771364215.jpg>第9步: 热锅冷油放入葱花炒香。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196771763617.jpg>第10步: 然后倒入焯好水的胡萝卜、木耳、山药大火快速翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196771744960.jpg>第11步: 倒入彩椒翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196771437419.jpg>第12步: 家里有鸡汁的话可以加1勺鸡汁提鲜。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196771677252.jpg>第13步: 2克食盐调味。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196771899689.jpg>第14步: 1小勺酱油调味。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163196792866191.jpg>第15步: 格琳诺尔有机亚麻籽油，宝宝的聪明油</a>
