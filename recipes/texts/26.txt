recipe_name: 榨菜蒸鱼
practice: 蒸
flavor: 咸鲜味
time: 15分钟
grade: 初中水平
materials: 主料新鲜草鱼一条榨菜一包辅料植物油适量盐适量蒸鱼豉油10克料酒10克
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158272569760499.jpg>第1步: 主料：新鲜草鱼一条

辅料：植物油、盐、姜、榨菜一包、香葱、蒸鱼豉油、料酒</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158272569897263.jpg>第2步: 把草鱼洗净，把鱼脖子切一刀，去掉鱼头，在切掉鱼尾，轻轻拍鱼背来回，把鱼线轻轻抽出切块</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158272570058246.jpg>第3步: 把姜切丝、香葱切末</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158272570346269.jpg>第4步: 把鱼块加入大碗里，加姜丝、料酒拌均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158272570329964.jpg>第5步: 把榨菜加一半铺在鱼块身上，腌制15分钟左右</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158272570490142.jpg>第6步: 把剩下一半的榨菜铺在盘底
把腌制好的鱼块整齐码在盘中</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158272570595268.jpg>第7步: 再把姜丝和榨菜铺在上面

把蒸锅加水大火烧开，把鱼盘子加进去

大火蒸8分钟</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158272570663615.jpg>第8步: 取出，加蒸鱼豉油，撒上香葱末</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158272570737512.jpg>第9步: 热锅凉油，加植物油，烧10成热，直接浇到香葱末榨菜上面即可上桌了</a>
product: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158272583794112.jpg>榨菜蒸鱼成品图</a>
