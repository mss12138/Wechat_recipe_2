recipe_name: 自制咸肉
practice: 腊
flavor: 咸鲜味
time: 15分钟
grade: 中级掌勺
materials: 主料五花肉适量辅料泡菜盐适量花椒适量八角2个桂皮4块
step: <a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113320297259.jpg>第1步: 准备好适量五花肉</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113320288971.jpg>第2步: 腌肉之前先来炒盐，炒锅烧热，放入泡菜盐</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113320288121.jpg>第3步: 用小火将泡菜盐炒香，炒制微微泛黄，盛出冷却；也可以加入花椒大料一起炒，因为我后面会加，这步就没加了</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113320237837.jpg>第4步: 五花肉不用洗直接放入盆中，再将放凉的泡菜盐均匀揉搓五花肉的表面，并且确保每一个地方都要抹到</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113320217564.jpg>第5步: 全部抹好，轻轻压实压紧</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113320296264.jpg>第6步: 然后将盆放在阴凉通风处，腌制7~10天的时间，每天翻面，入味更均匀</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113320295457.jpg>第7步: 锅中放入八角、桂皮、花椒、1小把泡菜盐和适量清水，大火煮开后关火</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113320235138.jpg>第8步: 待完全冷却后倒入腌肉盆中，将腊肉浸泡在调料水中</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113320240852.jpg>第9步: 泡至1天，中途翻面，这样晒干后的咸肉不易变化，也不易发黑</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113320204086.jpg>第10步: 泡好后用绳子串起来，放室外通风处沥水晾晒</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113372518827.jpg>第11步: 差不多需要凉晒半个月左右（冬天），晾干后把它们放至阴凉通风处或是冰箱冷冻室，冷冻起来就可以了</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113365423504.jpg>第12步: </a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164113366608210.jpg>第13步: 放入蒸锅中，蒸15分钟左右，蒸好，取出即可食用</a>
