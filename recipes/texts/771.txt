recipe_name: 熘肉段
practice: 熘
flavor: 酸甜味
time: 10分钟
grade: 初中水平
materials: 主料猪瘦肉200克青椒1个红椒20克辅料油适量盐适量葱花20克蒜瓣3个姜末10克淀粉110克生抽30毫升白糖2勺醋1勺鸡精适量料酒1勺胡椒粉适量
step: <a href=https://st-cn.meishij.net/rs/150/151/14787900/n14787900_162289367448857.jpg>第1步: 准备好所有的食材。</a>
<a href=https://st-cn.meishij.net/rs/150/151/14787900/n14787900_162289368069906.jpg>第2步: 淀粉加水稀释后静置。</a>
<a href=https://st-cn.meishij.net/rs/150/151/14787900/n14787900_162289367940622.jpg>第3步: 猪肉洗净切成手指大小块，加入料酒，糖，盐胡椒粉腌制片刻。</a>
<a href=https://st-cn.meishij.net/rs/150/151/14787900/n14787900_162289368039544.jpg>第4步: 青红椒切块，取小碗加入生抽，糖，醋，鸡精，葱姜蒜适量清水少许淀粉调匀备用。</a>
<a href=https://st-cn.meishij.net/rs/150/151/14787900/n14787900_162289368427277.jpg>第5步: 淀粉水沉淀后倒掉上面的清水，把猪肉放入淀粉中再加一勺食用油。</a>
<a href=https://st-cn.meishij.net/rs/150/151/14787900/n14787900_162289368333563.jpg>第6步: 然后把猪肉用力抓拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/150/151/14787900/n14787900_162289368638632.jpg>第7步: 锅中油烧热，用筷子一个个放入猪肉炸制。猪肉变色捞出，然后再重新复炸一遍。</a>
<a href=https://st-cn.meishij.net/rs/150/151/14787900/n14787900_162289369305157.jpg>第8步: 复炸过得猪肉沥干油待用。</a>
<a href=https://st-cn.meishij.net/rs/150/151/14787900/n14787900_162289369074878.jpg>第9步: 炒锅留底油倒入调好的酱汁大火煮开。</a>
<a href=https://st-cn.meishij.net/rs/150/151/14787900/n14787900_162289369371651.jpg>第10步: 酱汁出香味，倒入炸过的猪肉和青红椒。</a>
<a href=https://st-cn.meishij.net/rs/150/151/14787900/n14787900_162289369570412.jpg>第11步: 快速翻拌均匀即可关火。</a>
