recipe_name: 豆豉烧鲫鱼
practice: 烧
flavor: 咸鲜味
time: 30分钟
grade: 未知
materials: 主料鲫鱼1000克辅料阳江豆豉80克白砂糖20克味精5克鸡粉10克料酒100克香菜200克蒜头150克指天椒20克葱80克姜100克酱油100克蚝油80克芝麻油30克
step: <a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928555043695.jpg>第1步: 需要鲜杀的鲫鱼</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928555216974.jpg>第2步: 清洗干净去内脏，去鱼鳃</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928555520173.jpg>第3步: 背部放两刀，适于更容易炸干入味</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928555702767.jpg>第4步: 准备姜、葱、香菜根部、蒜头、豆豉、指天椒</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928556095128.jpg>第5步: 将姜、葱、香菜根部，拍散容易腌制出味</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928556323007.jpg>第6步: 加入料酒，抓匀出味</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928556403602.jpg>第7步: 把洗好的鲫鱼放在盆里，加入腌制味料抹匀</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928556789934.jpg>第8步: 准备好蒜头，姜，指天椒，葱叶，香菜根部</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928556929387.jpg>第9步: 一切准备就绪，把鲫鱼捞起擦干外表水分。</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928557222149.jpg>第10步: 起锅烧油，有温冒青烟，放入鲫鱼关小火慢炸只鱼身干脆香（这里注意高温下锅时，鱼的身上都有水分容易爆油，记得盖上锅盖）</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928557505272.jpg>第11步: 炸成这个样子就可以了，表皮酥脆，肉质偏干。</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928557704334.jpg>第12步: 锅里放油，放入蒜头，姜温火炸出香味</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928557987027.jpg>第13步: 再放入葱叶，也炸出香味，炸焦干</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928558200631.jpg>第14步: 再放入香菜根部，指天椒，炒香</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928558474138.jpg>第15步: 最后再放入豆豉，炒香之后就可以加水</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928558637978.jpg>第16步: 水差不多1000克，再放入调料烧开</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928558970357.jpg>第17步: 把鲫鱼放进去小火焖煮</a>
<a href=https://st-cn.meishij.net/rs/204/217/14991954/n14991954_162928559119053.jpg>第18步: 焖煮汤汁浓稠放味精，再改大火烧一下就完成装盘</a>
