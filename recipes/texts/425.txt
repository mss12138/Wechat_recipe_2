recipe_name: 爽脆鲜嫩下饭的肉丝炒芹菜胡萝卜
practice: 炒
flavor: 家常味
time: 5分钟
grade: 初级入门
materials: 主料猪里脊1块芹菜1棵胡萝卜半根辅料油少许盐少许酱油适量玉米淀粉1小勺料酒少许
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162971007533578.jpg>第1步: 芹菜清洗干净，胡萝卜刮皮清洗干净；瘦肉用冻肉或者鲜肉均可；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162971007697575.jpg>第2步: 冷冻变硬的肉趁着似化非化的时候切成均匀的细丝，放少许玉米淀粉和料酒抓捏上浆；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162971007599771.jpg>第3步: 芹菜和胡萝卜切细丝；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162971007606321.jpg>第4步: 热锅凉油，肉丝入锅，用铲子迅速划散，随着温度上升，肉丝根根独立并变色；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162971007682801.jpg>第5步: 倒适量酱油，扒拉几下盛出备用；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162971007653337.jpg>第6步: 另起锅，倒少许油，胡萝卜丝入锅翻炒；胡萝卜水分少，油少干扒拉易糊锅，淋少许水增加湿度，利于胡萝卜均匀受热、成熟；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162971007739705.jpg>第7步: 胡萝卜变软，倒入芹菜，大火翻炒；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162971007706716.jpg>第8步: 根据口味撒少许盐，倒适量酱油；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162971007797263.jpg>第9步: 将炒好的肉丝倒回锅，翻拌均匀出锅。</a>
