recipe_name: 蒜泥白肉
practice: 拌
flavor: 酸辣味
time: 10分钟
grade: 初级入门
materials: 主料五花肉1条黄瓜2条辅料卤料：八角3颗卤料：香叶2片卤料：桂皮1块卤料：姜1大块卤料：草果1颗卤料：花椒1撮卤料：干柚子皮1小块浇汁：大蒜1头浇汁：生抽40克浇汁：米醋20克浇汁：盐2克浇汁：白糖8克浇汁：香油10克
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296663802622.jpg>第1步: 五花肉一大条，可多煮点儿分2-3顿吃，还能做回锅肉；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296663861096.jpg>第2步: 卤料准备好，品种和用量不限这些；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296663865173.jpg>第3步: 将肉切成两块，与卤料同入凉水锅中；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296663830940.jpg>第4步: 大火煮开后撇掉浮沫，转小火，盖盖子焖煮30-40分钟，筷子 能轻松扎透肉厚的地方；肉块浸泡在汤里至温热捞出；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296663891630.jpg>第5步: 修掉肉块两边不规则部分，切成厚约2毫米的长形薄片；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296663833628.jpg>第6步: 黄瓜切掉两端，长度与肉片长度相等；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296664124569.jpg>第7步: 用削皮刀刮出薄片；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296664199989.jpg>第8步: 取一片肉，放一片黄瓜；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296664153672.jpg>第9步: 对折；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296664144438.jpg>第10步: 将肉片黄瓜摆盘，多余的黄瓜片围边摆盘；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296664292069.jpg>第11步: 大蒜先剁碎，再用蒜杵子捣成蒜泥，蒜泥能充分释放出大蒜中的营养物质，味道也更浓香；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296664279372.jpg>第12步: 用适量生抽、米醋、盐、香油调成蒜泥浇汁；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162296664436616.jpg>第13步: 用小勺将蒜泥汁浇在肉片上即可食用。</a>
