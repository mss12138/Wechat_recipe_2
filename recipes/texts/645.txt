recipe_name: 汉堡牛肉饼
practice: 煎
flavor: 咸鲜味
time: 15分钟
grade: 新手尝试
materials: 主料牛肉400克猪肉100克辅料香菇50克胡萝卜40克熟玉米粒60克生抽10克老抽5克盐4克胡椒粉2克花椒水60克水淀粉60克姜末15克
step: <a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162485684054768.jpg>第1步: 干香菇提前泡软备用</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162485684022010.jpg>第2步: 牛肉和猪肉剁馅放入盆里，放入盐，胡椒粉，生抽老抽</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162485684981736.jpg>第3步: 放入姜末</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162485684998864.jpg>第4步: 分3次加入花椒水，顺一个方向搅拌到上进儿，每次都要搅拌到花椒水吸收再加下一次。加了花椒水，牛肉饼的口感会更嫩。也可以用葱姜水代替。</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162485685056157.jpg>第5步: 放入香菇碎，胡萝卜碎和玉米粒</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162485686431190.jpg>第6步: 搅拌均匀后放入水淀粉</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162485686333623.jpg>第7步: 搅拌到上劲儿，饧半小时左右</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162485686495019.jpg>第8步: 分成约100克一个，团成饼，沾面包糠</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162485687386545.jpg>第9步: 放入模具锅里，盖盖煎</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162485687348460.jpg>第10步: 一面煎金黄后翻面煎。两面煎至金黄色就可以了，两面大约煎8分钟。</a>
