recipe_name: 鱼头泡饼
practice: 煮
flavor: 咸鲜味
time: 90分钟
grade: 初级入门
materials: 主料鱼头2100克面粉200克水130克辅料盐5克油40克黄豆酱4汤匙毛葱2个生姜10看红辣椒2个八角少许桂皮少许香叶少许料酒4汤匙生抽2汤匙大蒜10克香菜少许
step: <a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082204865.jpg>第1步: 面粉和水混合揉成面团（开始会很粗糙，过几分钟再揉一下就会光滑），醒面</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082296948.jpg>第2步: 鱼头去鳃去鳞去内脏，洗净</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082256048.jpg>第3步: 准备配料，毛葱和生姜切片</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082258172.jpg>第4步: 锅中放油烧热，放入鱼肉煎一下（可以煎两面，也可以煎一面）</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082354019.jpg>第5步: 放入葱姜辣椒，八角桂皮香叶爆香</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082339065.jpg>第6步: 放入黄豆酱</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082452253.jpg>第7步: 放入料酒</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082416389.jpg>第8步: 放入生抽</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082414651.jpg>第9步: 添水，没过鱼块</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082441817.jpg>第10步: 放盐，大火烧开转中火炖煮</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082428004.jpg>第11步: 炖鱼时烙饼，醒后的面团擀薄，抹油，卷起后擀成饼状，用平底锅或者电饼铛烙熟备用</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082592654.jpg>第12步: 香菜切段，大蒜切末</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082536323.jpg>第13步: 炖30分钟后，汤汁变少</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082515489.jpg>第14步: 再炖5分钟左右即可</a>
<a href=https://st-cn.meishij.net/rs/211/229/14744961/n14744961_162331082321295.jpg>第15步: 将鱼头盛出，将饼泡入鱼汤中，味道真的很鲜美哦！</a>
