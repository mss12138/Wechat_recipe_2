recipe_name: 麻辣豆腐包子
practice: 蒸
flavor: 香辣味
time: 30分钟
grade: 高级厨师
materials: 主料老豆腐600克榨菜粒80克面粉400克辅料小苏打0.5克蒜末20克花椒粉3克辣椒粉15克生抽10克小葱适量食盐适量热油适量黑胡椒粉适量温水205克酵母4克
step: <a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162169347558761.jpg>第1步: 豆腐切丁，切得豆腐丁越小越好</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162169347682141.jpg>第2步: 烧一锅水，放入食盐，豆腐焯水2分钟，去除豆腥味</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162169347879400.jpg>第3步: 多切一点葱花，这样做好的包子吃起来比较香，然后放一点点小苏打，搅拌均匀备用</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162169348053795.jpg>第4步: 蒜末也多准备一点，增加香味</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162169348256206.jpg>第5步: 豆腐调味。豆腐中放入生抽、食盐、花椒粉、榨菜粒、黑胡椒粉、蒜末、辣椒粉、葱花，最后浇上热油，不用过度搅拌，拌匀即可</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162169348448035.jpg>第6步: 和面。温水里面放入酵母溶解，边搅拌边倒入面粉，搅拌至看不到干面粉，揉成团，盖上保鲜膜静置十分钟</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162169348655994.jpg>第7步: 静置好的面团揉至表层光滑，再次盖上保鲜膜，静置十分钟</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162169348751508.jpg>第8步: 将静置好的面团取出，不用揉直接搓成长条，切成大小相等的小剂子</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162169348966756.jpg>第9步: 擀成大薄片，加入豆腐馅料，顺着边缘捏褶子，褶子的宽度尽量保持一致，这样包出来的包子颜值也比较高</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162169349100782.jpg>第10步: 放入蒸笼里面进行第一次发酵，用手按一个小窝，轻轻回弹就代表发酵好了</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162169349307598.jpg>第11步: 将包子放入蒸锅里面，水开以后，大火蒸15分钟，再虚蒸五分钟避免塌陷</a>
