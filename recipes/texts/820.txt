recipe_name: 鸭子这样炖，肉软汤鲜营养丰富，好喝暖胃
practice: 炖
flavor: 咸鲜味
time: 2小时
grade: 未知
materials: 主料鸭子1只辅料葱姜适量料酒适量温水适量盐适量红枣若干枸杞适量山药半根
step: <a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162209998937305.jpg>第1步: 处理干净残留鸭毛，将鸭子洗净剁成块儿。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162209999021370.jpg>第2步: 鸭肉冷水下锅，放入葱姜料酒去腥，水开后再煮几分钟，煮出多余血沫。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162209999213794.jpg>第3步: 超好的鸭肉过凉水，多冲洗几遍，冲去浮沫后备用。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162209999446922.jpg>第4步: 锅中放油，先放葱姜炝锅，再倒入鸭肉翻炒。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162209999501410.jpg>第5步: 炒至鸭肉表皮微微发焦就可以了，这样一炒会让鸭肉更加紧实，口感更好。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162209999614987.jpg>第6步: 倒入尽量多一些的温水或热水，水要一次性加够。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162209999894679.jpg>第7步: 加盐调味大火煮开后盖盖转小火慢炖90-120分钟，鸭肉软烂即可。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162209999910066.jpg>第8步: 之后再放入山药。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162210000047436.jpg>第9步: 红枣和枸杞，继续再炖10分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162210000286658.jpg>第10步: 新疆老爸老妈寄来的大枣有点太出彩了，大个红枣可以少放几个。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162210000451624.jpg>第11步: 鸭肉软烂，鸭汤鲜美浓郁。</a>
