recipe_name: 黄花菜最下饭的家庭做法，一焯一炒，地道
practice: 炒
flavor: 香辣味
time: 15分钟
grade: 未知
materials: 主料鲜黄花菜适量辅料猪肉1小块青椒若干红椒1-2根葱段适量盐适量生抽适量豆瓣酱1勺
step: <a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495553185864.jpg>第1步: 买来的新鲜黄花菜。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495553264757.jpg>第2步: 掐断底部的同时，一抽，这个芯就出来了，一大兜黄花菜，去芯的工作也是蛮大，边看美剧边处理吧。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495553450966.jpg>第3步: 去芯的黄花菜洗净备用。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495553612277.jpg>第4步: 锅里的水烧开后倒入黄花菜焯水2分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495553881160.jpg>第5步: 将黄花菜攥干水分放入盘中备用。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495554073570.jpg>第6步: 青红椒切块儿。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495554270652.jpg>第7步: 在准备葱姜，一小块猪肉切成片儿。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495554418526.jpg>第8步: 锅中放油或者不放油都可以，倒入猪肉煸炒出油，接着再放入姜蒜一起炒香。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495554515229.jpg>第9步: 再舀入一勺豆瓣酱炒出红油。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495554663715.jpg>第10步: 倒入青红椒一起翻炒。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495554717431.jpg>第11步: 炒至青红椒断生，放入黄花菜一起翻炒。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495554944657.jpg>第12步: 加盐，生抽调味儿，放入葱段再炒几下就可以出锅了。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495643904806.jpg>第13步: 闻着就很香，看着也很有食欲。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162495644134564.jpg>第14步: 好吃的辣炒黄花菜就完成了。</a>
