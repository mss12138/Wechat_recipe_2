recipe_name: 杏鲍菇炒猪耳朵
practice: 炒
flavor: 家常味
time: 10分钟
grade: 初中水平
materials: 主料杏鲍菇300克卤猪耳朵1个红辣椒1个辅料植物油适量盐适量生抽适量
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164475812216968.jpg>第1步: 主料：杏鲍菇3个、卤猪耳朵一个、青红辣椒两个
辅料：植物油适量、盐适量、姜5克、生抽一大勺、鸡精1克</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164475812251892.jpg>第2步: 把杏鲍菇洗洗，用淡盐水泡泡，多清洗几次，捞出来切片</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164475812209928.jpg>第3步: 把青红辣椒洗净去籽切块、猪耳朵切你喜欢的形状，一般都是切片比较多些，因为是已经卤好的，不需要加更多香料炒了，味道已经很好了，姜切末</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164475812203341.jpg>第4步: 热锅凉油，油热7成，下切好杏鲍菇翻炒翻炒出香味，不可久炒</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164475812209376.jpg>第5步: 加辣椒翻炒出辣味，这样好入味</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164475812290931.jpg>第6步: 加盐翻炒均匀，提前加盐是因为卤猪耳朵是咸味的，不可以在加盐了</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164475812276059.jpg>第7步: 加卤猪耳朵快速翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164475812213614.jpg>第8步: 加生抽翻炒到猪耳朵和杏鲍菇都出香味以后，爱吃辣味重的，可以加尖椒或者小米粥一起翻炒，味道更好</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164475812291889.jpg>第9步: 然后加少许的水翻炒出香味，如果太干的话要加一些，不干可以不加了，我是大火爆炒的比较干一些，加鸡精翻炒均匀即可出锅了</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164475812380212.jpg>第10步: 杏鲍菇很鲜美可口，猪耳朵劲道爽口有韧劲，特别的好吃，下酒下饭都是美味，尤其是逢年过节招待客人尤其的好，特有面子</a>
