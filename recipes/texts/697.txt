recipe_name: 蒜蓉粉丝蒸田螺
practice: 蒸
flavor: 香辣味
time: 30分钟
grade: 中级掌勺
materials: 主料田螺肉400克绿豆粉丝一把大蒜一整个剁辣椒一勺姜末一勺辅料油适量盐适量生抽一勺蚝油一勺料酒两勺
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186154195270.jpg>第1步: 绿豆粉丝提前放入温水中泡软备用。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186154170603.jpg>第2步: 田螺肉捡出杂质，放入清水中，加入一勺盐，用手反复抓揉搓洗，换水2、3次即可。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186154186117.jpg>第3步: 将洗干净的田螺肉倒入冷水锅中，加入几片姜片和两勺料酒、适量盐汆水，汆好的田螺肉捞出来，控干备用。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186154141940.jpg>第4步: 充分泡软的粉丝铺在盘底，稍微用手撒开点，铺匀。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186154197449.jpg>第5步: 煮好的田螺肉也均匀地铺在粉丝上。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186154216227.jpg>第6步: 锅中倒入一勺植物油，倒入蒜末，小火煸炒至微微变黄，香味飘出。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186154225276.jpg>第7步: 蒜香味飘出后，将剁辣椒倒入锅中一起翻炒，我的剁辣椒是自己家里做的，放了些黑豆豉，特别香。
再加入一勺生抽，两勺清水，翻炒均匀，蒜蓉酱就炒好了。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186154327867.jpg>第8步: 将炒好的蒜蓉酱铺在田螺肉上。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186154266018.jpg>第9步: 放入蒸锅大火蒸15分钟就可以了。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186154341841.jpg>第10步: 出锅后撒上葱花点缀下就可以上桌啦！香喷喷的蒜蓉田螺，趁热更好吃哟！</a>
