recipe_name: 麻辣鸡块
practice: 炒
flavor: 麻辣味
time: 30分钟
grade: 新手尝试
materials: 主料麻公鸡半只辅料青椒2个青麻椒1串干辣椒2个姜5片豆瓣酱1大勺糖1小勺亚麻公社胡麻油15毫升黄酒50毫升盐1克老抽5克生抽10克藤椒油5毫升
step: <a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806059897212.jpg>第1步: 买了一只麻公鸡，切了一半来用。</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806060956851.jpg>第2步: 切成块，多用清水冲洗浸泡几次，去除血水，这样炒出来不会有血沫</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806060845293.jpg>第3步: 加老抽，生抽，胡椒粉和少许的盐。盐一点即可</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806060872416.jpg>第4步: 抓揉均匀，腌30分钟，让鸡肉入个底味</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806061961154.jpg>第5步: 锅里放油，放入干辣椒，麻椒和姜片煸炒出香味</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806061954827.jpg>第6步: 放入鸡块炒到变色后放一点糖</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806061909330.jpg>第7步: 放入豆瓣酱，豆瓣酱放一边煸炒一下</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806061969301.jpg>第8步: 加入黄酒再把豆瓣酱炒出红色再和鸡块一起炒均匀</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806062901289.jpg>第9步: 加入半碗开水大火煮开转小火，盖盖焖炖15分钟</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806062940413.jpg>第10步: 15分钟后不盖盖，开大火收汁</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806062934623.jpg>第11步: 汁快收干时放入青椒块，翻炒一下，淋上藤椒油就可以出锅了。</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162806063416133.jpg>第12步: 麻辣鸡块就做好了，肉质鲜嫩味道香。“亚麻公社胡麻油，植物油中的脑白金”</a>
