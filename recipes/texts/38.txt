recipe_name: 奥尔良烤鸡翅根
practice: 烤
flavor: 香辣味
time: 30分钟
grade: 新手尝试
materials: 主料鸡翅根10只辅料蜂蜜1勺食盐1小勺纯牛奶20克新奥尔良腌制粉20克
step: <a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403874083922.jpg>第1步: 准备好食材（PS:牛奶漏拍了)</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403874062661.jpg>第2步: 鸡翅根清洗干净， 用盐水浸泡15分钟，把鸡肉中的血水泡掉，用于去除鸡肉中的腥味</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403874169025.jpg>第3步: 把泡好的鸡翅根捞出来，控干水分放入碗中，放入新奥尔良腌制粉</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403874054664.jpg>第4步: 再倒入20克纯牛奶，让鸡翅根更加水嫩，同时有淡淡的奶香，吃起来更加香醇；当然也可以将牛奶换成清水</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403874033169.jpg>第5步: 将鸡翅根抓揉均匀，盖上保鲜膜，腌制过夜，腌制时间越长越入味，时间长短根据个人喜好来定即可；中途翻下面，更易入味</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403873933164.jpg>第6步: 取出炸篮，把腌制好的鸡翅根放在里面，200度，烤10分钟；余下的腌汁留着备用</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403874562775.jpg>第7步: 取出炸篮，表面刷腌汁，翻面，再刷一遍，这样上色好又能防止表面风干</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403874540610.jpg>第8步: 温度不变，继续烘烤15分钟，想要颜色好看，中途可取出翻面、刷汁</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403874937365.jpg>第9步: 舀一勺蜂蜜放入碗中，与余下的腌汁拌匀</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403875032875.jpg>第10步: 刷在鸡翅根的表面</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_164403875151598.jpg>第11步: 送入空气炸锅中，再烤5分钟左右，烤到表面呈金黄色即可</a>
