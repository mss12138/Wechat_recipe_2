recipe_name: 番茄藕丁
practice: 炒
flavor: 酸甜味
time: 15分钟
grade: 新手尝试
materials: 主料莲藕300克番茄2个彩椒（红、黄）30克洋葱20克辅料食盐2克白糖2克番茄沙司1勺格琳诺尔食用油少许
step: <a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160242078407.jpg>第1步: 先把需要的食材提前准备好。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160242133481.jpg>第2步: 洋葱切丝备用。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160242200955.jpg>第3步: 莲藕去皮洗净先切薄片后再切小丁备用，切好的藕丁放入清水中避免莲藕变色。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160242412105.jpg>第4步: 西红柿顶部划十字，放入碗中倒入开水烫去外皮后切小丁备用。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160242562324.jpg>第5步: 彩椒洗净控干水分同样切小丁备用。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160242650940.jpg>第6步: 提前烧一锅开水，水开后放入提前切好的莲藕丁焯水，然后捞出过冷水控干水分。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160242777543.jpg>第7步: 锅中倒入少许格琳诺尔有机亚麻籽油放入洋葱炒香。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160242873306.jpg>第8步: 倒入切好的西红柿丁，中小火炒制西红柿开始出汤汁，如果觉得太干可以加少许清水盖上盖子焖出汤汁。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160243012297.jpg>第9步: 炒至西红柿软烂汤汁明显浓稠即可。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160243139833.jpg>第10步: 倒入提前切好的藕丁翻炒均匀，让藕丁均匀裹上西红柿汤汁。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160243292268.jpg>第11步: 倒入彩椒翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160243416840.jpg>第12步: 加入一勺番茄沙司。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160243568695.jpg>第13步: 加2克左右的白糖提鲜。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160243784188.jpg>第14步: 2克食盐调味。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160243884041.jpg>第15步: 大火快速翻炒均匀即可出锅。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163160243932662.jpg>第16步: 格琳诺尔有机亚麻籽油，宝宝的聪明油。</a>
