recipe_name: 吃得两眼放光，超简单五花肉方子，巨过瘾
practice: 焖
flavor: 咸鲜味
time: 60分钟
grade: 未知
materials: 主料五花肉1块鹌鹑蛋500克豆皮3-4张辅料葱姜适量清水适量八角若干桂皮2块香叶若干冰糖适量生抽适量老抽适量蚝油适量啤酒1罐
step: <a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162203898606217.jpg>第1步: 五花肉洗净切薄片。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162203898830493.jpg>第2步: 准备豆腐皮儿（切成条打成结），鹌鹑蛋。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162203899048689.jpg>第3步: 锅里不放油，倒入五花肉煸炒出多余油脂，炒至微微焦黄盛出备用。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162203899157167.jpg>第4步: 葱姜放入砂锅底部打底。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162203899206287.jpg>第5步: 之后倒入五花肉，放冰糖，淋入生抽 老抽 蚝油，放八角，桂皮，香叶，在倒入鹌鹑蛋。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162203899373527.jpg>第6步: 倒入一罐啤酒，在加入适量清水，大火煮开后，盖盖转小火焖20分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162203899514171.jpg>第7步: 在将豆皮儿放入砂锅中，继续焖10分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162203899797071.jpg>第8步: 让豆皮儿和其他所有食材都饱吸汤汁。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162203899992308.jpg>第9步: 可以尝一下咸淡，如果觉得淡的话，再加适量的盐;如果觉得味道正好的话，就可以盛出开吃了。</a>
