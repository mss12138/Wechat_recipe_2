recipe_name: 湖南小炒肉
practice: 炒
flavor: 香辣味
time: 10分钟
grade: 初中水平
materials: 主料五花肉50克猪里脊肉160克青椒150克红椒1根辅料蒜末少许豆豉少许盐少许植物油适量味达美臻品特级生抽1茶勺味达美臻品蚝油1茶勺玉米淀粉少许
step: <a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162549777863963.jpg>第1步: 准备好所需的食材，将五花肉和里脊肉洗净后分别切成薄片；将青红椒洗净后切成滚刀块；豆豉淋上少许植物油浸泡一下，这样口味更好；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162549777616345.jpg>第2步: 将里脊肉用玉米淀粉和味达美臻品蚝油抓匀腌制一下，表面淋上少许植物油；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162549777742327.jpg>第3步: 准备好蒜末；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162549777693890.jpg>第4步: 提前准备好米其林餐厅主厨推荐的干饭小炒汁：在小蝶中倒入味达美臻品蚝油和特级生抽各1茶勺，并调匀；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162549777540267.jpg>第5步: 炒锅烧热后，不要放油，小火直接将青红椒干煸至虎皮状，放少许盐翻炒入味，再淋入少许植物油炒匀后盛出；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162549777693425.jpg>第6步: 炒锅洗净后烧热放入少许植物油，下五花肉小火煸炒出油脂；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162549785946787.jpg>第7步: 再放入蒜末、豆豉炒出香味，再放入里脊肉翻炒变色；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162549786895487.jpg>第8步: 淋入调好的小炒汁翻炒入味；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162549787505339.jpg>第9步: 出锅前倒入炒好的青红椒；</a>
<a href=https://st-cn.meishij.net/rs/23/197/5111773/n5111773_162549789497029.jpg>第10步: 一盘香辣下饭的湖南小炒肉就做好啦！</a>
