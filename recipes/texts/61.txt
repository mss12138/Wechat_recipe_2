recipe_name: 香橙鸡翅【宝宝辅食】
practice: 煎
flavor: 咸鲜味
time: 30分钟
grade: 新手尝试
materials: 主料鸡翅中10个辅料橙子2个
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315613814849.gif>第1步: 鸡翅划斜刀，肉厚的背面部分和鸡皮上面都可以划一下，这样更能腌入味。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315619223694.gif>第2步: 鸡翅里加上姜片、蒜片、2勺酱油、1勺老抽、1勺素蚝油、1勺蜂蜜和适量鲜菇粉调味。这个调味我自己吃着是刚刚好的，你们可以根据孩子的口味适当增减点调料。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315618893594.gif>第3步: 用手把各种调料和鸡翅抓匀，让鸡翅充分的裹上调料。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315625124245.gif>第4步: 过年剩下的橙子对半切开，把橙汁挤到放鸡翅的碗中，同样是抓拌均匀了。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315627796494.gif>第5步: 盖上保鲜膜，让鸡翅腌制2小时以上，要是想入味的，可以多腌制一会儿。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315628226162.gif>第6步: 再拿一个橙子，切片备用。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315631453813.gif>第7步: 等到鸡翅腌好的时候取出来，放入烤盘，把切好的橙子片盖在鸡翅表面。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315632272639.gif>第8步: 提前预烤箱，将烤盘送到烤箱里，上下火200度烤30分钟。

*如果是用空气炸锅做，同样是200度，先烤10-12分钟，看一下表面上色的情况，表面金黄就给鸡翅翻个面，顺便把橙子片揭下去。再烤12分钟左右，熟透就行了。
因为各家空气炸锅的火候都不一样，所以这个温度时间，要灵活一些调节。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164315634572511.gif>第9步: 烤制10分钟之后，就能揭掉橙子片了。继续烘烤，烤好取出来，装盘就行。</a>
