recipe_name: 可乐鸡翅
practice: 焖
flavor: 甜味
time: 30分钟
grade: 中级掌勺
materials: 主料鸡中翅8个可乐1罐辅料食盐少许料酒2勺花雕酒少许生抽1勺老抽1勺熟白芝麻少许
step: <a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162066501883236.jpg>第1步: 鸡翅正反两面各切2刀，这样容易入味好熟，也容易出血水</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162066501987014.jpg>第2步: 鸡翅加入清水，浸泡20分钟，让鸡翅出血水，尽量多清洗几遍，直到血水洗干净为止，然后挤干水分备用，洗出血水的目的是让鸡翅吃起来不会腥。</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162066502076475.jpg>第3步: 鸡翅加入少许食盐、1勺料酒、半勺生抽、半勺老抽、少许花雕酒（既可以增色，也可以去腥），搅拌均匀，然后盖上保鲜膜，腌制20分钟</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162066502270315.jpg>第4步: 热锅热油，将鸡翅放入锅中，小火将鸡翅表面煎到定型，切记不可以乱翻，微黄后翻至另一面，一定要煎出鸡翅中多余的油脂，这样才更香，直到将鸡翅煎至两面金黄，煎熟煎透，这样加入可乐的时候鸡翅才会回软，有胶质感。</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162066502452718.jpg>第5步: 加入半勺生抽、半勺老抽、1勺料酒，1罐可乐、少许食盐，水开后转中小火焖10分钟，切记不可以开大火，因为可乐中有很多糖，开大火很容易发苦</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162066502551005.jpg>第6步: 待汤汁完全收入鸡翅中，汤汁粘稠后即可出锅</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162066502714648.jpg>第7步: 表面撒上一层熟白芝麻，开吃</a>
