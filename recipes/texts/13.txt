recipe_name: 凉拌西兰花
practice: 拌
flavor: 其它口味
time: 10分钟
grade: 新手尝试
materials: 主料西兰花1棵新鲜香菇3个胡萝卜半个辅料六月鲜12克轻盐昆布酱油1.5汤匙芝麻油1汤匙米醋1汤匙玉米油1汤匙盐2茶匙
step: <a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795347353157.jpg>第1步: 主要食材合集。我买的胡萝卜比较大就切了一半，小胡萝卜可以用一个。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795347974678.jpg>第2步: 西兰花分成小朵。切开时要从茎那里切开，然后分开。如果从花头那切，容易切碎。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795347912347.jpg>第3步: 切好的西兰花泡水5-10分钟，水中放1茶匙盐。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795347965092.jpg>第4步: 胡萝卜和香菇洗干净切成薄片。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795347933935.jpg>第5步: 烧一锅水，水开后放1汤匙玉米油。再加半茶匙盐，然后搅匀。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795347980539.jpg>第6步: 先将香菇和胡萝卜倒下去煮两分钟。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795348697203.jpg>第7步: 再倒西兰花煮一分钟。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795348572635.jpg>第8步: 将全部材料捞起，控干水。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795348563761.jpg>第9步: 加1-2汤匙酱油。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795348460122.jpg>第10步: 再加一汤匙芝麻油或橄榄油。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795348460416.jpg>第11步: 如果嫌味道寡淡，可以再加1汤匙米醋或果醋。</a>
<a href=https://st-cn.meishij.net/rs/208/121/13280458/n13280458_164795484409840.jpg>第12步: 拌匀就好了。</a>
