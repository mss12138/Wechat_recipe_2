recipe_name: 健康蒸菜，好吃不腻，香菇蒸鸡翅
practice: 蒸
flavor: 咸鲜味
time: 30分钟
grade: 初级入门
materials: 主料鸡翅4只香菇3个土豆1个辅料葱3根姜丝一把料酒2勺蚝油1勺生抽2勺玉米淀粉1勺盐1勺
step: <a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162221218825363.jpg>第1步: 准备材料。土豆清洗干净后用削皮刀去皮。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162221219724882.jpg>第2步: 香菇清洗干净去柄。剁开的鸡翅清晰干净备用（霖爸已经提前剁好了，如果是整只的话用红色处理生肉用菜板剁开就好）。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162221222269783.jpg>第3步: 葱切成葱花，姜切丝。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162221271001980.jpg>第4步: 鸡翅中加入葱段和姜丝，然后倒入料酒2勺去腥。倒入2勺生抽酱油调味。倒入一勺蚝油提鲜。加入一勺盐、1勺玉米淀粉。用筷子搅匀腌制15分钟。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162221273807818.jpg>第5步: 腌制期间，土豆用绿色蔬果菜板切成薄片铺底，香菇也切片。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162221275545957.jpg>第6步: 把切好的土豆和香菇放入盘中。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162221287473053.jpg>第7步: 腌制好的鸡翅，排在香菇上，腌制的料汁也一起倒入，葱姜不要。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162221288400841.jpg>第8步: 冷水上锅，大火蒸25分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/121/39/13134871/n13134871_162221290911785.jpg>第9步: 蒸完取出，撒上葱花即可。</a>
