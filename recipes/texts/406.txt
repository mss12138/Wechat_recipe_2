recipe_name: 梭子蟹蒸蛋【宝宝辅食】
practice: 蒸
flavor: 家常味
time: 10分钟
grade: 新手尝试
materials: 主料鸡蛋2个梭子蟹1只辅料葱丝、姜丝、酱油适量
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163022957702786.jpg>第1步: · 参考月龄 ·
16个月以上，对食材不过敏的宝宝
· 食材准备 ·
鸡蛋2个 / 梭子蟹1只
辅料：葱丝、姜丝、酱油</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163022957803649.gif>第2步: 新鲜的梭子蟹清洗干净，掰开</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163022957873045.gif>第3步: 去掉里面的腮和不能食用的内脏。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163022957869165.gif>第4步: 处理好的螃蟹对半切开，摆入碗中。宝宝版的就你就蒸好把肉抠出来，摆在蛋羹上。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163022957899917.gif>第5步: 鸡蛋打入碗中，搅散。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163022957833940.gif>第6步: 加入1.2倍的清水和盐，同蛋液一起搅拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163022957877828.jpg>第7步: 然后把蛋液，过筛到放了螃蟹或者螃蟹肉的碗中。过筛后的蛋液蒸出来口感更加滑嫩细腻，平时在做一些食谱的时候，也可以用上这个小技巧。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163022957983528.gif>第8步: 锅中水沸转中小火，上锅蒸制15分钟，关火后继续焖10分钟。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163022957939897.gif>第9步: 蒸蛋熟透取出，淋上点酱油，撒上葱丝、姜丝去更好调味。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163022957998487.gif>第10步: 给孩子吃的，你们就把肉抠出来，熟蟹肉铺在蒸蛋上，多些鲜美的滋味。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_163022957933718.jpg>第11步: 螃蟹的鲜味都完全浸入到蛋液的内部了，肉和蛋混在一块吃进嘴里，就是说不出来的香和爽快！</a>
