recipe_name: 香辣烧鸡
practice: 烧
flavor: 香辣味
time: 60分钟
grade: 初中水平
materials: 主料三黄鸡800克辅料植物油适量
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163178816918639.jpg>第1步: 主料：三黄鸡800克
辅料：植物油、香辣烧汁</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163178816979277.jpg>第2步: 把烧鸡洗净切小块，姜洗净切片，香葱洗净切段</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163178816903389.jpg>第3步: 凉水下锅大火烧开，捞出来洗去血沫，香葱洗净切段</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163178816943081.jpg>第4步: 炒锅烧8成热，加入植物油，下焯水的鸡块煸炒出水分出香味，这样水分去掉了，加烧汁好吃</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163178816950955.jpg>第5步: 加入黄肖氏香辣烧汁，再次翻炒均匀上色，炒出酱香味以后（可以多加点，这样酱香味浓更好吃）</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163178817002223.jpg>第6步: 多翻炒几下，炒出香味，不用加调味料了，因为黄肖氏香辣烧汁已经够味了（几十种香辛料精心配比，五味调和融为一汁。可以替代盐糖酱醋，烧菜只需要放这一汁，就能做出美味菜肴！）</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163178817076317.jpg>第7步: 加入清水或者开水大火烧开（水一定要能没过所有的鸡块为准，因为烧肉比较耗水），开大火烧开</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163178817037621.jpg>第8步: 然后开小火，盖上盖子焖烧30分钟左右，最好能掌握好火候和时间，煮的差不多可以打开锅盖（一般不用加盐）</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163178817045793.jpg>第9步: 然后把火调大准备收汁，收汁的过程要不断的搅拌，防止烧焦，等到汁收差不多就可以起锅了，一道美味的酱烧鸡就做好了</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163178817014562.jpg>第10步: 出锅前，可以撒上香葱段提味提香气，增加食欲，做好的香辣烧鸡是不是色香味俱全！</a>
