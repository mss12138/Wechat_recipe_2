recipe_name: Q弹好吃的肉皮冻
practice: 炖
flavor: 家常味
time: 60分钟
grade: 初级入门
materials: 主料肉皮800克辅料八角2棵姜1块小葱3棵酱油适量盐少许大蒜1头醋适量生抽适量辣椒油少许
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927365209873.jpg>第1步: 肉皮准备好，用量随意；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927357380238.jpg>第2步: 将肉皮内侧的肥肉切掉，肉皮再切小块；如果不嫌弃有肥肉并且爱吃肥的可不去掉，凉后有凉拌白肉的口感；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927357249568.jpg>第3步: 入凉水锅，水开后煮5分钟捞出，去除掉多余的油脂和杂质；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927357222673.jpg>第4步: 另起锅，放适量水、酱油、盐、小葱段、姜块、八角；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927357381478.jpg>第5步: 煮开后将焯过水的肉皮入锅；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927357369565.jpg>第6步: 小火，盖盖子，焖炖至肉皮熟烂，用筷子能轻轻夹断；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927357526667.jpg>第7步: 捡去八角、姜片、葱段；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927357548191.jpg>第8步: 准备净碗或者保鲜盒，将肉皮和汤平均倒入碗中；盖上盖子，入冰箱冷藏至完全凝固；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927357594275.jpg>第9步: 准备蘸料：大蒜剁末、小葱切末；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927357557817.jpg>第10步: 倒适量生抽、醋，能吃辣的再来点儿辣椒油或者小米辣，混合均匀；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927357579849.jpg>第11步: 这是凝固好的肉皮冻，用刀尖插入肉皮冻和碗壁之内，倒扣，轻松脱模；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162927357530550.jpg>第12步: 按自己的喜欢，切块、切片、切条均可，可直接食用，亦可蘸着蒜醋汁食用。</a>
