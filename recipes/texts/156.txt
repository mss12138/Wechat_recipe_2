recipe_name: 蒸饺
practice: 蒸
flavor: 家常味
time: 30分钟
grade: 新手尝试
materials: 主料大白菜800克豆腐400克肉末300克大葱1棵香菇200克鸡蛋3个红薯粉条1把香菜50克中裕多用途麦芯粉1000克辅料盐5克生抽4勺蚝油1勺甜面酱2勺十三香粉3克芝麻油1勺食用油4勺酵母粉10克
step: <a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782448534737.jpg>第1步: 面粉和酵母粉一起放入面盆中加入适量温水和面，软硬适中即可，静置发酵。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782448567695.jpg>第2步: 豆腐切小丁，锅中加入2勺食用油，将豆腐丁煎制金黄色，盛出备用，锅中留下底油继续加热，将鸡蛋炒碎盛出备用。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782449348776.jpg>第3步: 蘑菇焯水后切碎，粉条煮熟切小段备用。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782449063610.jpg>第4步: 起油锅放入2勺食用油，放入肉末和香菇碎翻炒。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782449255575.jpg>第5步: 加入甜面酱、酱油和十三香粉翻炒。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782449478241.jpg>第6步: 盛出放入馅料盆里，大葱切碎、鸡蛋碎、豆腐丁、粉条一同放入盆中。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782449799689.jpg>第7步: 加入盐和蚝油拌匀。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782450832733.jpg>第8步: 白菜切碎攥出水分，放入盆中，香菜切碎放入盆中，加入芝麻油再次拌匀。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782450151933.jpg>第9步: 面已发酵好，分成小剂子，取一份擀成大片。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782450407380.jpg>第10步: 包上馅料。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782450751150.jpg>第11步: 包成自己喜欢的样式，再次饧发15分钟。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782450839826.jpg>第12步: 凉水上蒸锅，蒸制18分钟，焖3分钟出锅食用。</a>
<a href=https://st-cn.meishij.net/rs/181/98/13899681/n13899681_163782452169653.jpg>第13步: 热腾腾的蒸饺，我的冬日限定。</a>
