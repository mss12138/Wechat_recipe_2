recipe_name: 鲜嫩开胃的番茄比目鱼
practice: 炒
flavor: 鱼香味
time: 15分钟
grade: 初级入门
materials: 主料比目鱼2块番茄2个辅料大蒜5瓣油适量盐少许水少许
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_163117480410523.jpg>第1步: 比目鱼放室温下解冻，西红柿用肉厚汁多的；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_163117480536520.jpg>第2步: 将化冻的比目鱼切3、4公分的大块；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_163117480542214.jpg>第3步: 西红柿切小块，大蒜切厚片；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_163117480456978.jpg>第4步: 不粘平底煎锅加热，倒适量油，油温7、8成热时将鱼皮朝下均匀码放在锅底；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_163117480416382.jpg>第5步: 待底部定型后轻轻翻动，煎另一面；同时把蒜片入锅煎出香味；待全部成熟后将鱼块夹出锅备用；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_163117480595440.jpg>第6步: 锅中留底油，西红柿块入锅，小火翻炒；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_163117480514836.jpg>第7步: 西红柿变软、出汤、略变浓稠，将打成卷的西红柿皮夹出不要；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_163117480553919.jpg>第8步: 加少许水增加汤汁，比目鱼块入锅，用铲子轻轻翻炒混合，关火；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_163117480500421.jpg>第9步: 红润的西红柿，雪白的比目鱼块，酸中带着微咸，更加衬托出鱼肉的鲜甜香嫩。</a>
