recipe_name: 泰式甜辣鸡肉沙拉
practice: 拌
flavor: 酸辣味
time: 15分钟
grade: 初级入门
materials: 主料黄瓜1节生菜6片鸡胸肉1块辅料葱姜少许食盐1小勺料酒1大勺柠檬 1个小青柠2个圣女果5个蛋黄酱1勺泰式甜辣酱3勺
step: <a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162623473923014.jpg>第1步: 准备好鸡胸肉和其它食材</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162623473986522.jpg>第2步: 鸡胸肉洗净，加葱姜放入清水锅中</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162623474057138.jpg>第3步: 煮开后，撇去表面浮沫，加入1勺料酒，然后转小火，再煮8分钟左右</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162623474118057.jpg>第4步: 开盖，用筷子戳下去，没有血水渗出；关火，盖上锅盖焖10分钟</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162627694717946.jpg>第5步: 取出鸡肉，放入事先备好的冰水中，浸泡5~8分钟</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162623474415709.jpg>第6步: 取出，顺着纹路用手撕成鸡丝放入盆中，备用</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162623474338240.jpg>第7步: 这时来处理准备好的配菜：生菜、黄瓜和圣女果洗净；再用食盐将把小青柠和柠檬表面揉搓几遍后冲洗干净</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162623474694091.jpg>第8步: 接着将圣女果对半切开，青柠、柠檬和黄瓜分别切片</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162623474691632.jpg>第9步: 然后一起放入鸡丝盆中，再挤入2勺泰式甜辣酱</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162623474767482.jpg>第10步: 用筷子翻拌均匀</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162623474934649.jpg>第11步: 取一个干净的盘子，将生菜垫底，放上拌好的鸡丝和果蔬，再挤上1勺丘比蛋黄酱，丰富口感</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162623474841366.jpg>第12步: 最后再挤上1勺泰式甜辣酱，增加甜辣味道即可</a>
