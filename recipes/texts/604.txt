recipe_name: 家常蜜桃酱
practice: 煮
flavor: 甜味
time: 60分钟
grade: 新手尝试
materials: 主料水蜜桃4个-5个盐适量柠檬1个辅料白砂糖150-200克
step: <a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213453790.jpg>第1步: 水蜜桃先用盐搓洗干净，再去掉果皮和果核，果皮和果核待会可以放一起煮，颜色会更好看。桃子果肉切成小块备用。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213487574.jpg>第2步: 柠檬1个挤出柠檬汁备用。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213479036.jpg>第3步: 果肉里放入白砂糖200克，挤出的柠檬汁。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213460624.jpg>第4步: 盖上保鲜膜，腌制1-2小时。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213435630.jpg>第5步: 先把果皮和果核放一点水，小火煮10-15分钟。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213436259.jpg>第6步: 煮出颜色后，丢掉果皮和果核。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213514893.jpg>第7步: 放入腌制好的桃子果肉，小火慢煮至浓稠。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213567503.jpg>第8步: 中途注意撇去浮沫。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213561445.jpg>第9步: 我煮过头了，有点糊底，颜色还特别深。_(|3」∠)_</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213535696.jpg>第10步: 放凉后倒入罐子。（注：罐子记得提前用开水泡过一遍，晾干水分）</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213566092.jpg>第11步: 家常蜜桃酱，完成O(∩_∩)O</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_162549213655860.jpg>第12步: 甜甜蜜蜜，清清爽爽，可以搭配喜欢的东西来吃哦。</a>
