recipe_name: 红烧金鲳鱼
practice: 煎
flavor: 家常味
time: 30分钟
grade: 未知
materials: 主料金鲳鱼1条辅料生姜10克小米辣4个
step: <a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300682762574.jpg>第1步: 首先准备好所需要的食材，并做基础处理；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300682797197.jpg>第2步: 鱼身两面均匀打上花刀；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300682839379.jpg>第3步: 放入姜丝、葱白，调入料酒、盐、胡椒粉，涂抹均匀，腌制片刻；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300682936371.jpg>第4步: 锅热倒油，油热放入金鲳鱼，煎至一面定型，能轻松拨动后翻面，注意鱼头鱼尾也要煎到，避免鱼腥；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300682923266.jpg>第5步: 直至煎至两面金黄后，装盘出锅；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300683061916.jpg>第6步: 就遮余油，倒入辣椒碎和姜丝，再倒入辣椒酱炒匀；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300683178582.jpg>第7步: 倒入清水，调入生抽、蚝油煮沸，下入金鲳鱼；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300683188336.jpg>第8步: 转中小火炖煮入味，中途可用勺舀取汤汁淋在鱼身，等到汤汁明显减少时，再次将金鲳鱼装盘出锅；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300683315444.jpg>第9步: 锅中倒入少许水淀粉，煮至浓稠；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300683323104.jpg>第10步: 最后淋至碗中，撒上葱花即可；</a>
