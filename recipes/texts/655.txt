recipe_name: 0基础做出土豆软烂牛腩酥烂的经典家常美
practice: 炖
flavor: 咸鲜味
time: 数小时
grade: 未知
materials: 主料牛腩1块大土豆1个番茄2个辅料姜片3-4片冰糖若干葱花适量料酒适量洋葱小半个蒜瓣若干八角2-3个香叶2片盐适量胡椒粉适量生抽适量
step: <a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162482897126086.jpg>第1步: 牛肉切块，冷水下锅，放姜片料酒去腥。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162482897231310.jpg>第2步: 煮出血沫后捞出冲几遍凉水，番茄切块。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162482897504086.jpg>第3步: 洋葱切碎，蒜拍扁，再再准备几个八角和香叶。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162482897770589.jpg>第4步: 砂锅里放少许油，倒入蒜 洋葱 八角 香叶炒香，然后倒入番茄块翻炒，炒出汤汁。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162482897996007.jpg>第5步: 倒入牛肉块一起翻炒。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162482898002140.jpg>第6步: 加入适量的温水，大火煮开后转小火盖盖慢炖七八十分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162482898279579.jpg>第7步: 再放入土豆块，加盐 胡椒粉 冰糖 生抽调味，继续小火慢炖20分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162482898476193.jpg>第8步: 出锅前再撒点葱花</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162482898618322.jpg>第9步: 牛肉烂而不柴，土豆绵软，汤汁酸甜浓郁，超好喝。</a>
