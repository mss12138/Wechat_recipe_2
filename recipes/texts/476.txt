recipe_name: 小笋炒肉
practice: 炒
flavor: 香辣味
time: 5分钟
grade: 中级掌勺
materials: 主料小笋500克猪肉300克辅料红辣椒2个大蒜1个生姜2片酱油1勺亚麻公社胡麻油适量盐适量
step: <a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_162843693107904.jpg>第1步: 食材： 小笋、猪肉、生姜、大蒜、红辣椒、亚麻公社胡麻油、盐、酱油、十三香。其中十三香可以更加自己喜欢增减，加入猪肉里，会更香哦。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_162843697215590.jpg>第2步: 小笋去壳，清洗干净，煮锅倒入适量的水，煮沸之后，放入锅里汆烫，捞出切好，备用。我这个小笋是提前汆烫好放冰箱，所以只需要取出清洗干净就可以。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_162843698493980.jpg>第3步: 猪肉清洗干净，切薄片。新鲜优质的猪肉，有肉腥味，用手按压，有弹性。肥瘦相间的五花肉，炒小笋是最好吃。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_162843701243596.jpg>第4步: 红辣椒、生姜、大蒜，分别清洗干净，切好备用。辣椒可以根据喜欢，增减，别太辣哦。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_162843699426352.jpg>第5步: 炒锅烧热，倒入小笋，小火炒锅，记住：不需要放油，就是干炒，把水炒干，这样的小笋才香，鲜嫩，好吃。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_162843702577796.jpg>第6步: 锅里倒入适量的亚麻公社胡麻油，倒入猪肉、生姜、大蒜爆炒，猪肉油呲呲，很香</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_162843706099178.jpg>第7步: 把小笋倒入锅，与猪肉翻炒，接着放红辣椒，加红辣椒不仅颜色好看，更香，更辣，很下饭。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_162843704188216.jpg>第8步: 放酱油、盐、翻炒入味，最后放香葱，炒均即可。香喷喷，端上桌了。</a>
<a href=https://st-cn.meishij.net/rs/97/32/4758097/n4758097_162843714843586.jpg>第9步: 亚麻公社胡麻油，植物油中的脑白金</a>
