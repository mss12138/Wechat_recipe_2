recipe_name: 花甲粉丝煲
practice: 煮
flavor: 香辣味
time: 15分钟
grade: 中级掌勺
materials: 主料蛤蜊500克金针菇300克粉丝一把生姜几片大蒜几瓣洋葱半个香菜两根青红椒各两个辅料蚝油一勺酱油一勺料酒适量盐适量
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163106489904964.jpg>第1步: 食材都准备好。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163106490081286.jpg>第2步: 花甲买来用清水加少许盐或几滴油泡2小时以上，让它吐出细沙，天热花甲容易死，要放进冰箱，龙口粉丝提前1小时用水泡软。
将蒜姜葱切末，洋葱，青红椒切小块。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163106490011890.jpg>第3步: 另起油锅，油要比平时烧菜多一倍。下洋葱爆香。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163106490040440.jpg>第4步: 加入蒜末翻炒几下，再把姜，辣椒，葱花一起全部倒入翻炒。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163106490041981.jpg>第5步: 再加入酱油、蚝油、盐。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163106490290818.jpg>第6步: 炒到这样，酱汁算是熬好了，非常香！</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163106490368017.jpg>第7步: 泡好的粉丝垫底下，再铺上金针菇，最后把蛤蜊铺在上面。（我用的是小沙锅，没有的可以用普通的锅子）</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163106490476148.jpg>第8步: 再将熬好的酱汁淋在蛤蜊上。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163106490465861.jpg>第9步: 加入一勺料酒，再倒入水，水没过食材的一半即可。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163106490426558.jpg>第10步: 小砂锅直接放在煤气灶上，大火煮开转中火煮5-8分钟。一道秒杀路边摊的花甲粉丝就出锅啦。最后可撒上香菜做装饰。汤汁都能喝完的花甲粉丝，你们学会了吗？</a>
