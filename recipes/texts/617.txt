recipe_name: 轻养瘦身虾仁豆腐丝瓜盅
practice: 蒸
flavor: 咸鲜味
time: 5分钟
grade: 初级入门
materials: 主料丝瓜1~2条基围虾20只嫩豆腐2大勺辅料六月鲜12克轻盐酱油3大勺盐(可不加)少许白砂糖(可不加)1大勺味达美臻品料酒2大勺味达美清香米醋1大勺
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162515313298601.jpg>第1步: 丝瓜刮去外皮切4厘米的段，我切了15段。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162515313225550.jpg>第2步: 丝瓜段从一端挖出内瓤，但不要挖穿底部。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162515313222062.jpg>第3步: 15只基围虾去头剥壳，留下最后一节虾尾不要剥。另取5只虾剥出虾仁切小粒。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162515313388681.jpg>第4步: 切粒的虾仁放入碗中，加料酒和轻盐酱油各1大勺。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162515313253131.jpg>第5步: 挖入2大勺豆腐，加一半盐糖，搅拌均匀待用。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162515313333250.jpg>第6步: 留着尾巴的虾仁也加入剩下的盐、糖、1大勺料酒和1大勺轻盐酱油，搅拌均匀，腌制20分钟。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162515313220051.jpg>第7步: 取虾仁从背部剖开，头尾不要剖断，把虾尾从剖出的孔洞中穿出，虾尾就直立在虾球中。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162515313223781.jpg>第8步: 舀一虾肉豆腐填入丝瓜盅，不要填满。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162515343780984.jpg>第9步: 再摆一个虾球，整理虾尾直立。冷水上锅，大火煮开再蒸5分钟。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162515344369181.jpg>第10步: 最后用1大勺轻盐酱油和米醋调一个味汁浇淋到丝瓜盅上，也可以根据自己口味调一个蘸料蘸着吃。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162515344910418.jpg>第11步: 颜值不错吧？翠绿的丝瓜盅盛着红色的虾球，颜色赏心悦目，营养丰富的低热量高蛋白瘦身餐，值得尝试。</a>
