recipe_name: 电饭煲盐焗鸡翅
practice: 煮
flavor: 咸鲜味
time: 60分钟
grade: 中级掌勺
materials: 主料鸡翅根1000克辅料葱姜适量栀子4个料酒适量食用盐适量盐焗鸡粉50克
step: <a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162238004953847.jpg>第1步: 鸡翅根清洗干净，放入冷水锅中，放入葱段、生姜，倒入一点料酒去腥</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162238005091479.jpg>第2步: 锅开以后将表面浮沫撇干净，再将鸡翅根捞出，放入冷水中清洗干净，备用</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162238005212945.jpg>第3步: 取四个栀子，用擀面棍敲碎，然后放入纱布中系紧包一个小料包，放入盆中</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162238005486700.jpg>第4步: 盆中加入葱姜、食用盐，再倒入开水浸泡几分钟</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162238005500611.jpg>第5步: 将颜色泡出来后，加入50克的盐焗鸡粉，搅拌均匀，放着备用</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162238005744570.jpg>第6步: 把清洗干净的翅根放入调料盆中，搅拌均匀，然后盖上保鲜膜，放入冰箱腌制一个晚上，让鸡翅入味上色</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162238005966617.jpg>第7步: 电饭煲内胆底部摆入葱段、姜片，然后将腌制好的鸡翅根码放到电饭煲中，再倒入一碗腌鸡翅根的汤汁，不要倒太多</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162238006145260.jpg>第8步: 将内胆放入锅中，按煮饭键，煮30分钟，将鸡翅根煮熟煮透</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162238006571954.jpg>第9步: 将鸡翅根从电饭煲中取出，装入盘中，晾至微凉，开吃</a>
