recipe_name: 清蒸小龙虾
practice: 蒸
flavor: 家常味
time: 15分钟
grade: 中级掌勺
materials: 主料小龙虾400克辅料食用油1大勺香葱1把生姜1块蒜头2瓣香菜2根生抽2勺料酒3勺蚝油半勺白糖少许
step: <a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895441977668.jpg>第1步: 准备好食材，小龙虾，我一般比较喜欢购买这种活的青壳龙虾，虾肉多，虾壳薄；相比于又脏又硬的红壳龙虾，既好吃又好清洗</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895441480992.jpg>第2步: 第一步处理小龙虾：先用手抓着龙虾背，这样就不用担心被虾钳夹到了；然后用剪刀剪去虾脚，为了好看，可以留着虾须和两只大的虾钳；再将两边的虾腮剪掉，接着用手掐住虾尾巴最中间那节，轻轻一扭一拽，虾线就拽出来了；最后用牙刷，把小龙虾全身刷洗干净；虽然有费事，但吃着更加地放心</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895441908599.jpg>第3步: 我家平时吃龙虾，都会先把虾头剪掉，再把里面的黑色虾胃去掉，然后留下黄色的虾膏，这个可是最鲜的哦；但今儿为了拍菜谱好看，所以就没剪了</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895441977292.jpg>第4步: 全部处理好后，再用清水多清洗几遍，控干水分；配菜洗净：香葱、香菜切段；蒜头切末，一块生姜大半切片，小半切末；备用</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895441952884.jpg>第5步: 接下来调碗蒸虾汁：碗中加入半勺蚝油，2勺生抽和少许白糖</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895441978130.jpg>第6步: 再加入3勺料酒，搅拌均匀，备用；蚝油和生抽都有咸味，就不用放盐了</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895442045280.jpg>第7步: 准备一个大盘子，铺上姜片，葱段和香菜梗，再撒上少许姜蒜末；留一半葱末，备用</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895441584767.jpg>第8步: 码上处理好的小龙虾，再淋上蒸虾汁</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895442500543.jpg>第9步: 把余下的姜、蒜末，再均匀地撕在上面</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895442544560.jpg>第10步: 锅中水开，把龙虾放进去， 盖上锅盖</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895442543238.jpg>第11步: 大火蒸12分钟左右，蒸的时间根据龙虾的量来，但不要过久，以免将虾肉蒸老，影响口感</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_162895442512473.jpg>第12步: 把蒸好的龙虾端出来，再把剩下的葱花和香菜撕在上面； 锅中油烧热，将热油浇在蒸好的龙虾上即可</a>
