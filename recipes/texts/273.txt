recipe_name: 海参蒸肉饼，鲜嫩多汁巨好吃！
practice: 蒸
flavor: 家常味
time: 30分钟
grade: 初级入门
materials: 主料猪肉300克海参1只虾米少许辅料蚝油2勺轻盐昆布酱油2勺小葱2根香菜2根胡萝卜少许胡椒粉少许姜丝少许
step: <a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163292916094144.jpg>第1步: 准备好全部食材。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163292916398891.jpg>第2步: 把猪肉清洗切成大块放入绞肉机。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163292941644727.jpg>第3步: 加入胡萝卜、姜丝、小葱，再来点蚝油、原汁酱油、胡椒粉。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163292943104496.jpg>第4步: 启动绞肉机，打成肉泥，但是不要太久，别太细腻。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163292946576360.jpg>第5步: 然后加入海参、虾米、香菜末，简单拌匀。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163292948467441.jpg>第6步: 倒入深盘摊平，再点缀点香菜叶。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163292950999735.jpg>第7步: 放入五金蒸箱，扣上盖子，装入蒸汽饭煲，水箱注入600ML纯净水。（一定要净化的纯净水哟！！！）</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163292954130908.jpg>第8步: 设置蒸模式，20分钟即可。</a>
<a href=https://st-cn.meishij.net/rs/73/127/10031823/n10031823_163292980978367.jpg>第9步: 出锅后，再撒点葱花，就可以上桌了。</a>
