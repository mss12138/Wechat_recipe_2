recipe_name: 白灼菜心
practice: 烧
flavor: 家常味
time: 15分钟
grade: 未知
materials: 主料菜心300克辅料大蒜1头红尖椒1个蚝油1匙蒸鱼豉油1匙生抽酱油1匙食盐5克鸡精5克白糖10克水淀粉1勺
step: <a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168785914255.jpg>第1步: 新鲜菜心清洗干净，去掉老叶，切去底部1厘米。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168786007649.jpg>第2步: 大蒜拍碎切成末备用。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168786282526.jpg>第3步: 红尖椒切成丝。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168786261042.jpg>第4步: 锅中烧水，放入食盐，食用油，水开后将菜心焯水一分钟。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168786307452.jpg>第5步: 捞出冷水过凉后，摆在盘子里。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168786455928.jpg>第6步: 锅中放油，放入大蒜末小火煸炒出香味。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168786506626.jpg>第7步: 放入蚝油炒匀。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168786699914.jpg>第8步: 随后放入一勺清水。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168786633537.jpg>第9步: 加入蒸鱼豉油，生抽酱油，食盐，鸡精，白糖。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168786771404.jpg>第10步: 最后加入水淀粉一勺，搅匀，使汤汁粘稠即可。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168786856703.jpg>第11步: 将汤汁浇在菜心上。</a>
<a href=https://st-cn.meishij.net/rs/06/99/15024756/n15024756_163168786927057.jpg>第12步: 撒上红椒丝即可享用。</a>
