recipe_name: 土豆烧鸡翅
practice: 烧
flavor: 家常味
time: 60分钟
grade: 初中水平
materials: 主料鸡翅8个土豆1个辅料植物油适量盐适量生抽2大勺老抽2大勺八角1个花椒粒1克草果1个姜10克
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328986149702.jpg>第1步: 主料：鸡翅中8个、土豆一个
辅料：植物油、盐、生抽、老抽、姜、花椒粒1克、草果1个、料酒、辣皮子5个（可选）</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328986269008.jpg>第2步: 把鸡翅洗净，凉水下锅煮来，捞出来待用，姜切片（鸡翅可以提前切花刀，这样好入味）</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328987804617.jpg>第3步: 热锅凉油，油烧到6成，下鸡翅翻炒出香味以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328987883015.jpg>第4步: 加红烧酱油翻炒均匀，再加生抽一起翻炒出酱香味，顿时香气扑鼻，注意如果想颜色更红润，可以用白糖炒上色</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328987852746.jpg>第5步: 沿着锅边淋入适量高度白酒或者料酒翻炒均匀出酒香味</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328987810580.jpg>第6步: 加香料和姜翻炒出香料的香香味以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328987859139.jpg>第7步: 加清水满过鸡翅，因为烧起来特别损耗水，可以清水加多一些</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328987811632.jpg>第8步: 加冰糖和辣皮子大火烧开以后，转中小火烧，最好加锅盖焖烧，大约烧20分钟，中间可以翻炒几次</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328986213948.jpg>第9步: 把土豆洗洗，去皮切块，再用凉水洗洗，这样淀粉少了很多，熟透了不易黏糊糊的</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164329002966205.jpg>第10步: 把土豆下锅翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328987836645.jpg>第11步: 加盐翻炒均匀，继续烧20分钟左右</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328987890519.jpg>第12步: 等到土豆很轻易的夹碎即可，大火收汁</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_164328987904194.jpg>第13步: 最后加鸡精或者味精（可选）起锅装盘，撒上香菜段即可。一道肉感十足的土豆烧鸡翅就做好了，做年菜真不错，算是吉祥如意菜了。</a>
