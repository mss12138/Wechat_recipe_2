recipe_name: 照烧鸡腿排
practice: 煎
flavor: 咸鲜味
time: 15分钟
grade: 初级入门
materials: 主料鸡腿2只胡萝卜半根甜玉米半根西兰花6小朵蘑菇6朵辅料胡麻油2大勺盐少许生抽1大勺老抽1大勺蚝油2大勺米醋1大勺料酒1大勺白砂糖1小勺
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162790525764524.jpg>第1步: 2个鸡腿去骨(我有个视频专门讲鸡腿去骨的，不会的小伙伴可到我主页查找)，撕掉鸡皮，用牙签两面扎孔，以便腌制入味，打断肌肉纤维，减少收缩。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162790525877009.jpg>第2步: 加少许盐，少许糖，1勺料酒去腥，抓捏一会，盖保鲜膜冷藏腌制半小时以上。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162790525885340.jpg>第3步: 调个照烧汁：碗中加糖，1勺米醋，1勺料酒，1勺生抽，1勺老抽，2勺蚝油，少许清水，搅拌均匀即可。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162790525896131.jpg>第4步: 烧一锅开水，不容易熟的玉米先下，玉米煮熟后，再下蘑菇、胡 萝卜、西兰花，所有蔬菜焯熟，捞起沥水。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162790525818306.jpg>第5步: 平底锅烧热下胡麻油，下鸡腿排小火慢煎，煎鸡排时用锅铲按压鸡腿排，煎到底面微有焦黄翻面，另一面也煎到焦黄色就好了，另一块鸡排也同样煎好。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162790525868016.jpg>第6步: 倒入调好的照烧汁，加盖大火煮开，翻面后再次大火煮开.</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162790525801991.jpg>第7步: 接着大火收汁，不要收得太干。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162790526023443.jpg>第8步: 装盘，搭配蔬菜，浇上酱汁，美美地享用吧！</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162790526097423.jpg>第9步: 亚麻公社胡麻油，植物油中的脑白金。</a>
