recipe_name: 虽是纯素菜，却连吃几天都不腻，轻松一炒
practice: 炒
flavor: 咸鲜味
time: 10分钟
grade: 未知
materials: 主料黄豆芽适量香干2块辅料葱蒜适量盐适量鸡精适量
step: <a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162186333054608.jpg>第1步: 准备两块儿香干。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162186333131893.jpg>第2步: 将香干先切片，再切条。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162186333303082.jpg>第3步: 黄豆芽洗净备用。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162186333522879.jpg>第4步: 准备适量的葱蒜。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162186333622494.jpg>第5步: 锅中放油，油热后，倒入一半的葱蒜呛锅。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162186333725550.jpg>第6步: 之后再倒入黄豆芽翻炒。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162186333861305.jpg>第7步: 炒至黄豆芽断生，加入适量盐调味儿。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162186333996968.jpg>第8步: 之后再放入香干一起翻炒，加鸡精提鲜，当最后剩下的葱蒜，倒入锅中继续翻炒。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162186334193047.jpg>第9步: 看着就非常好吃，闻着也很香。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162186334385790.jpg>第10步: 简单易做，好吃又下饭，到黄豆芽炒香干就完成了。</a>
