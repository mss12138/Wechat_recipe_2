recipe_name: 蛏子煎蛋【宝宝辅食】
practice: 炒
flavor: 家常味
time: 10分钟
grade: 新手尝试
materials: 主料蛏子400克胡萝卜20克鸡蛋4个辅料盐、鲜菇粉、葱花、姜片、葱段适量
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616026984.jpg>第1步: · 参考月龄 ·
12个月以上，对食材不过敏的宝宝
· 食材准备 ·
蛏子400克
胡萝卜20克 / 鸡蛋4个
辅料：盐、鲜菇粉、葱花、姜片、葱段</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616132276.gif>第2步: 蛏子清洗干净，清洗蛏子的时候要注意，贝类通常都要吐沙，我买的这个已经吐好沙了，只要再清洗贝壳就可以了。如果是没有吐沙的，要在浸泡蛏子的水中加入适量盐，然后静待吐沙，大概要有2、3个小时。这个期间要不断换水，直到沙子吐净。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616109786.gif>第3步: 蛏子放入沸水锅当中，加入姜片、葱段。这里加入姜片和葱段主要是为了更好去腥。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616119928.gif>第4步: 煮到蛏子开口，把它捞出来。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616101172.gif>第5步: 去掉蛏子的贝壳和黑边，仔细清理干净。这个蛏子你们要是买不到啊，就换成干贝、牡蛎啥的，营养和鲜味也都是有的。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616123109.gif>第6步: 胡萝卜切成细丝，备用。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616228871.gif>第7步: 空碗中打入鸡蛋，搅散。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616297833.gif>第8步: 鸡蛋中加入蛏子肉、胡萝卜丝、葱花、鲜菇粉，搅拌均匀。胡萝卜丝我家孩子更喜欢那种偏脆一点的口感，如果家里宝宝喜欢更软的口感，可以提前焯水一次。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616203901.gif>第9步: 锅中倒油，倒入蛋液，开小火煎制。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616246348.gif>第10步: 煎到底部凝固，翻面。因为这个煎蛋不是很好翻面，所以我干脆就一小块一小块的翻，更加省事儿。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616347112.gif>第11步: 煎到表面微黄，熟透取出。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162840616302363.jpg>第12步: 这个蛏子煎蛋品相诱人，味道鲜香，真的是一上桌就马上消失了半盘子。如今正好是吃蛏的时候，可以试试这道菜，给全家开开胃啊！</a>
