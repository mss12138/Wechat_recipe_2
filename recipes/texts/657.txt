recipe_name: 蒸牛肉豆腐金针菇
practice: 蒸
flavor: 家常味
time: 15分钟
grade: 初级入门
materials: 主料牛瘦肉400克豆腐1盒金针菇1把辅料红辣椒1段香葱1棵玉米淀粉20克料酒15克盐1克生抽少许热油少许
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972674704.jpg>第1步: 材料准备好：牛瘦肉，韧豆腐、金针菇、香葱、红辣椒；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972648659.jpg>第2步: 牛肉提前放冰箱冷冻，取出后稍化冻，用利切切薄薄的片；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972691201.jpg>第3步: 加适量生抽、料酒搅拌均匀，直到肉片全部融化变软；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972616245.jpg>第4步: 金针菇切掉老根，分成小撮；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972676698.jpg>第5步: 找个大深盘，将金针菇均匀铺底；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972661662.jpg>第6步: 再摆上豆腐块；豆腐宜选用质地稍硬一些的，不要用内酯豆腐；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972786487.jpg>第7步: 腌制入味的牛肉片里撒2小勺玉米淀粉；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972772313.jpg>第8步: 拌匀；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972815859.jpg>第9步: 将牛肉片展开，铺在豆腐上；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972888302.jpg>第10步: 放在凉水蒸锅里；盖上盖子，大火上汽后蒸15-20分钟；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972878598.jpg>第11步: 香葱切末，取一小段辣椒切碎丁；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162478972862790.jpg>第12步: 蒸好的牛肉豆腐金针菇从锅中取出，撒上香葱末和红辣椒碎，淋上热油即可。</a>
