recipe_name: 辣子鸡
practice: 炒
flavor: 香辣味
time: 15分钟
grade: 中级掌勺
materials: 主料鸡腿三个（约150克）葱姜适量干辣椒一把鸡蛋一个辣椒粉一勺花生米一把白芝麻一把辅料料酒两勺酱油3勺蚝油一勺盐适量
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700442566.jpg>第1步: 食材都准备好。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700431230.jpg>第2步: 鸡腿清洗干净，剁成块，大概一个能剁8块左右，太小了吃着肉少不过瘾。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700563734.jpg>第3步: 除干辣椒、一半花椒 、花生米、芝麻以外，所有的调料放入鸡腿搅拌，辣椒面是灵魂所在千万不能忘，爱吃辣的可以再多加一勺。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700524152.jpg>第4步: 充分搅拌好后腌制（用筷子粘点汁稍微试试咸淡，酌情增减和酱油）。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700553886.jpg>第5步: 下油锅炸之前把所有的能看的见的调料都挑出来， 挑完再加一小把玉米淀粉搅拌均匀就可以了。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700514105.jpg>第6步: 油烧至七八成热，将腌制好的鸡腿轻轻倒进去，小火炸熟，捞起再大火复炸一遍，</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700582466.jpg>第7步: 表面上色了，颜色变得焦黄捞出来，控油备用。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700548792.jpg>第8步: 接着来处理其余食材，花生米倒入油锅中炒至焦黄，我是用空气炸锅炸的，220度炸6分钟，这颜色刚刚好。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700493326.jpg>第9步: 干辣椒切段、洋葱切调备用。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700545909.jpg>第10步: 锅里留少许油，下葱段和干辣椒中小火煸炒，炒到辣椒微微变色，辣味飘出。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700536942.jpg>第11步: 将鸡腿倒进去一起混合翻炒。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163514700582810.jpg>第12步: 出锅前将炸好的花生米倒进去混合均匀，然后撒入熟芝麻即可，妥妥的米饭杀手呀！不信大家不爱！</a>
