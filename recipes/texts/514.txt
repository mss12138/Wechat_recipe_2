recipe_name: 奥尔良鸡翅根
practice: 煎
flavor: 其它口味
time: 15分钟
grade: 新手尝试
materials: 主料鸡翅根300克辅料奥尔良烤肉料18克亚麻公社胡麻油10克水30克
step: <a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162755997281747.jpg>第1步: 鸡翅根用清水多浸泡冲洗几遍，把血水泡出来，</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162755998212554.jpg>第2步: 捞出控水后，加入奥尔良烤肉了，用手抓揉一会儿，让其充分入味</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162755998269578.jpg>第3步: 冰箱冷藏腌半天</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162755998238891.jpg>第4步: 锅里倒入点胡麻油，无需多，大约10毫升即可</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162755998281967.jpg>第5步: 放入腌好的鸡翅，不要翻动，盖上中火盖子煎</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162755999192212.jpg>第6步: 煎到底部有金黄的焦壳，翻面继续煎，两个面大约煎了5分钟</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162755999227251.jpg>第7步: 加入约30毫升水，继续盖盖转中小火煎，一直煎到水干，这个过程大约也有5分钟。加水煎是利用蒸汽，让鸡翅根完全成熟</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162755999120315.jpg>第8步: 打开盖子，翻个面继续煎，一直煎到底部的汤汁都在锅底结成焦壳，就可以出锅啦。</a>
<a href=https://st-cn.meishij.net/rs/114/34/1258614/n1258614_162756000595180.jpg>第9步: 有烧烤风味的鸡翅根就做好啦，虽然简单，但很好吃的哦。“亚麻公社胡麻油，植物油中的脑白金”</a>
