recipe_name: 肉末烤茄子
practice: 烤
flavor: 香辣味
time: 30分钟
grade: 中级掌勺
materials: 主料长茄子2个蒜末适量肉末适量辅料食盐适量生抽适量蚝油适量料酒适量老抽适量小葱适量食用油适量胡椒粉适量辣椒粉适量
step: <a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162108778832546.jpg>第1步: 茄子洗净、去蒂</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162108778998107.jpg>第2步: 在茄子表面刷上一层食用油，然后放入预热好的烤箱，200度烤20分钟</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162108779111874.jpg>第3步: 锅中热油，然后加入蒜末，翻炒出香味</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162108779344263.jpg>第4步: 将肉末倒入锅中，炒至泛白</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162108779515173.jpg>第5步: 加入料酒、生抽、老抽、蚝油、食盐、胡椒粉，翻炒均匀，然后倒入清水，煮熟出锅</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162108779614498.jpg>第6步: 将烤好的茄子从中间切开，中间铺上一层蒜末打底</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162108779808973.jpg>第7步: 茄子中间继续加入肉末，撒上一层辣椒粉</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162108780083885.jpg>第8步: 放入预热好的烤箱，170度烤8分钟</a>
<a href=https://st-cn.meishij.net/rs/57/65/14891307/n14891307_162108780201001.jpg>第9步: 将茄子从烤箱中取出，撒上小葱、白芝麻，开吃</a>
