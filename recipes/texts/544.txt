recipe_name: 黄瓜炒虾仁&炸虾头
practice: 炒
flavor: 咸鲜味
time: 15分钟
grade: 初级入门
materials: 主料大虾1盘黄瓜1根红甜椒半个辅料小葱1棵植物油适量玉米淀粉5克黑胡椒粉2克盐适量中筋面粉15克生抽少许
step: <a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662642295.jpg>第1步: 大虾冲洗干净；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662666178.jpg>第2步: 虾仁脱壳，虾头中的内脏挤出，清洗干净；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662668426.jpg>第3步: 虾仁腹部和背部用刀尖划开，挑去虾线，用凉水淘洗2遍；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662632009.jpg>第4步: 虾头中放2小勺中筋面粉、2克黑胡椒粉、少许盐，拌匀；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662653746.jpg>第5步: 虾仁中放1小勺玉米淀粉，抓匀上浆；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662647206.jpg>第6步: 红甜椒、黄瓜、小葱清洗干净；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662747354.jpg>第7步: 红甜椒去蒂去籽切小块；黄瓜纵剖几条，去掉黄瓜芯，切小块；小葱切末；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662754986.jpg>第8步: 炒锅中多倒点儿油，中火加热，手离油约10公分高时能感到很强的热气时，放一个虾头入锅，边缘能立马儿起小泡说明油温够了；将其它的虾头依次入锅，待定型后用铲子轻轻拨动；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662739611.jpg>第9步: 火力由中火转中小火，待全部虾头都浅焦黄时，盛出，稍晾凉可食用；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662792119.jpg>第10步: 锅中留底油，将黄瓜和红甜椒入锅翻炒一分钟，变色盛出；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662792156.jpg>第11步: 铲子是镂空的，所以多余的油留在锅里了，无需额外倒油，将虾仁和葱花倒锅中轻轻翻炒；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662783126.jpg>第12步: 待虾仁微微卷曲，沿锅淋少许料酒去腥；</a>
<a href=https://st-cn.meishij.net/rs/207/244/2811207/n2811207_162720662865926.jpg>第13步: 倒入黄瓜和红甜椒，撒适量盐、生抽，翻炒均匀出锅。</a>
