recipe_name: 栗子烧鸡腿
practice: 烧
flavor: 咸鲜味
time: 30分钟
grade: 初中水平
materials: 主料鸡琵琶腿2只板栗仁200克辅料葱姜适量植物油3大勺六月鲜 红烧酱油1大勺六月鲜 特级原汁酱油1大勺味达美 臻品料酒2大勺盐少许白砂糖1大勺
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454519067890.jpg>第1步: 鸡腿是冷冻鸡腿，摊主帮忙斩成大块了。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454519170171.jpg>第2步: 栗子剥壳，剥出的栗子仁约200克，多点少点都无所谓。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454519169660.jpg>第3步: 准备好调味料，原汁酱油没有的话可以用生抽或蚝油，用量也是1大勺，或者按自己的口味调整用量。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454522639223.jpg>第4步: 鸡腿块用水泡半小时，目的是泡出血水。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454519150790.jpg>第5步: 鸡块沥水，加1大勺料酒，少许盐和1大勺白砂糖，抓匀盖保鲜膜腌制半小时。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454519175817.jpg>第6步: 锅中加3勺油，烧热下葱姜炒香，倒入腌制过的鸡块，炒到变色。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454519240243.jpg>第7步: 调味：下1大勺原汁酱油，1大勺红烧酱油，1大勺料酒。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454519334152.jpg>第8步: 倒入栗子仁。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454519317573.jpg>第9步: 加水没过食材，加盖大火煮开。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454519363157.jpg>第10步: 撇去浮沫后，加盖转小火焖煮20~30分钟</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454654011735.jpg>第11步: 再转大火收浓汤汁，装盘撒上葱花点缀</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_163454519704072.jpg>第12步: 咸鲜带甜的鸡肉、香甜软糯的栗子仁，令人回味无穷。</a>
