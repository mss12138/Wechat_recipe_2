recipe_name: 酥炸鸡翅根
practice: 炸
flavor: 酱香味
time: 15分钟
grade: 初中水平
materials: 主料鸡翅根8个辅料中筋面粉210克玉米淀粉70克小葱2棵姜片3片味达美臻品料酒1大勺葱伴侣6月香豆瓣酱1大勺葱伴侣6月香辣椒酱1大勺白砂糖1大勺
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769438528.jpg>第1步: 准备好食材和调味料。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769596535.jpg>第2步: 翅根洗净，用牙签多扎点小孔，方便腌制入味</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769456134.jpg>第3步: 挤干翅根中的水分，倒入一大勺料酒，一大勺白砂糖，还有葱姜。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769535822.jpg>第4步: 加一大勺葱伴侣6月香豆瓣酱</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769582166.jpg>第5步: 加一大勺葱伴侣6月香辣椒酱</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769599609.jpg>第6步: 上手抓匀，使每一根鸡翅根都裹上酱料，盖保鲜膜冷藏腌制1小时以上</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769515799.jpg>第7步: 配制酥炸粉：将玉米淀粉和中筋面粉混合均匀，可按口味加其他调料粉，如胡椒粉等。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769678469.jpg>第8步: 腌好的翅根先裹一层酥炸粉，</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769666254.jpg>第9步: 再在水碗中滚一下</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769826597.jpg>第10步: 再裹一层粉，继续泡水，最后再裹粉，抖掉多余的炸粉，所有的翅根都按此流程裹好炸粉。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769895742.jpg>第11步: 倒大半锅油，烧到摄氏170℃度，投入翅根，不时翻动以使受热均匀，约炸3~4分钟，呈现金黄色捞出沥油，全部初炸完成。油温升到200℃左右，将炸好的翅根投入复炸30秒左右捞出。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769858774.jpg>第12步: 外壳酥脆，鸡肉细嫩，配上甜辣酱或番茄酱，好吃又解腻。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162744769906198.jpg>第13步: 来个近景！</a>
