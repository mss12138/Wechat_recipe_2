recipe_name: 藤椒肉片
practice: 煮
flavor: 麻辣味
time: 15分钟
grade: 中级掌勺
materials: 主料莴笋1根黄豆芽200克猪瘦肉300克辅料藤椒油适量食用油 1勺香菜1根生姜1块香葱2根蒜头4瓣生粉2小勺料酒2小勺蛋清3小勺食盐半小勺白糖半小勺松茸鲜1袋青花椒1勺白胡椒粉半小勺川味麻辣汁适量
step: <a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920901762553.jpg>第1步: 准备好食材，配菜我准备的是莴笋和黄豆芽</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920902555353.jpg>第2步: 黄豆芽掐去根部洗净，莴笋去皮洗净切滚刀块，莴笋叶切段；生姜切丝、葱蒜切末、香菜切段；另外准备少许青花椒，备用</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920901971562.jpg>第3步: 猪肉清洗干净，切成薄片；新手下厨，若担心切不好，可以将猪肉在冰箱微微冻硬一会儿，切起来就顺手多了</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920901908859.jpg>第4步: 切好后再用水冲洗几遍且挤干水分，然后加入料酒、葱姜、食盐、白糖、生粉、蛋清和白胡椒粉，轻轻抓拌均匀</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920902030909.jpg>第5步: 再倒入少许藤椒油锁住水分，腌制10分钟，这样煮出来的肉片吃着清香鲜嫩还不柴没腥味</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920902179815.jpg>第6步: 炒锅加热，倒入食用油，下入葱姜蒜爆香</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920902436181.jpg>第7步: 下入黄豆芽、莴笋块及莴笋叶，大火翻炒</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920902570852.jpg>第8步: 炒至断生就可以了，不宜炒太久，炒久口感就不脆了，盛出来放入碗里</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920902541332.jpg>第9步: 不用洗锅，直接加适量清水，水量能莫过全部食材即可；再倒入川香麻辣汁，麻辣汁是复合型调料，既有麻辣红油也有咸味</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920902551765.jpg>第10步: 加入一袋松茸鲜调味，这个既有松茸菌菇类的鲜味，也是有咸味的</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920902890094.jpg>第11步: 汤汁煮开后转小中火，把腌好的肉片一片一片地汆入锅中，全部入锅后，煮熟再下腌好的肉片，就用筷子划拨几下，让肉片不会粘连在一起</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920902930715.jpg>第12步: 然后转大火煮至沸腾即可，肉片薄容易熟，时间不宜过长，煮久就老了；再滴入少许藤椒油，增加椒香味道</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920903070143.jpg>第13步: 出锅前尝下味道，若觉得味淡可以补充少许的盐，然后连肉带汤一起倒入垫有蔬菜的碗中，上面撒上蒜泥、葱花和一把青花椒</a>
<a href=https://st-cn.meishij.net/rs/67/36/4071567/n4071567_163920902985236.jpg>第14步: 锅中重新倒入藤椒油，烧至油略微冒烟关火，将热油泼于蒜末和青花麻上，热油一泼，香味顿时扑鼻，最后再撒上葱花和香菜即可</a>
