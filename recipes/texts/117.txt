recipe_name: 家常糖醋里脊
practice: 炒
flavor: 酸甜味
time: 10分钟
grade: 新手尝试
materials: 主料猪里脊500克蛋清1个白胡椒粉1小勺料酒1勺盐1小勺辅料普通面粉1大勺玉米淀粉1大勺泡打粉2克蛋黄1个食用油1大勺白砂糖3大勺白醋1大勺番茄酱4大勺清水2大勺
step: <a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013671921423.jpg>第1步: 猪里脊500克，洗干净后沥干水分，切成细长条备用。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672092270.jpg>第2步: 加入盐1小勺、蛋清1个、白胡椒粉1小勺、料酒1勺。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672014144.jpg>第3步: 用手抓拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672011031.jpg>第4步: 包上保鲜膜，冷藏腌制1小时入味。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672044372.jpg>第5步: 趁着腌制的时候，我们做一点糖醋原料汁，把白砂糖3大勺、白醋1大勺、番茄酱4大勺、清水2大勺。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672000511.jpg>第6步: 放一起搅拌均匀备用。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672079052.jpg>第7步: 取出腌制好的里脊，普通面粉1大勺、玉米淀粉1大勺、泡打粉2克、蛋黄1个，先拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672084137.jpg>第8步: 再加入食用油1大勺。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672044869.jpg>第9步: 用手抓拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672096026.jpg>第10步: 油温加热至六成热，筷子放进去冒小泡泡。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672094233.jpg>第11步: 即可加入腌制好的里脊开始炸。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672052889.jpg>第12步: 先炸1分钟别动它，等定型了再拨散一下，炸至里脊的外壳金黄，里脊可以浮起来以后，捞出来备用。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672008477.jpg>第13步: 再把油加热至8成热，放入炸好的里脊复炸30秒。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672196694.jpg>第14步: 炸至外壳金黄焦脆，捞出来备用。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672169296.jpg>第15步: 把油倒掉，不需要洗锅，把准备好的糖醋原料汁倒入，稍微烧粘稠一点。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672157661.jpg>第16步: 倒入炸好的里脊，翻炒均匀即可。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672144594.jpg>第17步: 糖醋猪里脊, 完成O(∩_∩)O</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672197529.jpg>第18步: 酸甜可口，色泽金黄诱人，孩子最爱。</a>
<a href=https://st-cn.meishij.net/rs/57/98/14337057/n14337057_164013672107611.jpg>第19步: 方法和配方都很简单，学会了可以天天在家吃。</a>
