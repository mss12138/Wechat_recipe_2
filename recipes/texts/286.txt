recipe_name: 酸辣土豆丝
practice: 炒
flavor: 咸鲜味
time: 5分钟
grade: 新手尝试
materials: 主料土豆1个胡萝卜半根辅料洋葱少许醋1勺酱油1勺食盐2克格琳诺尔有机亚麻籽油适量干辣椒1个
step: <a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163266299178271.jpg>第1步: 先把需要的食材提前准备好。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163266299473181.jpg>第2步: 土豆、胡萝卜去皮洗净擦丝备用。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163266298645411.jpg>第3步: 洋葱切丝。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163266298955707.jpg>第4步: 提前烧一锅开水，水开后倒入土豆、胡萝卜丝焯水，开水中烫一下即可，时间不宜过长。
土豆、胡萝卜丝焯好水后捞出沥干水分。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163266299025440.jpg>第5步: 锅中倒入适量格琳诺尔有机亚麻籽食用油，放入干辣椒、洋葱炒香。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163266299296506.jpg>第6步: 倒入焯好水的土豆、胡萝卜丝大火快速翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163266299413032.jpg>第7步: 加1勺陈醋或白醋、1勺酱油、2克食盐调味调色，快速翻炒均匀即可。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163266302014667.jpg>第8步: 格琳诺尔有机亚麻籽油，宝宝的聪明油</a>
