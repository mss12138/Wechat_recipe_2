recipe_name: 可以吃的火山，你要不尝一尝？
practice: 拌
flavor: 茄汁味
time: 15分钟
grade: 新手尝试
materials: 主料土豆400克胡萝卜150克西红柿300克冷冻蔬菜20克辅料六月鲜10克轻盐特级原汁酱油2勺橄榄油20毫升
step: <a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748099444.jpg>第1步: 准备食材：土豆去皮切块放入蒸锅蒸煮15分钟，胡萝卜去皮切段蒸熟</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748342859.jpg>第2步: 将蒸熟的土豆倒入碗中</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748678862.jpg>第3步: 将土豆碾成泥</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748601736.jpg>第4步: 将蒸熟的胡萝卜碾成泥</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748512891.jpg>第5步: 将胡萝卜泥和土豆泥混合</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748408409.jpg>第6步: 将混合好后的土豆萝卜泥倒入一个碗中压实</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748626813.jpg>第7步: 将碗倒扣在盘中，然后取出碗，将胡萝卜土豆泥用叉子做出火山的造型</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748706523.jpg>第8步: 锅内倒入20ml橄榄油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748737854.jpg>第9步: 将番茄去皮切块倒入锅内进行煸炒</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748706786.jpg>第10步: 煸炒出沙</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748693885.jpg>第11步: 倒入20g冷冻蔬菜，翻拌均匀</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748729757.jpg>第12步: 倒入2勺酱油（六月鲜10克轻盐特级原汁酱油）</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748795643.jpg>第13步: 搅拌均匀炖煮2分钟即可</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_163013748720235.jpg>第14步: 将炖煮好的茄汁淋在土豆泥上即可</a>
