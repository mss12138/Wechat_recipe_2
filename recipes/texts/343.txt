recipe_name: 白萝卜炒鸡蛋
practice: 炒
flavor: 咸鲜味
time: 10分钟
grade: 未知
materials: 主料白萝卜250克鸡蛋2个辅料小葱少许食盐2克白胡椒粉1克格琳诺尔有机亚麻籽油适量
step: <a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163171314223893.jpg>第1步: 先把需要的食材提前准备好。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163171314356040.jpg>第2步: 白萝卜去皮。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163171314486969.jpg>第3步: 用清水冲洗干净擦成细丝备用。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163171314598957.jpg>第4步: 碗里打入2个鸡蛋，用筷子搅打均匀。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163171314755832.jpg>第5步: 准备少许小葱切碎。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163171314808643.jpg>第6步: 锅中倒入少许格琳诺尔有机亚麻籽食用油。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163171314950331.jpg>第7步: 倒入搅打好的鸡蛋液炒成块状盛出备用。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163171315051086.jpg>第8步: 锅中重新倒入少许底油，放入洋葱炒出香味。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163171315197987.jpg>第9步: 倒入白萝卜丝大火快速翻炒均匀，炒至白萝卜丝稍微变软即可。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163171315257843.jpg>第10步: 放入1克左右的白胡椒粉、加入2克食盐、倒入提前准备好的小葱配色、最后倒入炒好的鸡蛋翻炒均匀即可。</a>
<a href=https://st-cn.meishij.net/rs/143/203/14613393/n14613393_163171315313372.jpg>第11步: 格琳诺尔有机亚麻籽油，宝宝的聪明油。</a>
