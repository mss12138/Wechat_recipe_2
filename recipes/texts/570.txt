recipe_name: 吃一回爱一回，不愧是虾中最香的做法——
practice: 炒
flavor: 香辣味
time: 15分钟
grade: 初级入门
materials: 主料大虾300克藕150克土豆150克芹菜100克红椒1个辅料花椒10几粒蒜瓣2个辣椒酱适量盐5克白糖20克生抽30毫升孜然粉少许金龙鱼特香花生油适量
step: <a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633892794382.gif>第1步: 藕和土豆去皮切片。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633893688679.gif>第2步: 芹菜切段，青红椒切圈。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633894331366.gif>第3步: 大虾洗净开背。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633895365009.gif>第4步: 锅中倒入适量油烧至七成热。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633896365602.gif>第5步: 倒入土豆和藕片炸60秒左右，捞出控油。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633897104655.gif>第6步: 锅中油继续烧至六成热，放入大虾炸至表面金黄后捞出。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633898042811.gif>第7步: 锅中加入适量油，烧至六成热。放入花椒、蒜瓣、红辣椒和辣椒酱煸炒出香味。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633899139767.gif>第8步: 倒入芹菜翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633899851131.gif>第9步: 倒入大虾、藕片和土豆。调入5克盐、20克白糖和30毫升生抽。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633900707299.gif>第10步: 翻炒均匀。出锅前，可撒上少许孜然粉提味。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633901531092.gif>第11步: 盛出装盘，满屋子都溢满了香味。</a>
<a href=https://st-cn.meishij.net/rs/10/120/30010/n30010_162633902305720.jpg>第12步: 蔬菜清脆爽口，虾肉外酥里嫩，一口一个，超级过瘾，夏日里跟朋友聚餐，只要这一锅就够啦！</a>
