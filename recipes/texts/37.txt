recipe_name: 香菇西蓝花炒虾仁
practice: 炒
flavor: 家常味
time: 10分钟
grade: 新手尝试
materials: 主料西蓝花200克虾仁150克鲜香菇3朵木耳少许辅料味达美臻品料酒1大勺六月鲜12克轻盐牡蛎酱油1勺淀粉1勺
step: <a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_164605290334089.jpg>第1步: 准备所需食材</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_164605290387549.jpg>第2步: 黑木耳用开水泡发洗净浮土泥沙</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_164605290342417.jpg>第3步: 西兰花去根掰成小朵</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_164605290491719.jpg>第4步: 鲜香菇洗净切成小块</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_164605290314740.jpg>第5步: 虾仁用盐和淀粉抓一抓，用水冲洗几遍捞出控水（盐和淀粉可以去掉杂质，可以使虾仁变得通透和脆嫩）</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_164605290303203.jpg>第6步: 虾仁加葱丝、白胡椒粉、味达美臻品料酒去腥，加淀粉抓匀腌制10分钟</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_164605290339121.jpg>第7步: 水烧开倒入黑木耳、西蓝花、香菇块焯水捞出</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_164605290401812.jpg>第8步: 炒锅加油烧热炒香葱碎，倒入虾仁炒至变色发红</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_164605290498196.jpg>第9步: 倒入所有蔬菜炒匀，加六月鲜12克轻盐牡蛎酱油调味，用水淀粉勾芡即可</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_164605352923492.jpg>第10步: 表面撒枸杞即可上桌</a>
