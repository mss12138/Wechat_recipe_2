recipe_name: 蒜苗炒蛋，先炒蛋还是先炒蒜苗怎么炒不
practice: 炒
flavor: 咸鲜味
time: 10分钟
grade: 未知
materials: 主料蒜苗1把鸡蛋4个辅料盐适量水淀粉适量糖适量十三香适量
step: <a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162410736671940.jpg>第1步: 将蒜苗洗净切段儿，蒜苗叶和蒜苗根，分开装盘。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162410736793597.jpg>第2步: 将四个鸡蛋打入碗中，加适量淀粉和盐，淀粉，会让鸡蛋变得更嫩。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162410736951038.jpg>第3步: 将鸡蛋打散，搅匀备用。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162410737009757.jpg>第4步: 能过热，油油热后倒入蛋液，不要一顿乱炒。推动鸡蛋，让蛋液慢慢全部凝固。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162410737135163.jpg>第5步: 蛋液全部凝固后在炒散</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162410737204493.jpg>第6步: 倒入蒜苗跟一起翻炒几下。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162410737339382.jpg>第7步: 再倒入蒜苗叶翻炒几下，待蒜苗叶炒至断生后，加盐，糖，十三香。</a>
<a href=https://st-cn.meishij.net/rs/217/185/2921467/n2921467_162410737461598.jpg>第8步: 最后再翻炒几下就可以出锅啦。</a>
