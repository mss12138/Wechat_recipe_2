recipe_name: 彩蔬虾球【宝宝辅食】
practice: 煮
flavor: 家常味
time: 15分钟
grade: 新手尝试
materials: 主料虾仁180克淀粉15克胡萝卜10克菠菜15克即食玉米粒15克辅料鲜菇粉适量柠檬适量
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164609500523211.gif>第1步: 现成的虾仁或者是现剥的都行，去掉虾线，剁成虾泥。

*我这里比较建议你们直接手剁，还保留了一些虾肉的颗粒感。如果是直接用搅拌机打，虽然省劲儿了，但是搅打得太细碎，失去了那种虾肉的感觉。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164609503226126.gif>第2步: 在虾泥里挤上适量的柠檬汁，腌制5分钟，这样也是为了能更好的去掉腥味。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164609507648686.gif>第3步: 胡萝卜、菠菜焯水捞出。胡萝卜你们可以煮得稍微久一点，让它熟透了。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164609508397214.jpg>第4步: 胡萝卜、菠菜和玉米粒切碎，备用。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164609511110578.gif>第5步: 在蔬菜碎里分别加上1/3的虾泥，我这是做了3个口味，你们也能用其他蔬菜来做，西蓝花、萝卜、芹菜啥的，只要孩子不排斥的就行。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164609512344077.gif>第6步: 小碗里分别加上1/3的淀粉和适量的鲜菇粉提味，如果是大宝宝的，对口味要求比较多的，可以少加点盐调一下。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164609628951139.gif>第7步: 把各种食材都充分搅拌均匀了，搅的时候注意一下，如果太稀了，感觉不太能成团的，可以稍微补一点淀粉。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164609632237948.gif>第8步: 在手上刷油，这样在做虾球的时候能更加方便。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164609632691193.gif>第9步: 取适量虾泥，揉搓成小球的样子。这个虾球的大小，你们就根据宝宝的咀嚼能力来就行。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_164609632837107.gif>第10步: 做好的小虾球放入沸水锅中，一直煮到虾球浮上来、熟透了，就能捞出来了。</a>
