recipe_name: 毛豆炒肉丁
practice: 炒
flavor: 酱香味
time: 15分钟
grade: 初中水平
materials: 主料毛豆300克猪肉200克辅料植物油适量盐适量姜5克料酒10克辣椒酱一大勺花椒粉1克
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163015210859392.jpg>第1步: 主料：毛豆200克、猪肉200克
辅料：植物油适量、盐适量、生抽5克、辣椒酱一大勺、姜5克、鸡精1克、花椒粉1克</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163015210814834.jpg>第2步: 把毛豆洗洗去壳，再次洗洗去皮，姜切末</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163015210851109.jpg>第3步: 把猪肉切小丁，最好加点猪肥肉炒出来比较香浓</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163015210845562.jpg>第4步: 热锅凉油，油烧到8成热，下猪肉丁和肥肉，炒到肉丁变色出香味，加料酒和姜末，把猪肉丁拨在一边</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163015210842667.jpg>第5步: 下毛豆翻炒均匀，加少许的盐多炒几分钟，最好把毛豆炒到外皮微微发皱，这样毛豆香味都出来，也快熟了</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163015210849718.jpg>第6步: 加生抽，把毛豆和猪肉丁一起翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163015210808110.jpg>第7步: 加一大勺辣椒酱大火快速翻炒出红油出酱香味，一定要炒出辣椒酱的香辣味</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163015210943341.jpg>第8步: 加入清水，焖烧一会，大约3分钟左右</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163015210963681.jpg>第9步: 加花椒粉炒出香味，大火翻炒收汁，这样好看也好吃，看着油光闪亮的</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_163015210988454.jpg>第10步: 加鸡精快速翻炒均匀即可出锅。</a>
