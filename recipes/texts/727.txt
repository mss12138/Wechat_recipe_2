recipe_name: 马兰干烧肉
practice: 烧
flavor: 咸鲜味
time: 2小时
grade: 中级掌勺
materials: 主料五花肉约1000克马兰干约100克辅料大葱1棵姜1大块八角2个桂皮1小块料酒1大勺黄酒约100毫升冰糖约60克生抽2大勺蚝油1大勺老抽1大勺盐少许
step: <a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337155277854.jpg>第1步: 马兰干是用新鲜马兰头焯水晒干后的野菜干，有浓郁的清香。使用前要先泡水，我泡了半天。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337153294429.jpg>第2步: 五花肉我买的是带骨的，请卖肉师傅把骨头剁断，皮肉不要剁开。铁锅烧到冒烟，五花肉肉皮朝下在锅底摩擦，这一步的目的是破坏肉皮汗腺，减少腥臊味道。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337153255863.jpg>第3步: 肉皮表面焦黑即可，也可以用火烧肉皮达到此效果。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337153205743.jpg>第4步: 用刀刮除肉皮上的焦斑，清洗干净。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337153202122.jpg>第5步: 五花肉冷水下锅，取一段大葱白切段，3~4片姜片，1勺料酒，加盖煮开后再煮约30秒左右。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337153319046.jpg>第6步: 冷水洗去血沫，焯肉的水丢弃不用。也有人焯水后撇去浮沫后留用汤水，但我老觉得撇不干净。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337153391607.jpg>第7步: 再次冷水入锅，水要淹没肉块。保留葱姜，另加八角桂皮，大火煮开转中小火煮40分钟。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337153396122.jpg>第8步: 捞出五花肉浸入冷水中，凉后切成3~4CM见方的肉块。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337153493509.jpg>第9步: 剩下的大葱切长段，垫在砂锅底部，姜切片铺在大葱上。为了不让马兰干与葱姜混在一起，我在葱姜上盖了一块硅胶笼布，但它太轻了，后面汤水煮滚后就把笼布掀了起来，所以你就不用学我加笼布了……</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337153420235.jpg>第10步: 把泡软的马兰干铺在葱姜上，再把肉铺在马兰干上，最好是肉皮朝下，我这些肉切得不规整，立不起来。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337163404662.jpg>第11步: 炒个焦糖浆，冰糖和小许水一起放锅中炒化，糖化后就不要搅拌了，别和我一样炒得糖都返砂了……</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337164303649.jpg>第12步: 当糖液变成棕色时加入开水，量多少没什么讲究，再炒均匀即可。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337166220780.jpg>第13步: 把糖浆浇淋到肉块上，最好每一块肉都浇到。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337166969117.jpg>第14步: 煮肉的汤也倒进砂锅。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337167556461.jpg>第15步: 加少许盐，2大勺生抽，1大勺蚝油，1大勺老抽，加盖。先小火煮5分钟让砂锅慢慢热起来，转大火煮沸后再转小火慢炖2小时左右。</a>
<a href=https://st-cn.meishij.net/rs/241/31/13695491/n13695491_162337168199537.jpg>第16步: 马兰干赋予红烧肉特殊的清香，而肉汁也让马兰干更丰腴。肉色红亮，肥而不腻，软而不烂，齿颊留香。</a>
