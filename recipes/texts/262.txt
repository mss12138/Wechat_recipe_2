recipe_name: 芋头丸子肉片汤
practice: 煮
flavor: 家常味
time: 60分钟
grade: 未知
materials: 主料芋头400克红薯淀粉适量猪肉200克辅料小米辣2个生姜10克香葱1颗香菜1颗
step: <a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300743847755.jpg>第1步: 首先将芋头表面泥沙搓洗干净，并放上蒸锅，隔水蒸至用筷子能轻易戳透即可；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300743917507.jpg>第2步: 蒸芋头期间，将里脊肉切成薄片，再将生姜、小米辣、香葱、花菜切碎，姜末、辣椒碎、葱白碎放置一起；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300744065639.jpg>第3步: 里脊肉里调入1勺料酒、1勺生抽、小半勺盐和小半勺胡椒粉，抓拌均匀后腌制10分钟入味，再倒入1勺红薯淀粉，并少量多次倒入清水，直至抓拌成较为粘稠的肉片，放置一旁腌制备用；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300744142030.jpg>第4步: 芋头蒸好后，剥去外皮，调入小半勺盐和1勺糖增加美味，再搅拌成粘稠细腻的芋泥，再少量多次倒入红薯淀粉，直至揉成不黏手的柔软面团；

注意：除了调味，还可以加入葱花、肉末等食材，增加丸子的口感和口味；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300744185782.jpg>第5步: 将面团分成若干个差不多大小的小剂子，分别揉圆；

注意：形状可根据大家喜好；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300744203687.jpg>第6步: 将揉好的丸子热水下锅，煮至浮起后，再倒入2次清水，直至丸子煮透，然后捞出过凉水；

注意：煮好的丸子一顿吃不完，冷却后可以密封冷冻冷藏保存；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300744214397.jpg>第7步: 起锅，锅中倒入少许食用油，油热后倒入姜末、辣椒末和葱白，翻炒出香辣味，虽然倒入1碗清水和1勺生抽，大火煮沸；

注意：不能吃辣则可以不放辣椒；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300744385018.jpg>第8步: 再倒入芋泥丸子，再次煮沸后转中小火盖上锅盖焖煮入味，揭开锅盖汤汁再次沸腾后，将肉片一片片下入锅中；</a>
<a href=https://st-cn.meishij.net/rs/120/39/13134870/n13134870_163300744427249.jpg>第9步: 待肉片煮熟后，调入适量盐和鸡粉，撒入葱花和香菜段，搅拌均匀即可食用；</a>
