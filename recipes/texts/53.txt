recipe_name: 高纤又补脑的凉拌轻食——西芹拌桃仁
practice: 拌
flavor: 酸甜味
time: 15分钟
grade: 新手尝试
materials: 主料西芹300克核桃仁50克辅料六月鲜10克轻盐特级原汁酱油15毫升有机亚麻籽油15毫升味达美清香米醋30毫升
step: <a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164508564141599.jpg>第1步: 准备食材：西芹去除老茎洗净切小段，核桃仁50克</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164508556383948.jpg>第2步: 锅内放入适量的清水（差不多600毫升）</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164508556416245.jpg>第3步: 将锅内的清水煮沸</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164508556417036.jpg>第4步: 将西芹洗净切成小段，放入锅内进行汆烫，汆烫可以去除西芹中的草酸</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164508556490448.jpg>第5步: 将西芹汆烫2分钟之后捞出</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164508556416780.jpg>第6步: 将汆烫好的西芹捞出装入盘中，倒入15毫升亚麻籽油，搅拌均匀可以防止西芹变色</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164508560220735.jpg>第7步: 倒入2大勺（30毫升）味达美清香米醋</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164508560239996.jpg>第8步: 倒入1大勺（15毫升）六月鲜10克轻盐特级原汁酱油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164508556403028.jpg>第9步: 搅拌均匀，让西芹充分吸收酱汁</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164508559032345.jpg>第10步: 放上50克熟核桃仁，翻拌均匀即可</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_164508585540478.jpg>第11步: 酸甜的口感，高纤又补脑，轻食凉菜必备</a>
