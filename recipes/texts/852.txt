step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166798724076.jpg>第1步: · 参考月龄 ·
15个月以上，对食材不过敏的宝宝
· 食材准备 ·
牛尾骨200克 / 胡萝卜50克
圆葱40克 / 西红柿1个 / 土豆1个
辅料：玉米油 / 鲜菇粉
盐 / 葱段 / 姜片 / 番茄酱</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166798855323.gif>第2步: 西红柿事先去皮。我这回用的是去皮器，家里没有这个的姐妹，可以先在西红柿顶端划个十字口，热水烫1分钟，剥皮就格外简单了。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166798741674.jpg>第3步: 去好皮的西红柿和圆葱切块，土豆和胡萝卜也都切成块状。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166798863787.gif>第4步: 牛尾骨放入锅中，加上姜片、葱段去腥提味</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166798893438.gif>第5步: 倒入清水，中火慢慢熬煮。葱花姜片在一起也有去腥的作用，这样就不用放其他的味重的调料，更适合宝宝的口味。如果家里没有牛尾骨用其他的像是排骨、脊骨代替也都是可以的。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166798832362.gif>第6步: 煮牛尾骨的过程当中，要注意撇去表面的浮沫。基本无浮沫析出，捞出牛尾骨。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166798981778.gif>第7步: 锅中刷油，放入西红柿，不断翻炒软化。炒到西红柿软化出汁，就可以了。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166798828699.gif>第8步: 加入1勺番茄酱，继续翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166799055145.gif>第9步: 倒入圆葱，翻炒均匀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166799064248.gif>第10步: 加入牛尾骨、土豆、胡萝卜和热水。这个时间是牛尾骨炖得正好的时候，如果宝宝月龄比较小，或者是咀嚼能力不是太好的话，可以再煮得时间长一点。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166799043748.gif>第11步: 水沸后转中小火，焖煮2个小时左右。这个时间是牛尾骨炖得正好的时候，如果宝宝月龄比较小，或者是咀嚼能力不是太好的话，可以再煮得时间长一点。

等到焖煮好之后，可以加入鲜菇粉、盐，搅拌均匀。这都是提鲜增味的调料，不过如果宝宝能够接受自然食材的味道，可以适当的减量。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162166799039521.jpg>第12步: 煮好，盛出。</a>
