recipe_name: 酸奶红薯蒸糕【宝宝辅食】
practice: 蒸
flavor: 家常味
time: 10分钟
grade: 新手尝试
materials: 主料红薯100克酸奶70克辅料鸡蛋1个
step: <a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162218796107696.jpg>第1步: · 参考月龄 ·
12个月以上，对食材不过敏的宝宝
· 食材准备 ·
红薯100克
酸奶70克 / 鸡蛋1个</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162218796232155.gif>第2步: 红薯去皮切丁，上锅蒸15分钟。我自己很喜欢红薯的味道，如果家里有其他的，像是紫薯或者是南瓜，做替换也是可以的。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162218796214690.gif>第3步: 红薯蒸熟，熟透取出，捣成红薯泥。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162218796265190.gif>第4步: 红薯泥静置晾凉，加入酸奶和鸡蛋。加了鸡蛋的蒸蛋糕，更嫩更蓬松。这个食材真的是太简单了，为了让孩子们更喜欢吃，我还加了点家里常喝的酸奶进去，给小蛋糕提升风味。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162218796211808.gif>第5步: 所有食材顺着同一方向搅拌均匀。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162218796232752.gif>第6步: 模具或者是碗中刷油，倒入已经搅拌好的红薯泥。找一个耐高温的小碗，在里面刷上一层油防黏，方便蒸熟脱模。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162218796385657.gif>第7步: 小碗用保鲜膜覆盖上，在四周扎上小孔。保鲜膜要用耐高温的那种，扎上小孔既避免了水蒸气直接接触蛋糕，影响最终造型和风味，又能够让热气更好循环，加快蒸制过程。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162218796505019.gif>第8步: 锅中加水烧沸，水沸后转成中小火，上锅蒸20分钟。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162218796669718.gif>第9步: 熟透取出，脱模。</a>
<a href=https://st-cn.meishij.net/rs/178/85/5833928/n5833928_162218796587756.jpg>第10步: 这是我特意研究的无油、无糖连面粉都不加的极简蒸蛋糕方子，从软糯度、味道和食材上也都更适合小宝宝们。这个六一，给孩子一个大大的惊喜吧！</a>
