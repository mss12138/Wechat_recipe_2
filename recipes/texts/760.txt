recipe_name: 蒜蓉豆角
practice: 汆
flavor: 家常味
time: 15分钟
grade: 中级掌勺
materials: 主料豆角一把大蒜一头剁辣椒一勺辅料葱姜蒜末适量豆豉辣椒两勺油适量盐适量
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186142137836.jpg>第1步: 锅里放入半锅水，滴入几滴油，撒入一勺眼，水烧开后将清洗干净的豆角放进去，汆烫至微微变软即可。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186142199059.jpg>第2步: 汆烫好的豆角立刻泡入清水中，迅速降温，保持豆角爽脆的口感。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186142156706.jpg>第3步: 顺着豆角的筋络，将它轻轻撕成两半。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186142142519.jpg>第4步: 留5、6根不要撕，切成10厘米左右的长度，后面用来卷。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186142177931.jpg>第5步: 取一根撕好的半边豆角，切口朝里面，卷在没撕的长豆角段上。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186142111098.jpg>第6步: 卷好后收口压在里面防止散开。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186142259023.jpg>第7步: 全部都卷好后，将豆角条摆入盘中。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186142260940.jpg>第8步: 再来一张，是不是很漂亮呀！</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186142299312.jpg>第9步: 接着咱们可以炒蒜蓉了。大蒜拍松后去皮，剁碎。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186142277755.jpg>第10步: 倒入油锅中，用小火炒出香味。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162186142288526.jpg>第11步: 然后加入一勺生抽，两勺清水河一勺豆瓣辣椒酱，煮沸后关火，将做好的蒜蓉汁淋在卷好的豆角上就可以了，吃的时候拌匀，鲜嫩又爽口的拌豆角，让人食欲大开。</a>
