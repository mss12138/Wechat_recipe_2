recipe_name: 吃一次就爱进骨子里的带鱼做法
practice: 蒸
flavor: 咸鲜味
time: 60分钟
grade: 新手尝试
materials: 主料鲜到鲜得带鱼3条剁椒2勺辅料料酒3勺姜6片花椒15粒白胡椒粉适量大蒜4瓣生抽1勺蚝油半勺食用油适量小葱末适量
step: <a href=https://st-cn.meishij.net/rs/07/02/13813007/n13813007_163263969202699.jpg>第1步: 带鱼清理干净之后剪成小段，我用的鲜到鲜得的带鱼是去头去尾的，所以只需要去除内脏之后清洗干净就可以了</a>
<a href=https://st-cn.meishij.net/rs/07/02/13813007/n13813007_163263969362719.jpg>第2步: 处理好的带鱼加入姜丝料酒，白胡椒粉，花椒粒儿，抓匀腌制半个小时去腥</a>
<a href=https://st-cn.meishij.net/rs/07/02/13813007/n13813007_163263969565207.jpg>第3步: 腌制好的带鱼裹上薄薄的一层淀粉，放到锅里煎至两面金黄，淀粉一定不要裹的太厚，否则吃起来表面就是一层面，会影响口感</a>
<a href=https://st-cn.meishij.net/rs/07/02/13813007/n13813007_163263969654943.jpg>第4步: 盘子底部铺一层葱姜片，然后将煎好的带鱼放在上面，加入葱姜之后也可以很好的去腥</a>
<a href=https://st-cn.meishij.net/rs/07/02/13813007/n13813007_163263969786626.jpg>第5步: 另起锅少倒一点点油，放入姜蒜末两勺剁椒小火，炒出蒜的辛辣味，然后加料酒，生抽，蚝油白胡椒粉继续小火慢炒一两分钟</a>
<a href=https://st-cn.meishij.net/rs/07/02/13813007/n13813007_163263969872381.jpg>第6步: 炒好的剁椒铺在带鱼的表面</a>
<a href=https://st-cn.meishij.net/rs/07/02/13813007/n13813007_163263969931219.jpg>第7步: 然后将盘子放到锅中水开之后，大火蒸八分钟</a>
<a href=https://st-cn.meishij.net/rs/07/02/13813007/n13813007_163263970113242.jpg>第8步: 正好的带鱼表面撒上小葱末，最好用小葱，我家里面没有小葱了，所以就用大葱叶切成丝来代替的，最后浇上热油就可以了，这样做的带鱼吸收了剁椒的香气，还保留了本身的鲜，吃起来特别的鲜美</a>
