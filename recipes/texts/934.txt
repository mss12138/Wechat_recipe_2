recipe_name: 尖椒炒河虾
practice: 炒
flavor: 家常味
time: 10分钟
grade: 初中水平
materials: 主料河虾300克青红尖椒4个辅料植物油适量盐适量生抽5克姜5克
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158281053972702.jpg>第1步: 主料：河虾300克、青红尖椒3个

辅料：植物油、盐、香葱5根、姜5克、生抽10克、味精1克、香菜2根</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158281054190018.jpg>第2步: 把河虾洗洗沥干水分，香葱、香菜洗净切段，最好切的稍微长一些，这样好看</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158281054292174.jpg>第3步: 把青红尖椒洗净切丁，不爱吃辣的，可以把尖椒换成柿子椒或者不太辣的辣椒品种，但是香葱可以多加。热锅凉油，油烧热到7成，下姜末、尖椒翻炒出辣味</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158281054382793.jpg>第4步: 加河虾翻炒出香味，如果太干，可以稍微加点水多炒一会</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158281054585327.jpg>第5步: 加生抽翻炒均匀，我买的河虾比较咸，不敢加盐了，如果不咸的河虾，这时候可以加盐了</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158281054606756.jpg>第6步: 加香葱段翻炒出香葱的味道</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158281054887918.jpg>第7步: 加味精翻炒均匀</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158281054904223.jpg>第8步: 关火，撒上香菜段</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_158281055127380.jpg>第9步: 用锅底的余温，把香菜的香味逼出来，这样很好吃好吃</a>
