recipe_name: 酸辣土豆条
practice: 拌
flavor: 酸辣味
time: 30分钟
grade: 初中水平
materials: 主料大土豆1个辅料植物油适量盐适量土豆淀粉两大勺生抽5克辣椒粉一勺
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160665895243969.jpg>第1步: 主料：面土豆两个
辅料：植物油、盐、香葱2根、土豆淀粉2大勺、大蒜2瓣、辣椒粉一大勺、生抽5克、米醋5克、鸡精1克、白糖1克</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160665895298995.jpg>第2步: 把土豆洗净去皮，切薄片，加入蒸锅大火蒸熟透取出来</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160665895225083.jpg>第3步: 把土豆洗净去皮，切薄片，加入蒸锅大火蒸熟透取出来</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160665895254281.jpg>第4步: 加两大勺土豆淀粉，和成光滑的土豆面团，以不粘手为好</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160665895336690.jpg>第5步: 揪一块，用手搓成喜欢的小土豆条，不要太大了，就这样的条条即可</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160665895365059.jpg>第6步: 锅水开了以后，下入土豆条，煮到土豆条飘起来，立刻捞起来，（可以加少许的盐，增加韧劲）</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160665895354081.jpg>第7步: 马上下入凉水里面，过凉捞起来沥干水分</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160665895341401.jpg>第8步: 把大蒜去皮切末，香葱洗净切末</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160665895421572.jpg>第9步: 把切好的大蒜末和辣椒粉，用冒烟的热油泼出来，出香味以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160665895415185.jpg>第10步: 加白糖和生抽、米醋、生抽、盐搅拌均匀以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160665908568225.jpg>第11步: 撒上香葱末即可上桌吃了，Q弹弹的劲道真好吃，酸辣可口太美味了。</a>
