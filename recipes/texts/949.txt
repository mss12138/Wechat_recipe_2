recipe_name: 超简单的三汁焖锅，好吃下饭
practice: 焖
flavor: 家常味
time: 30分钟
grade: 新手尝试
materials: 主料鸡腿3个食用油适量土豆1个洋葱半个青红椒各一个辅料生抽2勺蚝油3勺料酒2勺海鲜酱1勺黄豆酱2勺老陈醋2勺白砂糖2勺
step: <a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056734600848.jpg>第1步: 鸡块洗净剁成块，加生抽、料酒、蚝油1勺，葱姜适量。</a>
<a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056734857568.jpg>第2步: 抓匀后，盖保鲜膜腌制30分钟以上。</a>
<a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056735166762.jpg>第3步: 洋葱切小块 。</a>
<a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056735495240.jpg>第4步: 土豆切块。</a>
<a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056735751867.jpg>第5步: 青红椒切块。</a>
<a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056736031383.jpg>第6步: 调酱汁：碗里加入2勺黄豆酱、1勺海鲜酱（市场和超市都有卖）、1勺蚝油、2勺味极鲜、1勺老抽、2勺白糖，3勺水，用小勺子搅拌均匀，酱汁就做好了。</a>
<a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056736328883.jpg>第7步: 砂锅底部先倒上适量食用油，再铺上一层洋葱块，一层土豆块和香菇片，我还切了两个好看的香菇小花呢。</a>
<a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056736650007.jpg>第8步: 放入鸡块，倒入调好的酱汁。</a>
<a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056737127905.jpg>第9步: 开火，大火煮开后转成最小火，焖煮20分钟就好了。</a>
<a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056737452362.jpg>第10步: 出锅前加青红椒块点缀。</a>
<a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056737741119.jpg>第11步: 软嫩鲜香。</a>
<a href=https://st-cn.meishij.net/rs/50/123/6030800/n6030800_162056738222805.jpg>第12步: 上桌开吃吧。</a>
