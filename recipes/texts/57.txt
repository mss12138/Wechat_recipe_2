recipe_name: 干豇豆香菇烧土鸡
practice: 烧
flavor: 香辣味
time: 60分钟
grade: 初中水平
materials: 主料土鸡半只干豇豆100克干香菇10个辅料植物油适量盐适量豆瓣酱两大勺老抽5克花椒粒1克草果1个
step: <a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717191927974.jpg>第1步: 主料：干豇豆50克、干香菇10个、土鸡半只
辅料：植物油、豆瓣酱一大勺、姜、花椒粒、草果1个、老抽、大葱、辣皮子5个、料酒10克</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717191949537.jpg>第2步: 辅料：植物油、豆瓣酱一大勺、姜、花椒粒、草果1个、老抽、大葱、辣皮子5个、料酒10克</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717191963487.jpg>第3步: 土鸡洗净切快，不用焯水直接干炒，我这是新鲜土鸡，炖汤腻味了，就想烧着吃，香</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717191979879.jpg>第4步: 热锅凉油，（油稍微比平时炒菜多一些，因为干豇豆和干香菇很吃油）油热7成下鸡块炒出香味出水分，多翻炒几分钟，把鸡肉外皮炒到微微发卷以后，这样水分炒干了，香味出来才好吃</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717191932408.jpg>第5步: 把鸡肉拨一边，就把葱、姜、辣椒、花椒粒、草果1个加进去，快速爆出香味以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717192019274.jpg>第6步: 把鸡肉拨一边，就把葱、姜、辣椒、花椒粒、草果1个加进去，快速爆出香味以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717192069419.jpg>第7步: 再倒入老抽和料酒继续翻炒均匀出酱香味以后</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717192053602.jpg>第8步: 加清水或者高汤，水量不要太多，加干豇豆和香菇块一起烧开，或者不用烧开，因为我用电饭煲烧制，损耗水分不多，如果用炒锅烧，可以把水量超过鸡肉</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717192072094.jpg>第9步: 把鸡块食材都加进饭煲压力锅</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717205744002.jpg>第10步: 盖上电饭煲的盖子，插上电源，启动“煮粥”模式；系统默认时间是50分钟，如果想增减时间，可以手动设置</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717205727257.jpg>第11步: 等时间到了，尝尝盐味，一般加少许的盐即可，搅拌均匀，再焖几分钟就可以出锅吃了</a>
<a href=https://st-cn.meishij.net/rs/241/24/818741/n818741_160717205757046.jpg>第12步: 鸡肉熟烂又劲道，干豇豆吸收了鸡肉的 香味，绵软好吃，营养又美味，全家都爱吃。冬天炖、烧这样一锅鸡，滋补鲜美，太香浓了。</a>
