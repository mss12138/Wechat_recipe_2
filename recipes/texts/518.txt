recipe_name: 马齿苋鸡蛋饼
practice: 煎
flavor: 咸鲜味
time: 15分钟
grade: 中级掌勺
materials: 主料马齿苋一把水160克面粉60克鸡蛋一个辅料油适量盐适量甜面酱一勺
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854455400.jpg>第1步: 面粉倒入碗中，加入水。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854481049.jpg>第2步: 用筷子充分搅拌，直到没有干粉颗粒即可。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854409809.jpg>第3步: 新鲜的鱼腥草长这样，城里的路边小河边经常可以见到，不认识的朋友可以用按照图片搜一下哟。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854425435.jpg>第4步: 将鱼腥草清洗干净，切成小段。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854407770.jpg>第5步: 处理好的鱼腥草段放入面糊中，加入适量盐，充分拌匀。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854540762.jpg>第6步: 做这个饼，必须用不粘锅，倒入少许油，加热后，将面糊倒入锅中，轻轻转动锅身，让面糊均匀地布满锅底，中火慢慢将饼煎至凝固。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854575971.jpg>第7步: 待饼底凝固，饼边缘变得透明的时候，表面的面糊还有少许没凝固的时候就可以磕入一个鸡蛋。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854516969.jpg>第8步: 用锅铲将蛋黄捣散，均匀布满整个饼面。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854599726.jpg>第9步: 盖上锅盖，加快鸡蛋液的凝固。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854509181.jpg>第10步: 待表面的蛋液微微凝固后，翻个面，将两面都煎至微微焦黄就可以了。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854555069.jpg>第11步: 看，颜色超漂亮吧，一看就有食欲。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_162674854694304.jpg>第12步: 没有鸡蛋液的这个面均匀刷些甜面酱即可。</a>
