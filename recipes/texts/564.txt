recipe_name: 嫩烤鲈鱼
practice: 烤
flavor: 咸鲜味
time: 30分钟
grade: 新手尝试
materials: 主料鲈鱼1条生菜1颗辅料味达美臻品料酒2勺味达美臻品蚝油3勺味达美味极鲜酱油3勺花椒10克
step: <a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643089052934.jpg>第1步: 食材简单，一条鲈鱼，一颗生菜，味达美的料酒、味极鲜酱油和蚝油，加上一些花椒就行</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643088694433.jpg>第2步: 先取一个碗倒入2勺味达美臻品料酒</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643088789482.jpg>第3步: 倒入3勺味达美味极鲜酱油</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643088720714.jpg>第4步: 倒入3勺味达美臻品蚝油，搅拌均匀之后就是灵魂烧烤汁了</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643088908870.jpg>第5步: 鲈鱼事先洗净之后用到划上几刀，将烧烤汁淋在上面，将烧烤汁在鱼上面涂抹均匀使酱汁可以渗透到鱼肉中</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643089072403.jpg>第6步: 撒上一些花椒</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643089006704.jpg>第7步: 放入冰箱冷藏腌制2小时，如果可以冷藏一晚上会更入味</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643089078034.jpg>第8步: 取一张锡纸，将腌制好的鲈鱼放在上面</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643089023904.jpg>第9步: 锡纸周围折起成船状，将剩余的烧烤汁淋在上面</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643088909497.jpg>第10步: 包好之后放在烤盘上</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643089181946.jpg>第11步: 烤箱200°C烤制15分钟即可</a>
<a href=https://st-cn.meishij.net/rs/227/67/10016977/n10016977_162643089085567.jpg>第12步: 用锡纸包住烤制出的鱼肉会保留鱼肉本身的水分，口感更加的鲜嫩</a>
