recipe_name: 干烧鲳鱼
practice: 煮
flavor: 咸鲜味
time: 15分钟
grade: 新手尝试
materials: 主料鲳鱼4条香菇2朵彩椒少许猪肉馅50克辅料味达美醇香米醋1大勺味达美臻品料酒1大勺味达美臻品特级生抽2大勺糖1小勺
step: <a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_162799688558414.jpg>第1步: 准备所需食材</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_162799688560004.jpg>第2步: 鲳鱼洗净，去掉内脏，表面划几刀以便入味</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_162799688558083.jpg>第3步: 加胡椒粉、味达美臻品料酒、葱段、姜丝、盐腌制10分钟</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_162799688575759.jpg>第4步: 香菇、红椒切成小丁，葱、姜切成小粒</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_162799688599802.jpg>第5步: 锅加油烧热，逐个放下鲳鱼，煎至两面金黄盛出</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_162799688617785.jpg>第6步: 锅内留底油倒入肉馅炒出油脂</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_162806606966052.jpg>第7步: 加葱、姜粒、辣椒酱炒出红油</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_162799688668956.jpg>第8步: 倒入适量开水，放入鲳鱼、香菇丁，加味达美臻品特级生抽、糖调味，加味达美臻品料酒去腥，大火煮开10分钟</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_162799688688407.jpg>第9步: 煮至汤汁粘稠，沿锅边烹一勺味达美醇香米醋提升香气把鱼捞出装盘</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_162799688725583.jpg>第10步: 剩余的汤汁加红椒碎煮开</a>
<a href=https://st-cn.meishij.net/rs/105/225/1306355/n1306355_162806673441749.jpg>第11步: 把剩余的汤汁浇在鱼上即可</a>
