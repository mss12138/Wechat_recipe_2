recipe_name: 炒西兰花
practice: 拌
flavor: 家常味
time: 10分钟
grade: 中级掌勺
materials: 主料西兰花一朵金针菇一包胡萝卜一根辅料猪油半勺蒜末适量盐适量
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548420762.jpg>第1步: 西兰花用刀将花朵切下来，放入冷水中个，加少许盐泡十分钟，清洗干净，沥干水备用。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548540825.jpg>第2步: 金针菇先不切，直接将前端冲洗干净。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548539113.jpg>第3步: 再将蒂部切掉。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548522083.jpg>第4步: 然后切成两三段备用。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548517797.jpg>第5步: 西兰花一切两半。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548734876.jpg>第6步: 企鹅号的西兰花放入蒸锅，蒸8分钟即可，平时大家做西兰花都是汆烫，这样维生素就会融入水中，造成浪费，这次咱们用蒸的方式将西兰花蒸熟再烹饪，西兰花口感好，营养不浪费。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548608935.jpg>第7步: 胡萝卜先斜着切成段。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548621403.jpg>第8步: 再切成菱形片。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548744666.jpg>第9步: 炒蔬菜，用猪油更好吃，荤素搭配营养也更好。舀一勺猪油，放入锅中，加热。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548752521.jpg>第10步: 炒香蒜末，将胡萝卜片炒软。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548607751.jpg>第11步: 胡萝卜微微变软后，将金针菇倒进去稍作翻炒。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_161581548622265.jpg>第12步: 待金针菇微微变软后，将蒸好的西兰花倒进去，加盐调味，炒匀即可。</a>
