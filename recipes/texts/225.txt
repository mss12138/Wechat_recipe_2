recipe_name: 玉木耳炒肉
practice: 炒
flavor: 家常味
time: 10分钟
grade: 中级掌勺
materials: 主料玉木耳一把猪肉100克青红辣椒各两个辅料油适量盐适量蒜末适量酱油一勺料酒一勺
step: <a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952759667.jpg>第1步: 玉木耳准备好</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952769811.jpg>第2步: 所有佐料准备好。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952700719.jpg>第3步: 玉木耳要提前用温水泡半个小时，然后放入锅中煮开，捞出来排入凉水中。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952738066.jpg>第4步: 玉木耳比较干净，蒂也不多，将底部带黑色的部分摘掉即可。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952786177.jpg>第5步: 大蒜拍碎后切成末，姜片切丝，青红椒也切斜片备用。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952794658.jpg>第6步: 猪肉切片，我家不喜欢吃肥肉，所以先将肥瘦分开，肥肉用来煎油。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952702750.jpg>第7步: 瘦肉片放入碗中，加入一勺酱油。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952715015.jpg>第8步: 加入一勺料酒。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952736183.jpg>第9步: 最后淋入一勺植物油，将猪肉封油，这样炒出来的猪肉口感更嫩，而且还不会沾锅。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952738453.jpg>第10步: 锅里倒入一勺油，将肥肉片倒入锅中，小火将油脂煸炒出来。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952747312.jpg>第11步: 然后将肥肉捞出来，倒入猪肉片，大火煸炒至变色，捞出来备用。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952753424.jpg>第12步: 将泡好的玉木耳和青红椒、蒜末都倒入油锅中，大火快炒一分钟左右。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952706262.jpg>第13步: 加盐调味，炒匀。</a>
<a href=https://st-cn.meishij.net/rs/208/102/1025708/n1025708_163438952747206.jpg>第14步: 将炒好的瘦肉片倒入锅中，加入少许酱油上色即可。</a>
