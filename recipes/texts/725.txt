recipe_name: 低卡低脂莜面碗托，好吃不长肉
practice: 蒸
flavor: 香辣味
time: 30分钟
grade: 未知
materials: 主料莜麦面100克清水150克辅料蒜一头花椒粉，孜然粉，姜粉适量生抽 醋蚝油适量盐少许油少量辣椒油少许
step: <a href=https://st-cn.meishij.net/rs/206/231/5120456/n5120456_162349198651419.jpg>第1步: 将莜面清水等材料准备好</a>
<a href=https://st-cn.meishij.net/rs/206/231/5120456/n5120456_162349198714693.jpg>第2步: 莜面加入碗中，倒入清水，搅拌成细腻稍微浓稠的面糊状</a>
<a href=https://st-cn.meishij.net/rs/206/231/5120456/n5120456_162349198905037.jpg>第3步: 取两只盘，底部刷一层油</a>
<a href=https://st-cn.meishij.net/rs/206/231/5120456/n5120456_162349199194477.jpg>第4步: 将莜麦面糊倒入盘底</a>
<a href=https://st-cn.meishij.net/rs/206/231/5120456/n5120456_162349199303951.jpg>第5步: 将盘子放入蒸锅，盖上盖子蒸15分钟左右</a>
<a href=https://st-cn.meishij.net/rs/206/231/5120456/n5120456_162349199523924.jpg>第6步: 将蒜切末，加入葱花，花椒粉孜然粉姜粉少许盐，将油烧热浇在上面，再调入生抽醋少许蚝油拌匀</a>
<a href=https://st-cn.meishij.net/rs/206/231/5120456/n5120456_162349199802575.jpg>第7步: 将碗托取出放凉切成细条摆入碗中</a>
<a href=https://st-cn.meishij.net/rs/206/231/5120456/n5120456_162349200200001.jpg>第8步: 将调好的料汁浇在上面，加辣椒油调味</a>
<a href=https://st-cn.meishij.net/rs/206/231/5120456/n5120456_162349200570177.jpg>第9步: 将料汁拌匀即可，香辣开胃，超级好吃！格琳诺尔有机面，营养美味无腹担</a>
